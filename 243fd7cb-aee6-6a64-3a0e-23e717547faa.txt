<br/>　　慕容丹很快也意识到，叶尘想要逃跑，不由得冷冷一笑，如影随形般追了上去。

<br/>　　转眼之间，
<br/>　　方圆数百里之内，尽是一片废墟，哀嚎声不绝于耳，犹如人间地狱！
<br/>　　周围围观的那些观众们，怎么也想不到，昔日一向慈眉善目的宗主大人，竟然有着如此疯狂残暴的一面！
<br/>　　冷蝉衣早已经美眸瞪得老大，望着眼前发生的一切，满脸的不可思议之色，
<br/>　　“师尊！您冷静点！不要杀人了！不要再杀人了！”
<br/>　　冷蝉衣忍不住高呼起来，可惜慕容丹根本不予理会，他现在一心只想得到吞天鼎，为此即便是毁掉整个云丹城也在所不惜。
<br/>　　其实就连叶尘也没有想到，这慕容丹竟然如此心狠手段！
<br/>　　他原本以为，躲进云丹城的这些普通人之中，慕容丹多少会有所顾忌，然后他便可以趁机逃跑。
<br/>　　没想到，慕容丹根本就不在乎这些人的生死。
<br/>　　“难道我今日，只能和他同归于尽了吗？”
<br/>　　眼看着慕容丹紧追不放，叶尘根本没有机会逃脱，而若是照目前的情况继续僵持下去，一旦逆生百转的加持效果消失，他便是砧板之肉，将再无任何翻身的可能！
<br/>　　就在叶尘心头一横，准备和慕容丹同归于尽的时候，叶尘忽然一眼瞥见前方人群之中，一名青衣老者，手里紧紧握着一个丹瓶，满脸的骇然之色。
<br/>　　“嗯？是了，此人就是之前在天藏阁，主持拍卖会的那个老头！那他手里拿得，应该就是我炼制的那枚完美品阶的化婴丹！”
<br/>　　这道念头在叶尘脑海中飞速闪过，叶尘顿时两眼猛然一亮，
<br/>　　“等等！化婴丹！”

<br/>　　下一刻，
<br/>　　青衣老者忽然感觉肩头一沉，似乎被什么人抓住了肩头，然后眼前一花，整个人如同腾云驾雾一般，完全找不到东西南北。

<br/>　　就在青衣老者一阵懵逼之时，耳边响起了一道淡漠的声音，
<br/>　　青衣老者这才发现，刚刚救下自己的，赫然正是那位，将正是那个将云丹城搅得天翻地覆的神秘青年！
<br/>　　青衣老者见是叶尘，哪敢有任何犹豫，慌忙将手中的丹药交还给叶尘。

<br/>　　青衣老者本想开口求饶，才刚刚开口说出一字，整个人便被叶尘，像丢垃圾一般，随手丢了出去，虽然有些狼狈，但好歹保住了一条命。
