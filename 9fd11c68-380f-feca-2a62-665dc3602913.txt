　　没有得到霍斯曼德的支持，并没有让Derrick放弃他的想法，他跟“毒皇后”玛丽科娃取得了联系。
<br/>　　玛丽科娃有着二分之一的俄罗斯血统，她十九岁获得世界小姐冠军，所有人都可以预见，玛丽科娃未来会有一条星光璀璨的星途，好莱坞、高价广告代言、各种商演片约接到手软，完全就是数钱数到烧掉点钞机的节奏。
<br/>　　可是出乎所有人的意料，玛丽科娃在获得世界小姐冠军后，就跟人间蒸发了一样，突然消失了。没有人知道她去了哪里，她就这么消失了。
<br/>　　玛丽科娃再次出现的时候，是在十年后，就在人们忘记她的时候，玛丽科娃又出现了，她已经不再是当年那个让世界惊艳的世界小姐冠军，她有了一个新的外号——“毒皇后”，她成了通缉犯，罪名是制毒、运毒、贩毒、并且涉及贩卖人口和军火。
<br/>　　除了玛丽科娃自己，没有人知道为什么她会从一个世界小姐冠军变成一个全球通缉犯。
<br/>　　玛丽科娃已经不再是当年那个青涩稚嫩的小姑娘，岁月仅仅只是在她的眼角为她留下几道鱼尾纹外，并没有对她的容颜造成太多的破坏，她依旧是那么的美丽和性感，她就是上帝用心打造出来的最出色的造物作品。
<br/>　　铺着一整张北极熊毛皮的椅子上，玛丽科娃慵懒的靠着，看着桌子上的笔记本电脑，就算是隔着屏幕，玛丽科娃也能感觉到从Derrick眼睛中流露出来的，对自己的迷恋。玛丽科娃对此并不感到意外，这样的迷恋，她见过太多太多了。
<br/>　　Derrick从联邦监狱中被带出来，见过玛丽科娃之后，会答应她为她服务，除了玛丽科娃答应的那些条件外，还有就是因为她的美貌。
<br/>　　“这就是我的想法。”Derrick眼睛中带着迷恋，看着玛丽科娃，说道：“而且我相信我一定可以获得成功。”
<br/>　　玛丽科娃抚摸着自己大腿上那只白色波斯猫柔顺的毛发，思考了好一会儿，随后点头道：“我会让霍斯曼德帮助你，你有任何条件都可以向霍斯曼德提，他有疑问就说是我给你的授权。”
<br/>　　“谢谢你，queen！”Derrick对着电脑中的玛丽科娃轻轻鞠了一躬，说道：“我不会让您失望的，queen！”
<br/>　　Queen！皇后！
<br/>　　外界给玛丽科娃的外号是“毒皇后”，所以“Queen”就成了玛丽科娃手下人对她的称呼。
<br/>　　有Queen，自然还会有King，但是很少有人知道玛丽科娃身后的“King”是谁，不少人甚至幻想着自己就是玛丽科娃身边的“King”！这其中就有Derrick。
<br/>　　得到了玛丽科娃的授权，霍斯曼德再也没有对Derrick的研究提出什么意义。
<br/>　　霍斯曼德按照Derrick的要求，霍斯曼德把一些已经对“水晶珠”成瘾的男人作为试验对象，送到“维伦恩”生物制药公司的研究中心大楼，供Derrick进行试验。
<br/>　　“我还需要更多的试验对象。”
<br/>　　Derrick看着因为服用了升级版“水晶珠”后，在实验过程中出现休克症状，像死狗一样被拖出实验室的男人，拿起电话对霍斯曼德提出要求。
<br/>　　“Derrick，我觉得我们必须要停止试验了。”霍斯曼德说道：“从你开始试验到现在，短短两个星期的时间，因为试验而猝死的男性已经超过了五十人，而几乎相同数量的女性，因为试验而出现不良症状，如果试验再继续进行下去，我们很有可能会被FBI盯上。”
<br/>　　“你打算要终止试验吗？”Derrick对着霍斯曼德质问道。
<br/>　　“不不不，我没有这个意思。”霍斯曼德回答道：“我认为，我们需要暂缓试验。第一批制造出来的‘水晶珠’数量并不多，而现在我能找到的已经对‘水晶珠’成瘾的男性有一大部分被送到你那里去了，我们现在没有多余的实验对象给你进行实验了，而如果我们用正常人进行试验，会惹来大麻烦！”
<br/>　　“亚洲那边呢？”Derrick对霍斯曼德说道：“我记得有一些‘水晶珠’被投放到了亚洲观察使用情况，从那里找一些实验对象来，正好可以观察不同人种对‘水晶珠’升级后的反应，以便我收集更多的实验数据。”
<br/>　　说完，Derrick就挂断了电话，命令自己的助手把下一组实验对象送进实验观察室，进行试验。
<br/>　　“疯子！”
<br/>　　霍斯曼德看了眼自己手中不断传来忙音的电话，咒骂了一句，最后还是拨通了李霄力的电话，这是玛丽科娃交代的事情，他必须要办好！
<br/>　　……
<br/>　　杭山市，一个建造于上世纪八十年代的老式住宅区，这里就是钱鑫的家。
<br/>　　钱鑫在得到了三儿的承诺后，这两天的日子就更难过了，晚上睡不着，白天没精神，盯着两个大眼袋把他家里人吓得不轻，询问钱鑫是怎么回事，钱鑫扯谎说是打游戏打得晚了没睡好。
<br/>　　这天本来应该是钱鑫晚班，可是他现在满脑子想的就是怎么跟陆岑酣战，哪里还有要上班的想法。
<br/>　　钱鑫用筷子拨弄着碗里的米饭，半天都没吃下去一口，钱母看着自己儿子没精打采的样子，很是担心，说道：“是不是身体不舒服，怎么脸色这么难看。”
<br/>　　“天天晚上不睡觉打游戏，脸色能好就见鬼了！”钱父冷哼一声，面色不善的说道。
<br/>　　“你怎么说话的！？”钱母不乐意了，把筷子往桌子上一拍，对着钱父骂道：“这是你儿子！”
<br/>　　“我当初真后悔没把他射在墙上！”钱父冷声说道。
<br/>　　“钱松华！”钱母站起身来，对还坐在桌子边的钱父说道：“你看不惯我们母子俩，我们就走！你眼不见心不烦！”
<br/>　　“走走走，你们都走！”钱松华把碗一推，站了起来，对钱母说道：“要不是你这么惯着他，他能像现在这样？”
<br/>　　“行了，你们别吵了！”钱鑫站起身来，拿起窗台上小电驴的钥匙，说道：“我去上班了。”
<br/>　　“记得自己去看医生。”钱母冲着已经关门离开的钱鑫喊道。
<br/>　　“慈母多败儿！慈母多败儿！”
<br/>　　房间里，钱松华抽着烟，埋怨着自己老婆对儿子的纵容和溺爱。
