
<br/>　　<br/>　　    “小鱼儿没事了以后，其实我也没太放下心，因为打蛇不死的话，蛇还得顺着棍子往上爬，说不好就会再咬你一口，老弟是不是这个道理？”沈君给王惊蛰倒了杯酒，举杯示意喝一口。<br/>　　<br/>　　    王惊蛰端着杯子，看了眼对方，他知道接下来谈的才是肉戏，之前说的那些全是铺垫，沈君在农家院里搞的这个也不是纯粹的答谢宴，背后肯定还得有别的明堂。<br/>　　<br/>　　    一杯酒喝完，沈君抹了下嘴唇上的酒液，接着说道：“我找到了给小鱼儿吊坠的那个女明星刘珺瑶，下手的肯定不是她，她就一靠着脸蛋和身材混的戏子，哪里会懂得这些牛鬼蛇神的玩意儿啊？所以我还得把幕后真凶给找出来才能放心，然后就从刘珺瑶那里问了一下，才知道那个吊坠是一个叫胡胡的人给她的”<br/>　　<br/>　　    听到这个名字，王惊蛰一丁点反应都没有，毕竟不记得之前的事以后，他也想不起来这个胡胡是曾经在长安差点把他给圈进去的玄门山字脉的人。<br/>　　<br/>　　    “打听到这个胡胡的人后，我就托人查了下他的底细，他在你们这行里还真是个挺有名的人，胡胡是玄门中人，山医命相卜五脉中山字脉的山主，年纪不大没到三十，也算是个角色了”<br/>　　<br/>　　    王惊蛰“哦”了声，还是没什么反应。<br/>　　<br/>　　    “唰”沈君把杯子里的酒一饮而尽，砸吧着嘴说道：“我这个人的性子吧，就是对待朋友忠肝义胆，但对于敌人必须一是一二是二的，我就小鱼儿这一个亲人，她如果是生老病死而终的话我没什么可说的，但要是差点死在人手里，我就不太愿意了，都是江湖儿女，谁没一点脾气呢？”<br/>　　<br/>　　    王惊蛰皱了皱眉，说道：“你和我说这些，不光是有脾气吧？”<br/>　　<br/>　　    “唉，对喽，发脾气那是没能耐的人，有能力的都会把脾气转化成动力了”沈君忽然歪着身子，拍了下王惊蛰的大腿，笑道：“小老弟，我找你来，除了是说声谢谢外，也想你再继续帮我个忙，江湖事江湖了，面对一个玄门山字脉山主，我恐怕不太好出手，而我的身边也没什么合适的人选”<br/>　　<br/>　　    “你拿我当打手了？”王惊蛰挑着眉毛说道。<br/>　　<br/>　　    “哈哈……”沈君愣了愣，单很快就摇头说道：“扯淡，我当你是朋友，如果是想找打手话，可能咱们坐在这里的时候，我就已经把支票给你，直接就谈钱了”<br/>　　<br/>　　    王惊蛰说道：“君哥，你应该知道的，陈老板找我过来，我是因为人情的原因，而我把小鱼儿救回来之后，这个人情就已经过去了，不存在谁欠着谁了对吧？坦白的讲，我们这类人是很少愿意招惹麻烦的，除非不得已，没人会无故去结仇，因为你一个不小心就容易牵扯太多了”<br/>　　<br/>　　    “我知道……”沈君点头说道。<br/>　　<br/>　　    王惊蛰端起酒杯，跟对方碰了下，说道：“既然你说是朋友，那喝酒聊天都可以，麻烦就别让我沾了”<br/>　　<br/>　　    “行，喝酒”沈君表情几乎没什么异样的就跟王惊蛰对喝了一个，然后接下来他果真也没再提这个事，重新把话题又给聊到了风花雪夜上，并且一直到喝完酒之后，他也一个字都没有在提过。<br/>　　<br/>　　    晚上十点左右，酒过三巡了，王惊蛰和沈君吃完饭从农家院里出来。<br/>　　<br/>　　    “呼，挺长时间没喝这么多了，今个是真不容易啊，算是开戒了，我二十岁之前喝酒是因为无所事事，三十岁前喝酒是因为了拼搏，等到了三十岁以后，也没几个人能让我主动把酒杯给抬起来了”<br/>　　<br/>　　    王惊蛰顿时拱了拱手，说道：“您真给面子，君哥”<br/>　　<br/>　　    “哈哈，好了，好了，客套的事不提了，以后还有再见面的机会，我就让司机送你回去吧”沈君随后就叫了一个马仔过来，然后开车把王惊蛰送了回去。<br/>　　<br/>　　    看着离去的那辆车，沈君刚才微醺的神情又瞬间恢复了正常，有个埋在在他旁边轻声说道：“君哥，他真不答应啊？那接下来得怎么办呢，要不要再准备点什么？”<br/>　　<br/>　　    “不好说啊……”沈君摆了摆手，说道：“看看吧，今晚等他的电话”<br/>　　<br/>　　    在车上的时候，王惊蛰给菜刀文打了个电话，问他们在家干什么呢，电话里头全是震耳欲聋的音乐声，还有呼喝的吵闹声，没等对方回答，王惊蛰就知道他们几个肯定闲不住去丁武的酒吧里了。<br/>　　<br/>　　    “都是二十来岁血气方刚的青年，谁能在家一天天的呆着啊？我们出来造作了，在缪斯呢，你过来找我们吧”菜刀文打着酒嗝说道。<br/>　　<br/>　　    “候着吧，早知道我不打电话，直接回去好了”王惊蛰头疼的揉了揉脑袋，跟司机说了声去缪斯，就闭着眼睛休息了会。<br/>　　<br/>　　    半个多小时后，车到了缪斯门口，王惊蛰刚推开车门，司机忽然叫了他一声，然后从副驾驶拿起个盒子递给了他。<br/>　　<br/>　　    王惊蛰接在手里看了两眼，不用想都知道，这肯定是沈君让给的。<br/>　　<br/>　　    “什么东西啊？”<br/>　　<br/>　　    司机说道：“君哥说了，一点小礼物，算是对您帮忙的答谢，呵呵……王先生您可不要拒绝，我就是个司机，只能听上面的令，您要是不收的话，那就是为难我了”<br/>　　<br/>　　    王惊蛰犹豫了下，说道：“他这是知道，如果是自己给我的话，我可能会拒收，这才让你转交了”<br/>　　<br/>　　    司机笑了：“大人物的社交都是很有技术含量的，反正我是不懂……”<br/>　　<br/>　　    “替我谢谢君哥，麻烦了”王惊蛰拿着盒子从车上下来，走进了酒吧，手里的东西他也没太在意，也只以为是沈君出于感谢而送的礼物，因为盒子挺轻的，没感觉到什么份量。<br/>　　<br/>　　    进来后，就看见菜刀文，林汶骐还有小草坐在一张卡座上，丁武在另一边跟酒吧的经历交代着事。<br/>　　<br/>　　    “嘎吱”王惊蛰推开沙发，挨着小草坐了下来，把手里的盒子放在了桌子上。<br/>　　<br/>　　    小草眨了眨眼睛问道：“什么日子，怎么想起给我送礼物了呢……”
