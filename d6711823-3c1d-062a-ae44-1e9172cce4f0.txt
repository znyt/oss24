　　处理好手头上的事情，李承乾发现，似乎还有一个更大的麻烦要处理——老头子非要出海去看世界。
<br/>　　凭心而论，李承乾现在并不想让老头子离开长安，因为他还需要一个人帮自己镇场子，如果老头子在这个时间离开，刚刚登上帝位的他多少还是觉得心里毛毛的。
<br/>　　带着一肚子的纠结，李承乾换了一身以前常穿的便服，窜到了大明宫，来到了位于太液池边的含凉殿。
<br/>　　老皇帝李二陛下这个时候正在做出行的准备，见李承乾来了便招呼他到书房坐下问道：“你不在自己宫处理政事，却来朕这里做什么？”
<br/>　　李承乾苦着脸郁闷的说道：“父皇，儿臣这不是心里没底么，您就不能扶上马，送一程？”
<br/>　　“胡说八道，什么扶上马，送一程，偏生你小子事儿多。”老李看着儿子撒赖的样子，不由有些好笑，瞪了他一眼继续说道：“眼下你已经是大唐的皇帝，就要拿出皇帝的威风，总是这般惫懒怕是形象都被你自己给毁了。”
<br/>　　“毁就毁了吧，反正我在那些老家伙眼里也没啥形象，不过父皇您就不能晚些再走么？现在好歹等到初夏的时候也行。”李承乾这次就是抱着撒泼打滚的态度来的，自然是怎么无赖怎么来。
<br/>　　但他还是低估了老李离开的决心，不，或者应该说他低估了老李想要长生的决心，只是老头子把这事儿瞒的很紧，别说李承乾，就连长孙皇后甚至都被蒙在鼓里。
<br/>　　一面是长生，一面是留在长安帮着儿子稳定已经稳的不能再稳的朝局，换成任何人都知道如何先择吧？更不要说李二这几年年龄越来越大，心中对长生的渴望亦是愈发执着。
<br/>　　当然，李二其实心里也明白长生不老这种事情并不现实，可是到了他现在这个位置，除了长生还有什么可追求的呢，皇帝当了这么多年，说句不好听的，如果不能长生继续当下去也没啥大意思。
<br/>　　土地什么的现在在李二眼中也就是个数字，国库里的钱也同样如此，不过就是一组不断增加的数字罢了，这些数据经过这么多年，已经不能再让他激动，所以海外仙山自然就成了这位前皇帝的唯一寄托。
<br/>　　李承乾并不清楚这一点，直到现在他还以为老头子退位都是因为魏征的一句话，心中对老头子还是一肚子愧疚，甚至一度还有把皇位还给老头子的冲动。
<br/>　　“承乾啊，你是朕的儿子，要对自己有信心，朕相信你一定可以做的比朕好，你的心思朕很清楚，不过雏鸟总有自己独自离家的时候，现在已经到了你自己飞的时候，朕……”李二说到后来，双眼微微有些湿润。
<br/>　　纵然他这一次是想要找长生之路，但想起往日为了政务，与儿子和女儿们聚少离多，心中也有些难受。
<br/>　　“那，父皇这次远行有什么打算么？可有朝臣同行！”李承乾见实在无法必变老头子的想法，只能退而求其次，向老头子问起此行的打算。
<br/>　　“你舅舅是要随行的，还有张亮，尉迟和叔宝！”说起同行之人，老头子也不瞒李承乾，反正这事儿早晚他都会种知道，告诉他也无所谓。
<br/>　　“张亮？”李承乾皱了皱眉头。
<br/>　　“怎么，你想朕把他留下？”
<br/>　　“不，儿臣觉得张亮不怎么靠谱，五百假子，好大的威风！”李承乾很想说张亮那家伙脑后有反骨，但无凭无据之下，他也不能信口开河，所以只能点到为止的说道。
<br/>　　老头子若有所思的看了一眼李承乾：“你对他不放心？”
<br/>　　“是的，舅舅和尉迟恭、秦琼都是您身边的重臣，对他们儿臣放心，可是这张亮的话，如果可以不带还是不带的好。”
<br/>　　对于李承乾的建议，老李不置可否，只是淡然问道：“可若是不带张亮，若遇水战如何解决？”
<br/>　　“父皇，张亮虽然擅长水战，但现在的战船和武器已经与以前大不相同了，他的战斗理念并不一定适合现在的潮流和趋势。”
<br/>　　“哦？莫非你这铁甲舰还有什么门道？”
<br/>　　“不，门道什么的没有，只是以前我们的战船主要是以接舷战为主，打起仗来要努力靠近敌舰。可是现在不同了，铁甲舰全部装配了那种可以远距离攻击的贞观炮，这样便可以在数千步之外向敌舰发起攻击，接舷战什么的完全用不上了。”
<br/>　　“竟然是这样么？”李二咂了咂嘴，犹豫了起来。
<br/>　　他这一次出行要带女眷，还要去寻仙，带上一个不稳定的因素似乎真的有些让人不大放心。可若是不带上张亮又能带谁呢？既懂水战，又靠谱的人可不大多了。
<br/>　　“父皇，其实原本儿臣是想向您举荐李震的，也就是李绩家的大公子，但是刚刚儿臣又想起一人来，不知父皇还有没有印象。”李承乾发现老头子陷入犹豫，嘴角微微一挑，卖起关子。
<br/>　　“谁，说来听听。”老头子看李承乾的样子便知道这小子又想卖乖，不由有些好笑的遥遥点了点他：“若是真有合适人选，朕有赏。”
<br/>　　“嘿嘿，那儿臣先谢过父皇赏赐！”李承乾识趣的拱了拱手，接着便说道：“父皇可还记得独孤青云？！”
<br/>　　“青云……！”老头子先是一愣，但很快脸上便浮起一丝怀念，喃喃说道：“他现在还好吧？人在什么地方？”
<br/>　　“还在崖州，守着他的破岛谁也不让上去。”李承乾摊了摊手。
<br/>　　“朕听说他在南海还有个了不得的绰号，似乎叫什么‘海龙’？”
<br/>　　“对，是有这么个绰号，这老家伙在南海可是没少打仗，经验上不比张亮差，而且他当初还给父皇您当过护卫，算来忠诚度也够。”
<br/>　　“嗯，如果是他的话，还真可以。”李二缓缓点了点头说道。
<br/>　　他可没有忘记，当初正是这个独孤青云在南海清剿了无数海盗，使南海一带盗匪绝迹。
<br/>　　以他的经验来说，无疑是比张亮要强上许多，至少对于海路的熟悉程度上，不用让人操心。
