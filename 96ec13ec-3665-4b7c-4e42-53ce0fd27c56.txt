　　最终叶帝他们这边选择了：上单鳄鱼，打野蜘蛛，中路狐狸，下路则是ez和娜美这两个英雄。
<br/>　　而处在蓝色方的ka战队则是选择了：上单纳尔，打野挖掘机，中路机械先驱，下路则是小炮和布隆这两个英雄。
<br/>　　“这一局ka战队这边的阵容选的很好啊，虽然把香蕉的鳄鱼放了出来，不过他们这边反手却是拿下了一个纳尔，我们都知道这个英雄打鳄鱼还是非常好打的，看来ka战队这边真的是早有研究啊。”看着双方的选人，米粒开口分析着。
<br/>　　“没错，而且他们下路的选人也是有着不错的counter作用，尤其是这个布隆，他的盾牌是可以完美阻挡ez和娜美的大招的，尤其是娜美的大，是很容易被布隆挡下来的。”林雅楠跟着开口说道。
<br/>　　“不过虽然阵容对方强上一筹，但是这个游戏还是要看选手的临场发挥的，这一局大魔王可是拿出了他的拿手英雄狐狸，之前我们已经无数次看到他用狐狸打下的战绩了，而且最重要的一点是夜雪的蜘蛛这个点，香蕉如果在上路打的强势一些，那么他们两个这个动物组合是完全可以强杀对方的。”米粒继续说着。
<br/>　　而就在两人分析着比赛的时候，场上双方已经是进入到了游戏之中，过了15秒的等待时间之后，双方队员飞快的跑出了基地，在各个路口把控着。
<br/>　　很明显双方都是不想打这个一级团，现在已经是四分之一决赛了，大家打的都很谨慎，不光是ka战队如此，就是叶帝他们也都是如此。
<br/>　　他们的实力虽然要比ka战队强，但是强也是强的有限，并不是绝对的实力碾压，所以叶帝他们也怕阴沟里翻船的事情发生。
<br/>　　一级团太不稳定了，如果被对方的关键位置拿到人头的话，那么他们这边可就真的不好打了，尤其还是对方有着布隆这样的团战bug在，叶帝几人自然是更加的不敢动手了。
<br/>　　1分40秒双方正常对线，在前期大家倒是只是补着小兵，并没有做出太多的动作。
<br/>　　而上路的香蕉在面对着对方的纳尔的时候，出门装也是罕见的选择了多兰盾这件装备，此刻硬扛着纳尔的几下伤害，愣是用q技能清掉了三个小兵，而他的血量也是没有耗费掉太多。
<br/>　　在前几分钟两边打的都是非常的安稳，并没有做出什么太过挑衅的动作。
<br/>　　当时间到达5分钟的时候，香蕉利用对面纳尔变大的这段没法压制自己的时间，硬生生的将兵线推了出去，并且还将自己的怒气叠到了半管以上。
<br/>　　到了这个时候香蕉终于是找到了反击的机会，只不过这个时候纳尔也学的聪明了，看着鳄鱼怒气这么满，只是在远处利用自己的q技能补兵，根本不给香蕉消耗自己的机会。
<br/>　　鳄鱼这个英雄真正让人惧怕的就是在他拥有着怒气的时候，显然香蕉也是非常清楚这一点，在自己怒气叠上来后，他便是不再释放自己的技能。
<br/>　　他知道自己如果将怒气消耗光之后，那么纳尔肯定会重新反击回来的，而现在自己有怒气，那么就可以很好的起到一个威慑的作用了，而且更重要的是这个时候夜雪的蜘蛛已经赶过来了。
<br/>　　“这一波夜雪的蜘蛛来上路了，他们想要越塔强杀，感觉有机会！”看着白队这边的兵线，在看着夜雪狂歌的身位，米粒顿时激动的开口喊了起来。
<br/>　　看着鳄鱼还在将兵线向着自己的防御塔下推过来，ka的纳尔这个时候也是反应了过来，只不过眼前还有两波兵他实在不想就这么撤退。
<br/>　　要是让他亏掉这两波小兵的话，那么接下来他可就没法像之前那样继续压制鳄鱼了，而且仗着自己手中还有e技能和闪现，纳尔倒是想搏一搏。
<br/>　　万一真的能够反杀一个，那自己可就真的是天秀了，甚至可能一战封神。
<br/>　　每一个选手都有一个想要封神的念想，他自然也是不例外，所以在犹豫了一番后，纳尔便是小心翼翼的躲在了塔下，现在大家还都没有到六，只要自己能够躲过蜘蛛的e技能眩晕，到时候利用q技能减速，那么很可能会反杀对手的，毕竟现在大家还没到六，都很脆。
<br/>　　下一刻，在无数观众的关注下，白队这边的兵线终于是推到了对方的塔下，而就在兵线到达对方塔下的一瞬间，夜雪狂歌手中的蜘蛛已经是从对方它后的三角草丛中走了出来。
<br/>　　看到蜘蛛出现的一刹那，纳尔心下一咯噔，果然还是来了，刹那间他的目光便是变的专注了起来，这一波如果自己死了，那上路就彻底爆炸了，如果能够反杀对手甚至不死，那么自己就一战封神。
<br/>　　然而想法是美好的，现实却是很骨干，在他还在全神关注等待着躲避蜘蛛e技能的时候，这边香蕉已经是一个果断的闪现来到了纳尔的身边，同时w技能开启，刹那间纳尔便是被眩晕在了防御塔下。

<br/>　　“这个时候夜雪的蜘蛛还没有释放自己的e技能，他在等，要错开和鳄鱼w技能眩晕的这段时间，以求打出最大的输出，不过这样的话对于选手的要求实在是太高了，哪怕有一毫秒的耽搁，可能这边纳尔就闪现逃出去了，那样的话他们可就亏大了。”看着场上的局势，米粒连忙开口说道。
<br/>　　“wow！！！蜘蛛的e技能命中了，这下纳尔没有逃跑的希望了。”下一刻看到蜘蛛的结茧命中到对手的时候，林雅楠顿时兴奋的大声喊了起来。
<br/>　　与此同时场下也是响起了震耳欲聋的欢呼声，在两人完美控制的衔接下，在纳尔那一脸灰败的目光中，纳尔的身躯终于是缓缓的倒在了地上。
<br/>　　过了片刻召唤师峡谷便是响起了蜘蛛击杀纳尔的声音，刚刚是香蕉在顶塔，所以在打出自己的一套输出后，他便是撤回了场外，没敢继续耽搁下去。
