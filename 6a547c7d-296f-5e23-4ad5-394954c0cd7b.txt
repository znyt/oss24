　　现在叫龙千绝赶来，是绝对来不及的。
<br/>　　除非是现在楚风眠放弃这佛寺之中的宝藏离开，才可以让五行刀圣的偷袭计划落空。
<br/>　　可放弃佛寺，楚风眠却是无论如何也做不到，佛寺之中可是有着人皇罐碎片。
<br/>　　就算是其中的任何一件宝物楚风眠都不要，这人皇罐碎片楚风眠也必须要得到。
<br/>　　再说逃得了一时逃不了一世，只有提升实力，才可以真正的抗衡三大势力。
<br/>　　一直逃下去，永无宁日。
<br/>　　“以我们三人的实力，联手之下都不是五行刀圣的对手，就连跟他抗衡都不可能。”
<br/>　　燕皇沉思了一下，开口道。
<br/>　　“我在古籍之中，见到过这关于五行刀圣的记载，据说他在九劫古帝巅峰之时，就曾经以一敌五，对抗五名九劫古帝而不败。”
<br/>　　“现在此人已经是踏入了半步天人境，实力不知道提升了多少，我们跟他硬拼，只有死路一条。”
<br/>　　“不能应拼，我们只要是进入到了佛寺之中，那五行刀圣在想要出手，就没那么容易。”
<br/>　　楚风眠也开口道。
<br/>　　楚风眠也猜测了一番，以五行刀圣的身份，亲自出手，自然不可能单单是为了杀楚风眠一个人而来。
<br/>　　楚风眠自认他还没有这个本事，可以让一名半步天人境的强者特意来杀他。
<br/>　　五行刀圣的真正目标，应该还是这佛寺，佛祖洞府其中的宝藏，才是这五行刀圣出世想要得到的东西。
<br/>　　五行刀圣这种半步天人境的强者，如今唯一的目标便是冲击天人境，这些古老洞府之中的许多东西，都对他们冲击天人境有着大用。
<br/>　　就像是楚风眠曾经在狮霸武祖的洞府之中，得到了那一瓶武祖精血，其中便是就蕴含天人境的力量。
<br/>　　对那些想要冲击天人境的半步天人境强者来说，那一瓶武祖精血绝对都是他们梦寐以求的至宝。
<br/>　　五行刀祖多半是为了佛寺之中的宝物而来，至于偷袭杀楚风眠，也是顺手的事。
<br/>　　只要是楚风眠能够逃过第一次偷袭，等到楚风眠冲入到了这佛寺之中的时候，五行刀圣也必然会在其中去抢夺宝藏，而不可能一直是去追杀楚风眠。
<br/>　　倒时候鱼龙混杂，楚风眠在想要逃也不困难。
<br/>　　只是重要的是，逃过这第一次偷袭。
<br/>　　“一会佛寺开启，我独自一人抢先进入其中，你们稍后在进。”
<br/>　　楚风眠考虑了一下，随即开口道。
<br/>　　“楚兄，你这是？”
<br/>　　“这太危险了。”
<br/>　　燕皇跟青梦听到楚风眠的话，都是脸色一变的说道。
<br/>　　像是这样大型宝藏刚刚开启的一刻，便是无数强者蜂拥而去的时候，无数强者都想要在这一刻抢占先机，步入其中。
<br/>　　谁想要第一个进入洞府，谁就会受到众人的围攻。
<br/>　　许多的九劫古帝，都是在抢夺先机的时候，死在了众人的轰击之下。
<br/>　　楚风眠要是冲过去，纵然是楚风眠的实力比起许多九劫古帝还要强大，可是若是想要从上百名九劫古帝的眼皮底下冲进去，几乎是在找死。
<br/>　　不用上百名九劫古帝其中出手，其中一成的人出手，都足以将楚风眠轰杀成碎片。
<br/>　　“横竖都是一死，这是最后的机会，再说我有着五成把握，可以冲进去。”
<br/>　　楚风眠眼神之中露出了几分坚定之色。
<br/>　　这个计划，无比冒险，但是却是楚风眠想到最好的计划了。
<br/>　　与其是在这里，等待着五行刀圣的偷袭，不如说去抢占先机。
<br/>　　楚风眠肉身坚韧，神树也因为得到了建木树枝，现在已经是发生了蜕变，也未必是没有可能可以抢占先机冲进去。
<br/>　　“五行刀圣的目标是我，应该不会为难你们二人，如果遇到危险，可以前往北域，我在北域之中建立起了一方势力，名为剑道门，你们可以先去那里。”
<br/>　　楚风眠对着青梦跟燕皇说道。
<br/>　　这佛寺一旦开启，其中的危险无人可以知道，这样的混乱之下，也许九劫古帝都难以自保。
<br/>　　遇到危机，只要是躲在剑道门之中，以剑道门布置出的无数剑阵防御，也足以抵挡一些危险。
<br/>　　“就这样决定了。”
<br/>　　楚风眠看了青梦还想要开口说些什么，楚风眠一把打断道。
<br/>　　随即楚风眠便是将这周围的禁制扯开，楚风眠的目光看向了周围，在这周围之中，隐藏着不少的强者。
<br/>　　当然也有许多的九劫古帝，根本不隐藏，就站在那里，这就是九劫古帝的自信，不怕任何人的算计。
<br/>　　“单单是三大势力的九劫古帝，居然就有三十多位。”
<br/>　　楚风眠拿出了窥神镜，一股灵力注入其中，便是发现在这方圆百里之中，三大势力的九劫古帝，都足足有着三十多位。
<br/>　　至于其他不属于三大势力的九劫古帝，还要更多，方圆百里之内，足足聚集着一百多名九劫古帝。
<br/>　　在大罗天星域之中，上百名九劫古帝同时出现的场面，都足足数万年未曾出线过了。
<br/>　　今日就在这赤红大陆之中，强者云集，当然许多的九劫古帝，甚至都不是大罗天星域的武者，而是属于其他星域的强者。
<br/>　　“人皇罐……”
<br/>　　就在楚风眠扫视周围情况的一刻，在楚风眠身躯之中的人皇罐突然间的苏醒，躁动了起来。
<br/>　　这一股躁动，便是证明在这附近，又有人皇罐碎片出现了，在吸引着如今楚风眠的人皇罐。
<br/>　　楚风眠顺着人皇罐的躁动，目光看了过去，突然看到了另外一座山峰之上，聚集着的一群武者。
<br/>　　这一群武者，身着衣衫无比接近，显然是出身自同一方势力，其中为首的，是一名脸色傲气的年轻男子。
<br/>　　正是曾经楚风眠刚刚来到赤红大陆，就见到的严家一行人，那为首之人，就是严家的公子君少，严君。
<br/>　　只是比起上一次见，这严家一群人的强者变的更多，多出了两尊老者，站在了这严君的身后，守护这严君。
<br/>　　这两尊老者身上都蕴含着雄厚的天地之力，散发出去，镇压一切。
<br/>　　这是两位九劫古帝。
<br/>　　正是因为这两尊老者身上的气息，这严家众人才占据了那一座山峰，无人敢去窥探。
