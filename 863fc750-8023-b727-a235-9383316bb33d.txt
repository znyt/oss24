　　“臣参见陛下，参见长公主！”随着李承乾声音落下，崔钰的身影出现在大堂的正门口。
<br/>　　“崔钰，你真的是判官？”长乐看着与常人一般无二的老崔，有些不大确定的问道。
<br/>　　“是的长公主殿下，希望殿下今后能替微臣保守这个秘密!”崔钰尴尬的笑了笑。
<br/>　　“好了，时间有限，长乐，你有什么事情大可以后再与崔钰慢慢聊，眼下我还有些事情要他办，就先这样吧。”李承乾抬手打断还想再问些什么的长乐，而后又对崔钰说道：“我们走吧，把该办的事情办了。”
<br/>　　“是，请陛下随臣来！”崔钰先是对长乐辞行，而后转为李承乾引路，向着外面走去。
<br/>　　“哥……”长乐见李承乾要走，心里也开始有些慌乱。
<br/>　　第一次来的时候她迷迷糊糊的，还有些搞不清楚是怎么一回事儿，后来李承乾到了，长乐立刻就像是找到了主心骨，虽然知道是身处阴间，倒也没有怎么害怕。但现在知道李承乾要离开，长乐顿时就感到了一阵恐惧，拉了一下李承乾的衣袖试图挽留。
<br/>　　“放心吧，用不了多少时间的，哥一定尽快接你出去。”李承乾回头在长乐略有些虚无的小手上捏了一把。
<br/>　　两人现在俱是魂体，故而彼此间可以有实质上的接触，如果是一魂一实体那便再也没有了接触的机会。
<br/>　　……
<br/>　　从崔钰的地盘上离开，李承乾回到阳间，此时外面已经天光大亮，不过因为长乐的事情，并没有人来打扰他。
<br/>　　李承乾见无人打扰，索性也没有出去，叫过崔钰沉声问道：“崔钰，事情到了这个地步，朕只问你一句话，朕想要长乐归来，你有没有什么办法。”
<br/>　　“陛下，臣无能！”崔钰摇了摇头。
<br/>　　“那么你有什么办法没有？”李承乾继续问道。
<br/>　　“若说办法，怕是只能找阎君才可以，只是臣也不敢保证这件事是否能成。”崔钰显的有些犹豫，似乎真的很为难。
<br/>　　这一次，李承乾并没有为难崔钰，只是略有失望的哦了一声，但很快便又打起精神吩咐道：“朕不管你用什么办法，总之朕要见你口中的那个阎君一面，这件事情你去安排，越快越好。”
<br/>　　“陛下，臣以为这并不是什么好主意，正所谓生死有命，富贵在开，万事强求不得。”崔钰似乎知道一些什么东西，但却没有明确的说出来。
<br/>　　“朕管不了那么多，既然你没有权力，那就去联系有权力的人，朕意已决！去吧！”李承乾摆了摆手，示意崔钰下去。
<br/>　　聪明人有聪明人的好处，也有聪明人的优点，崔钰一次又一次被呼来喝去却不敢有半句怨言，这在李承乾看来绝对不是因为自己身上的主角光环，而如果不是主角光环，那就一定是自己身上有让崔钰畏惧的东西。
<br/>　　古时帝王受命于天，也就是说人间的皇帝一般来说都是在所谓‘天’上挂了号的人物，换句话说就是有公职在身的人。
<br/>　　这就好像后世集团企业一样，上面一个总公司，上面一堆的子公司，他李承乾就好比是总公司某官员，兼职了某子公司总经理，而地府则是另一个子公司，最高管理者最多也就是与他平级。
<br/>　　而崔钰这家伙就是李承乾这个子公司招来的员工，不过眼下被借调到地府那边去了。
<br/>　　这样的情况下，李承乾认为自己是有权力将崔钰开除的，而他一旦将崔钰开除，那么他就不是公司里的人，借调的身份也会立刻消失，那么他就会变成一个无业游民。
<br/>　　如果自己这个皇帝的地位如果再高一些，也就是说在总公司的身份地位再高一些，他甚至应该可以直接任免地府里的一些官员，或封或赦不一而足。
<br/>　　也正是因为有了这样的理解，所以李承乾才会让崔钰给去那个所谓的阎君带话，约谈长乐的事情。
<br/>　　待到崔钰离开，李承乾轻轻敲了敲桌子：“来人！”
<br/>　　“陛下！”随着李承乾的命令，门外进数道人影，夜魅，小白，黑子，称心、杨天、吴辰……
<br/>　　“命令，派出所有人手，全国缉拿玄寻雪，王成虎，长白山水月，另外，锁拿钦天监李淳风，道士袁守诚，袁天罡叔侄，玄都观道士秦英入大理寺！还有，若有妄议长乐之事者，亦全部锁拿，反抗者，杀无赦！”
<br/>　　“是，臣等遵旨。”所有情报科一系没有任何人质疑李承乾的命令，齐齐躬身退了出去。
<br/>　　长乐的事情发生的太过突然，每个人都知道李承乾必然心情不好，在这个时候别说他只想抓几个人，就是他想要大开杀戒，也不会有任何人意外。
<br/>　　杨天等人都退出去之后，夜魅和小白两个李承乾的贴身护卫却没有离开，两人双双跪倒尘埃：“陛下，昨日之事乃臣等失职，请陛下处置！”
<br/>　　“你们两个起来，昨天的事情与你们无关！”李承乾摆摆手，示意两人起来，语气有些沉重的说道：“长乐的事情朕了有错，你们已经尽力，无需如此！”
<br/>　　夜魅和小白并不意外李承乾会没有迁怒他们，再次施礼谢过之后双双起身，同时小声提醒道：“陛下，几位娘娘在外面已经守了一夜了，您看是不是……”
<br/>　　“让她们都去休息吧，朕这段时间会很忙，让她们该干什么干什么去！”
<br/>　　“是！”小白应了一声，迅速退走，不多时又已经回转，看着沉默中双拳紧握的李承乾说道：“陛下，逝者已矣，您还应该注意身体才是。”
<br/>　　“逝者？”李承乾抬起头：“长乐到底有没有死还是未知，一会儿你们派人出去，小心收敛好长乐的身体，过段时间朕另有安排。”
<br/>　　“陛下……”小白两人几乎以为李承乾疯了，微微变色道：“陛下三思啊！”
<br/>　　“你们不必多言，这件事我自有安排。”李承乾抬手止住两人继续说下去，顿了顿问道：“外面还有谁在？”
<br/>　　“长孙大人，程大将军，几位程小将军还有苏大人，孔大人都在。”小白回答道。
<br/>　　“叫长孙冲进来，其他人让他们都散了，若是有什么事情朕自会找他们。”
