　　这种来历不明的茶李承乾自然是不会喝的，这便是所谓的防人之心不可无！
<br/>　　无聊的等了一段时间，药行大堂的后面传来一阵脚步声，同时伴随着一阵爽朗的笑声：“有贵客光临，鄙行蓬荜生辉啊。”
<br/>　　“杨神医客气了，在下慕名而来，望神医不要觉得唐突才是。”李承乾抱了抱拳，但却没有起身。
<br/>　　本来嘛，他可是堂堂太子，而对方只是一个传说中的骗子，想让他起身相迎完全不够格。
<br/>　　好在那杨常林似乎也不在意这些，回了一礼之后坐到李承乾对面直奔主题说道：“这位小兄弟不知是得了什么病症？”
<br/>　　“头痛！”李承乾指了指自己的脑袋。
<br/>　　“头疼之症却是分很多种了，不知小兄弟到底是哪个位置？”
<br/>　　“哪里都痛。”反正也是来找茬的，李承乾也懒得想到底是哪里痛。
<br/>　　“这……这样的话只怕就要用百消丹了。”这杨常林竟然连脉都不用号，捻着胡子便开始推消自己的百消丹。
<br/>　　“百消丹？真的有神医说的这么神奇？”
<br/>　　“这位小兄弟，老夫还是那句话，谁用谁知道。”杨常林老神在在的说道。
<br/>　　“嗯，既然神医对自己这么有信心……，那就跟本宫走一趟吧！”李承乾见那老灯信誓旦旦的样子，再也不想和他磨叽下去了，面色一肃说道。
<br/>　　“什，什么？”杨常林被‘本宫’二字吓了一跳，小眼睛眨了半天也没想明白李承乾的身份到底是什么。
<br/>　　“东宫就缺杨神医这样的高手坐诊。”李承乾笑着说道。
<br/>　　这下，杨常林只觉得自己要疯了，东宫啊，那可是大唐太子的地盘，岂是他这个草民能进入的？而且重要的是他除了那张嘴之外根本不会任何的医术，如果去了东宫，那岂不是什么都暴露了。
<br/>　　想到这里，杨常林连忙对李承乾说道：“这位小兄弟……”。
<br/>　　“大胆杨常林，太子面前容不得你放肆。”苏猛原本正站在李承乾的身后，现在李承乾既然已经露了底，他自然不用再装。
<br/>　　“太，太……”杨常林不由自主打了一个哆嗦。
<br/>　　“请吧！”李承乾从椅子上站起来，比了一个请的手势。
<br/>　　大唐太子登门求医，按说杨常林应该十分高兴才是，可是现在的他却在两腿发颤，连试了几次都没有从椅子上站起来。
<br/>　　不过苏猛却不管那些，见他不起来立刻走了过来，抓起他的一只胳膊往上一拉立刻将他拉了起来，拖着他就往外走。
<br/>　　“不，不要，我，我不要去皇宫，我，我……”杨常林几乎被吓疯了，想到去了东宫被人发现自己只是一个骗子，丫几乎吓的尿了裤子。
<br/>　　“怎么？杨神医这点面子都不想给本宫？”走在前面的李承乾听见身后杨常林杀猪一样的叫声，扭过头皮笑肉不笑的问道。
<br/>　　“不，不是，我，我，我只是为，为那些看不起病的百姓医，医病，皇，皇亲国戚不，不……”杨常林吞吞吐吐的说道。
<br/>　　“看不起病？就本宫所知，我大唐医病全都是不花钱的吧？你竟然说看不起病？”李承乾脸上的笑容显的十分诡异，看的杨常林心中发毛。
<br/>　　“不，不是，我不是那个意思，我的意思是，我看的都是那些医不好的病。”
<br/>　　“既然这样，本宫的病就是那种医不好的，你跟着本宫来吧。”
<br/>　　“我……”
<br/>　　杨常林还想再狡辩，结果苏猛却一点也不客气，一掐他的脖子，像拎小鸡儿一样将其塞进了马车。
<br/>　　便是这样，杨常林被带回了大明宫。
<br/>　　……
<br/>　　敬事房，李承乾的马车消无声息的停在了门口。
<br/>　　里面的管事立刻从里面迎了出来。
<br/>　　这破地方，就算是高级一点的内侍平时都不会来，现在突然来了大人物，管事如何能不小心接待。
<br/>　　只不过停下来的只是李承乾的马车，从车上下来的是苏猛这汉子。
<br/>　　在敬事房管事目瞪口呆的情况下，苏猛从车上拖下一个小老头，往几个内侍人中推：“管事，这家伙净身。”
<br/>　　“净身？”管事回头看了看头发都有些白了一的杨常林，哭笑不得的说道：“那，那个，这，这位大人，这老头儿岁数太大了，现在净身怕是有些晚了。”
<br/>　　“是么？这么说你是想让我这样去回复殿下？”苏猛乜了那管事一眼。
<br/>　　“呃，不，不不，马上，下官马上给他净身。”
<br/>　　去他大爷的，爱谁谁吧，不就是净个身么，管他是死是活呢，能不得罪东宫那位还是不要得罪的好，管事如此想着，对那几个按住杨常林的内持摆了摆手，示意把人带进去。
<br/>　　“不，不要，救救你们，不要，我，我不想净身，我还没娶老婆，我，我不想净身啊！”杨常林挣扎着喊道。
<br/>　　他可是从头到尾听完了苏猛与那管事的对话，如果再不知道会发生什么，这么多年岂不是白活了。
<br/>　　可是他的挣扎都是徒劳无功的，几个孔武有力的内侍最后还是将他按到了一个铁架子上，三两个扯光了他身上的衣服，然后杨常林便听到一边磨刀霍霍的声音。
<br/>　　此时的他开始后悔了，早知道来长安会有这样的后果，当初就应该在洛阳那边缩着，虽然每天骗来的钱少一些，但总好过被切了那东西吧？
<br/>　　可是现在后悔又有什么用，那管事已经拎着菜刀过来了，一脸阴森的样子看的杨常林肝儿都颤。
<br/>　　“嘿嘿，老家伙，别怕，咱家手快的很，一刀下去你就什么烦恼都没有了。”
<br/>　　“不，不不，别，别这样，我，我可以给你钱，给你很多钱，你放了我，就，就说我已经死了，求求你，放过我！”杨常林在进行最后的挣扎。
<br/>　　“咱家放过你不是不可以，可是你说太子殿下会不会放过咱家？”雪亮的大刀片子在杨常林的眼前晃来晃去，那管事阴笑着问道。
<br/>　　“……”杨常林有些无语了，这样的情况下他真是不知道应该说什么。
<br/>　　而就在他沉默的那一刻，刀光闪过，下面一凉，世界安静了。
