<br/>        介绍：天若有情天亦老，人间正道是沧桑。
<br/>    
<br/>        效果：一身正气，对魑魅魍魉有一定的克制性。
<br/>    
<br/>        看了一眼光环的效果，陈明也是很满意，这个光环，用来对抗鬼仙的势力再好不过。
<br/>    
<br/>        盟主之位是坐上去了，但是还得办事啊，作为盟主，自然不可能什么都自己动手，反正凌仙的戒指里有着仙人级别的老爷爷，就让凌仙去安排这些战略吧。
<br/>    
<br/>        而且目前并没有开启全面战争，双方正在停战期，陈明也没有开启战争的想法。
<br/>    
<br/>        让凌仙装作传达自己的命令安排诸位君主布防，陈明自己下达了另外一条命令，所有正道宗门，必须将自己除了仙级功法以及仙术之外的所有功法术法上交，统一整个大陆的修行体系！
<br/>    
<br/>        普通大君级宗门对此自然是只能选择接受，大君级宗门之下，更是没有选择的余地。
<br/>    
<br/>        而这拥有君主级的八个宗门，就没那么容易了，铁牛谷倒是好说，反正陈明已经将他们的经文看完了，很是畅快地就答应了。
<br/>    
<br/>        不过其余的七大宗门，却是有些不愿意，这些经文，都是他们积累数千年的底蕴，宗门可以轻易示人。
<br/>    
<br/>        七位君主也是找到了陈明，八位君主汇聚一处，各自坐下，重瞳子修为最高，朝着坐在盟主之位的陈明开口道：“盟主，此举，是不是有点不妥？”
<br/>    
<br/>        陈明扫了众多君主一眼，脸上都有着不满之色，陈明心说都什么时候了，能不能活都是个问题，还考虑这些，怪不得赢不了。
<br/>    
<br/>        陈明道：“现在情况有多危急，诸位应该是明白的，若是我们没有成功，那便只有思路一条，你们是准备将这些东西带到棺材里去吗？”
<br/>    
<br/>        重瞳子道：“若是如此，我们数千年的积累，可是毁于一旦了。”
<br/>    
<br/>        陈明笑道：“你们无非是担心这些功法外传之后，你们无法保持如今的地位，其他的那些小宗门，即便是得到了这些功法，他们真的有那么多的资源培养大君吗？”
<br/>    
<br/>        紫天藏
<br/>    
<br/>        道：“虽说如此，可是毕竟是将培养大君的权柄交了出去！”
<br/>    
<br/>        陈明思索片刻，他实在是不想在这群只顾自己的君主上大出血，没有任何用处，这些家伙，贪心不足蛇吞象，根本喂不饱，而且，收取这些经文，必定有宗门会藏私。
<br/>    
<br/>        陈明挥手落下一柄灵剑，然后道：“我回燕山一趟，然后再回来商量。”
<br/>    
<br/>        传送回燕山，陈明找到了掌管炼丹房的老龟，以及燕山大管家白狐，陈明问道：“目前燕山有多少丹药库存，折算成灵石。”
<br/>    
<br/>        白狐道：“折算成灵石的话，应该有十八亿灵石的丹药。”
<br/>    
<br/>        陈明沉思片刻，十八亿灵石，还是有点太少了，现在要收集的，是整个大陆的功法术法。
<br/>    
<br/>        白狐问道：“不知宗主问这个做什么？”
<br/>    
<br/>        陈明道：“我想要用丹药来收购功法术法，不过只有十八亿，应该是远远不够的。”
<br/>    
<br/>        白狐听到陈明这么说，喜道：“宗主，你恐怕是忘了，燕山还有三千五百领主呢，他们的财力，恐怕不下百亿。”
<br/>    
<br/>        陈明道：“我这样用弟子的丹药去收购，不太好吧，燕山弟子真的会愿意？”
<br/>    
<br/>        白狐道：“山主，你太小看你在燕山领主中的号召力了，你可是他们的授业恩师，他们的一切，都是你赐予的！而且，山主收集这些功法术法，还不是要教给他们的？”
<br/>    
<br/>        陈明点点头，然后道：“有理，召集燕山领主，我要询问一番。”
<br/>    
<br/>        目前外界并无战事，燕山领主也都待在燕山修行，燕山的修行环境，可是比外界好多了，听到是陈明召集他们，众多燕山领主齐刷刷地下山，来到燕山本部之下的蒲团上坐好。
<br/>    
<br/>        陈明看到众多燕山领主很快集合，不由很是欣慰，不枉我培养这一群败家子。
<br/>    
<br/>        曾经的少年都变成了青年，一个个都是手握重兵的一番霸主，威势早就不同往日。
<br/>    
<br/>        陈明看向众多的记名弟子，开口道：“为师已经当上了仙门正道的盟主，不过却是还是无法收集整个大陆的功法，
<br/>    
<br/>        所以这一次，要看你们了。”
<br/>    
<br/>        张鸣开口问道：“师父，你就直说吧，要多少丹药收购这些功法术法，反正最后还是我们学的，就交给我们了！”
<br/>    
<br/>        陈明很是高兴地点点头，然后道：“一百亿灵石的丹药！”
<br/>    
<br/>        张鸣思考片刻，每个燕山领主只要出库存的三分之一的丹药，应该就够了，张鸣道：“师父，这些就交给弟子吧，师父喜欢经文，我们就给师父盖一座天下最大的藏经阁，师父一直以来都是不求回报的赐予我们宝物，现在，也到了我们为燕山出力的时候了！诸位师弟师妹，你们说对也不对！”
<br/>    
<br/>        众多燕山领主齐齐起身道：“不过一百亿而已，我们替师父买下来！”
<br/>    
<br/>        陈明欣喜不已，三年的时间，他们已经是成长为了一个个小燕山，“你们有这份孝心，师父很高兴！那师父就去安排这件事了。”
<br/>    
<br/>        数年了，门下的燕山领主终于成长为了他所想象的样子，曾经一个燕山就足以支撑半个大陆的丹药，那么现在，三千多个小燕山，又该有什么样恐怖的能力？
<br/>    
<br/>        现在，陈明得到了答案。
<br/>    
<br/>        再次传送回君主议事殿，陈明看了周围的君主们一眼，认真道：“我燕山，出价值一百亿灵石的丹药收购功法术法，只要是燕山藏经阁如今没有的功法术法，都按照市价收购！”
<br/>    
<br/>        一语道出，七位君主都是震动不已，燕山的财力，到底雄厚到了什么程度？
<br/>    
<br/>        这种财力，也太过骇人了！
<br/>    
<br/>        功法术法现在留着有什么用，又不能直接转化为战力，但是丹药，可是能够直接吃的，七位君主现在哪里不明白，重瞳子第一个起身，“诸位，既然燕山之主都这么说了，我也不好继续这么下去了，在下，先行一步！告辞！”
<br/>    
<br/>        白武霞当即起身怒骂道：“重瞳子，你好不要脸，定是先回去搜集功法了，不要让他占得了先机，燕山有的功法可就不值钱了！”
<br/>    
<br/>        七位君主急匆匆地离开了大殿，已经是回去准备搜集他们觉得不重要的功法送出去了。
