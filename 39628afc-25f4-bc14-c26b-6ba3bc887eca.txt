　　萧逸看向八龙焚火炉，感知了一番。
<br/>　　此时，炉内仍旧火焰熊熊，一双兽手与兽脚漂浮其中，丝毫未损。
<br/>　　在他的对比中，左手的威力，在后面两道封印冲破，出现右手与左脚后，威力显然更强。
<br/>　　如果他没有猜错的话，这妖兽尸骸间，显然有着互相增幅的作用。
<br/>　　也就是说，这些妖兽尸骸的真正威力，应该还被八龙焚火炉内的封印所压制着，并未完全发挥。
<br/>　　若是日后冲破更多的封印，将其余尸骸部分一并幻化而出，其威力必然会成倍增加。
<br/>　　而要冲破更多的封印，自然便要他拥有足够强的实力了。
<br/>　　拥有足够的实力，方能冲破炉内封印；而冲破封印，又能得到其内妖兽部位的增幅，实力更强。
<br/>　　强而逾强，逾强而更强。
<br/>　　毫无疑问，八龙焚火炉，给了他一个极好的提升实力的途径。
<br/>　　“看来，八龙焚火炉内的秘密，我势必要解开。”萧逸自语一笑。
<br/>　　……
<br/>　　“封。”萧逸低喝一声。
<br/>　　八龙焚火炉内，早已装满了金光力量。
<br/>　　若非北隐宗大长老的忽然出现，他早便封炉了。
<br/>　　轰…
<br/>　　早已布置好的冰玄封界阵瞬间发动。
<br/>　　一道道蓝色绚烂光芒，激射而出。
<br/>　　不多时，大阵彻底包裹了八龙焚火炉。
<br/>　　萧逸大手一挥，将火炉收回手中。
<br/>　　细细地看了十数分钟，炉内，浑厚的金光力量无半分要溢出的迹象。
<br/>　　这也代表着，这些一直无可奈何的金光力量，终于被他保存了下来。
<br/>　　那些金光力量，之前萧逸无论是用重宝玉瓶去装，还是用极致寒冰力量去装，最终都会溢散而出。
<br/>　　可现在，这些金光力量存放于八龙焚火炉内，金光力量的走向，乃是往炉内封印而去。
<br/>　　简单来说，之前用其它手段或者东西去装，金光力量会往‘外’走。
<br/>　　而现在，由于八龙焚火炉内有封印，而导致这些金光力量往‘里面’走。
<br/>　　金光力量本来就往‘里面’走，再加上冰玄封界阵的封印，彻底让得这些金光力量不会外泄。
<br/>　　萧逸满意一笑，将八龙焚火炉收回。
<br/>　　这时，目光再次看回金光禁地之下。
<br/>　　此刻这个巨大而深不见底的金光坑洞，早已失去了之前的金光熠熠。
<br/>　　对比之前坑洞之内金光力量浑厚，一眼看下去，毫无黑暗；此刻，早已金光全无，只余黑暗。
<br/>　　这般从上往下看去，整个坑洞，就是一个黑漆漆的瘆人巨洞，似极了一噬人的巨口。
<br/>　　萧逸看了眼，咽了口口水。
<br/>　　就他目力所及的深度下，已再也见不到半分光亮。
<br/>　　这个金光坑洞，确实无比巨大，也深得惊人。
<br/>　　但八龙焚火炉的空间，同样大得恐怖。
<br/>　　那片火焰空间，本就极大；在冲破封印后，又将原本被封印的折叠空间解开，融入其中，让得空间越发巨大。
<br/>　　如此大的空间，被金光力量‘挤’得满满的。
<br/>　　可想而知里头吸收了多少金光力量。
<br/>　　怕是这巨大金光坑洞之下，已无多少金光力量了。
<br/>　　“这下面到底有什么？”萧逸疑惑自语着。
<br/>　　按北隐宗大长老之前的得意之言所说，这金光禁地之下，有大好处。
<br/>　　不过，这些好处无人能得。
<br/>　　既然万金府将这里列为禁地，想来北隐宗大长老的话所言不假。
<br/>　　只是，这金光禁地到底有多深，下面又可能会有什么危险，萧逸一概不知。
<br/>　　这般贸然下去查探，萧逸也需谨慎几分。
<br/>　　“这个金光坑洞，想来存在了无数岁月。”萧逸自语思索着。
<br/>　　“如此长的岁月，却无人能得里头重宝。”
<br/>　　“连中域十八府之一万金府这等存在都只能列为禁地，而无可奈何…”
<br/>　　萧逸皱着眉。
<br/>　　半晌，咬了咬牙，脸色一凝。
<br/>　　“万金府无可奈何，不代表我萧逸也无可奈何。”萧逸眼眸中闪过一丝自信。
<br/>　　只要是能提升实力的途径，萧逸绝不会放过。
<br/>　　多少险地都闯过了，也不在乎多这一遭了。
<br/>　　再者，八龙焚火炉内的一双兽手，给了他极大的自信。
<br/>　　想罢，萧逸脚步一瞪，一个跃身，跳跃而下。
<br/>　　锵…
<br/>　　冷焱剑，凭空而现。
<br/>　　萧逸手中握剑，一道道星光之力溢散而出。
<br/>　　明亮的星光，将他身体为中心，周遭数十米范围内照耀得明亮至极。
<br/>　　萧逸的身躯，不断坠落。
<br/>　　一分钟后。
<br/>　　“已下降万米距离了。”萧逸皱眉自语着。
<br/>　　谨慎如他，自是边下落，边感知距离。
<br/>　　此刻，他已经下落万米距离以上，可下方，仍旧望不到底。
<br/>　　同时，由下而上吹袭而来的罡风气流，则愈发猛烈。
<br/>　　虽然坑洞内的金光力量被吸收光了，但那股原本吹袭金光力量而出的罡风气流却并未消失。
<br/>　　气流，仍旧不断往上吹袭。
<br/>　　而他愈发下降，也愈发感受到这股气流的冰冷和狂猛。
<br/>　　又是一分钟后。
<br/>　　“两万米深度了。”萧逸眉头皱得极紧。
<br/>　　十数秒后。
<br/>　　萧逸的目光，一直牢牢注视下方，忽然反应了过来。
<br/>　　“到底了。”
<br/>　　嗖…萧逸一剑劈出，以剑气抵消了下降的冲力。
<br/>　　萧逸的身躯，稳稳的落到了地面。
<br/>　　“竟足足有两万米深度。”萧逸面露惊色。
<br/>　　抬头看了眼坑洞上方，完全看不到坑洞洞口，甚至连半分光亮都看不到。
<br/>　　“嗯？”这时，萧逸看向了前方，脸色再次一惊。
<br/>　　前方，有一数十米高度的洞口。
<br/>　　洞口之处，恐怖而剧烈的罡风，不断吹袭而出。
<br/>　　“原来如此。”萧逸面露恍然之色。
<br/>　　这些狂猛的罡风气流，根本不是从坑洞底部吹出。
<br/>　　而是从面前这一洞口吹出，随后经过坑洞底部，吹袭而上。
<br/>　　这时，萧逸明显看到，前方洞口处，一缕缕浑厚金光飘荡其中。
<br/>　　看来，那些每晚便飘荡于金光险地各处角落的金光力量，便是从这个洞口中喷出的。
<br/>　　萧逸看了眼周遭。
<br/>　　地面上，遍地尸骸。
<br/>　　有的，是铮铮白骨，显然已死千百年以上。
<br/>　　有的，是半腐烂尸体，顶多死了数十年左右。
<br/>　　有的，尸体保存极好，看来是这些年才死于金光禁地之下的。
<br/>　　萧逸大致检查了一番，眉头微皱。
<br/>　　这些尸体，有的是被洞穿的伤势所致命，有的，则是重伤而亡。
<br/>　　但有的，却是带着千刀万剐的骇人痕迹。
