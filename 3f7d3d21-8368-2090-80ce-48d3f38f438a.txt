　　四级巅峰妖兽，实力介于洞玄九重与半步破玄之间。
<br/>　　比洞玄九重强，比半步破玄弱。
<br/>　　正常来说，同境界的人类武者，绝不是妖兽的对手。
<br/>　　所以才有了武技、灵器、短暂增幅实力的丹药等等。
<br/>　　柳苍崖等十八城的城主，宁浩等猎妖师队长，皆是洞玄九重。
<br/>　　一出手，便是玄阶巅峰武技。
<br/>　　轻松压制住各自对付的四级巅峰妖兽。
<br/>　　两个猎妖殿执法队队长，虽是半步破玄，但对付20头四级巅峰妖兽，难度还是很大的。
<br/>　　幸好，慕容克终究是没有脸只有自己不出手，便加入了他们的战斗。
<br/>　　三个半步破玄，力敌20头四级巅峰妖兽，倒是不相伯仲。
<br/>　　至于萧逸这边，百丈之宽的紫炎双翼，轻松拦下50头四级巅峰妖兽。
<br/>　　叶铭也是实力尽出。
<br/>　　两人联手下，应对着更多的妖兽，却比其他人更为轻松。
<br/>　　“火树银花。”叶铭大喝一声。
<br/>　　数十颗巨树凭空而现，树中须叶无数。
<br/>　　须叶看似轻柔，却坚硬无比，顷刻间将妖兽们束缚住。
<br/>　　下一秒，数十颗巨树轰然冒出一簇簇青色奇火。
<br/>　　青色奇火，看似清幽美丽，实则威力骇人。
<br/>　　一阵噼里啪啦声，须叶自动爆燃，被束缚的妖兽尽皆被烧成重伤。
<br/>　　“叶兄，厉害。”萧逸笑道。
<br/>　　他自然看得出来，单单叶铭现在展露的实力，已经比慕容克要强。
<br/>　　真不愧是药堂首席，不过是洞玄九重修为，实力已经比半步破玄要强得多。
<br/>　　也难怪叶铭在内堂各首席的排名中，一直是第一。
<br/>　　叶铭，修的是一门地阶的木属性功法。
<br/>　　刚才的数十颗巨树，就是功法上的招式。
<br/>　　而他本身，乃是青色武魂‘空青火’。
<br/>　　这种火焰，很是奇特，乃是火木双属性。
<br/>　　火焰为主，木属性为辅。
<br/>　　本身的攻击力并不大，但遇木即爆炎。
<br/>　　也就是说，任何木属性的东西，都能让它发挥出十倍以上的威力。
<br/>　　在炼药上，空青火，无疑能让炼药师对材料的把握和感知更为精准。
<br/>　　融化材料时，也因此而让火焰温度凭空大涨。
<br/>　　被炼药师誉为最不可多得的炼药火焰之一。
<br/>　　“易兄过奖了，你可比我厉害得多。”叶铭谦逊地说道。
<br/>　　当然，事实也确实如此。
<br/>　　漫天紫炎之下，哪怕是四级巅峰妖兽，也是触之即死。
<br/>　　大概半个时辰后，两人联手，将50头四级巅峰妖兽尽皆斩杀。
<br/>　　同一时间，其他人对付的四级巅峰妖兽，也齐齐毙命。
<br/>　　毕竟，除了萧逸和叶铭外。
<br/>　　其他人，如十八城城主、各猎妖队队长，都是东荒十八城中的狠人。
<br/>　　他们，都是同境界武者中的精英。
<br/>　　上百头四级巅峰妖兽，已然死亡。
<br/>　　另一边，数万人类武者的战斗，始终落于下风。
<br/>　　毕竟，妖兽的数量太过庞大。
<br/>　　萧逸等一众强者，再次加入战斗。
<br/>　　他们面对普通的四级妖兽，以及三级妖兽，几乎都是秒杀。
<br/>　　往往一挥手，便是大片妖兽死亡。
<br/>　　“按这个情况下去，最终失败的，必然是这些妖兽。”叶铭笑道。
<br/>　　“绿光城，算是保住了。”
<br/>　　“这次的兽潮，也该告一段落了。”
<br/>　　柳苍崖等城主松了口气。
<br/>　　“嗯？”这时，叶铭疑惑道，“易兄，怎么愁眉苦脸的？”
<br/>　　萧逸，一直都没有说话，脸色凝重。
<br/>　　嗖，紫炎火翼一振，数十头妖兽被烧成灰烬。
<br/>　　幻羽之翼就算幻化出来，本身也没有多少攻击力。
<br/>　　但由萧逸幻化的紫炎火翼则不同。
<br/>　　紫晶灵炎，本身就自带焚烧万物的特性。
<br/>　　妖兽被沾上，转瞬间就会被烧成灰烬。
<br/>　　“我总感觉，这次的兽潮，没那么简单。”萧逸沉声说道。
<br/>　　“易兄怕是多虑了。”叶铭笑道。
<br/>　　“易霄执事，事情还是往好的方向想为好。”
<br/>　　“您虽然戴着面具，但我看你的年龄并不大，只是个毛头小子，哦，不，年轻人。”
<br/>　　“怎么说话和想法都如此老成？”
<br/>　　猎妖殿的两位执法队队长打趣道。
<br/>　　“哎。”萧逸叹了口气，自语道，“希望我猜错了吧。”
<br/>　　说罢，再次奋力斩杀妖兽。
<br/>　　不过半个时辰，死在众人手中的妖兽，已有过万之数。
<br/>　　形势，一片大好。
<br/>　　忽然，‘吼’的一声。
<br/>　　那道震天兽吼再次出现。
<br/>　　“兽吼出现了，妖兽要退去了。”柳苍崖笑道。
<br/>　　“它们也不是傻子，要是还不退，最终只会被我们杀光殆尽罢了。”宁浩等人笑道。
<br/>　　“哼，一群废物，我还未杀够呢。”慕容克得意地笑着。
<br/>　　“有本事别逃。”慕容克叫嚣着。
<br/>　　众人一片欢天喜地。
<br/>　　然而，下一秒，他们的脸色却猛地大变。
<br/>　　因为，十数万妖兽，在兽吼声出现后，根本没有退去，反而越加凶猛。
<br/>　　“乌鸦嘴。”萧逸骂了一声，心头涌出一道不好的念头。
<br/>　　“遭了，不单止没有退，反而越来越多了。”柳苍崖看向远方，惊声说道。
<br/>　　远方，妖兽森林内，妖兽不仅没有退，反而是越来越多的妖兽成千上万地快速涌出。
<br/>　　不过半晌，场上的妖兽，已经激增到20万之数。
<br/>　　“我…”慕容克恨不得抽自己两耳光。
<br/>　　“事出反常必有妖，大麻烦要来了。”萧逸冷峻的目光，直直盯向妖兽森林。
<br/>　　同时，他蓦然地感觉到一丝不对劲。
<br/>　　但又说不出来这丝不对劲来自于哪里。
<br/>　　“吼。”又是一声震天兽吼。
<br/>　　“嗯？”萧逸眉头紧皱，他觉得，自己似乎知道哪里不对劲了。
<br/>　　“又是兽吼声。”柳苍崖急道，“怎么回事？按照之前的规律，兽吼声出现，妖兽应该退去的。”
<br/>　　“如今两次兽吼声，妖兽却越来越多。”
<br/>　　一众强者的脸色，都变得凝重无比。
<br/>　　数秒之后，一股炽热的气息，忽然从远方妖兽森林传来。
<br/>　　这炽热气息，让得整个空气，仿佛都燃了起来。
<br/>　　“好热。”柳苍崖满头大汗地说着。
<br/>　　“我们武者，特别是洞玄境这个层次，已然寒暑不侵。如今真气护体下，竟挡不住这些热浪。”
<br/>　　宁浩咬了咬牙，显然也意识到事情不对劲。
<br/>　　再过了数秒，那炽热气息，忽然席卷而来，宛如一道滔天火浪。
<br/>　　所过之处，树木枯萎，泥土焦黄。
<br/>　　数万武者的兵器，甚至出现了融化的迹象。
<br/>　　一众强者，竟在这股火浪之下，直接被震飞数十米。
<br/>　　“好强的气势。”萧逸大惊失色。
<br/>　　“你们看，那些妖兽…”叶铭脸色大变。
<br/>　　近20万妖兽，竟然齐齐匍匐跪下。
<br/>　　它们…仿佛在迎接属于它们的王者。
<br/>　　果然，下一秒，一道泛着滔天火焰的巨大身影，从天而降。
<br/>　　身上火焰，一瞬间将周围百米土地烧成灰烬。
<br/>　　“烈焰狂狮，该死，这可是破玄境妖兽，而且还是破玄境中的一霸。”猎妖殿的人员脸色变得无比灰丧。
<br/>　　“这几乎是妖兽森林最深处的霸主了。”
<br/>　　“连破玄九重武者，看到它都要狼狈逃命。”
<br/>　　宁浩等猎妖队队长只觉心惊胆寒。
<br/>　　“绿光城，完了。”柳苍崖呆愣住，双眼中毫无希望。
<br/>　　“不，整个东荒十八城都完了。”十八城城主面如死灰。
<br/>　　“不，还没完。”萧逸淡淡地站前一步。
<br/>　　沉稳的声音中，透露着不屈的坚毅。
<br/>(本章完)
