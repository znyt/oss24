        正在行驶中的加长型豪华轿车骤然停了下来，童筱颖未坐稳身子，直接被从沙发上甩了下来。
<br/>
<br/>        当她吃痛地坐在车厢地面时，车门忽然开了。
<br/>
<br/>        童筱颖连忙起身，弯着腰走了过去。
<br/>
<br/>        她一下车，身后这辆加长型豪华轿车突然重新启动，在还未等她缓过神来，车子便向着下山的路扬长而去。
<br/>
<br/>        刚刚还是晴空万里，倏地，乌云密布，天空划出一道骇人的闪电。
<br/>
<br/>        童筱颖环顾了一下四周，后方是一条下山的柏油路，那辆加长型豪华轿车早已湮入了尽头的黑暗之中，两边则是盘根错节的大树高耸入天。
<br/>
<br/>        而前方，则是一片树林，似乎深不见底。
<br/>
<br/>        童筱颖柳眉微蹙，仰头望着被树叶、树枝遮蔽的天空，接着一连串的闪电照亮了天空，更凸显出了前方这一片森林的诡异。
<br/>
<br/>        这是什么鬼地方？！
<br/>
<br/>        不会是大哥给她的什么惊醒吧？
<br/>
<br/>        童筱颖下意识地环抱双臂，莫名其妙地起了一身鸡皮疙瘩。
<br/>
<br/>        她能感觉到暴风雨似乎即将来临……
<br/>
<br/>        往前走？
<br/>
<br/>        还是往回走？
<br/>
<br/>        转眼间雷电交加，暴风雨果然来了。
<br/>
<br/>        一道闪电从她的头顶上飞掠而过，伴随而来的雷声几乎使她耳鸣。
<br/>
<br/>        童筱颖心一横，决定只身往那片树林里跑去。
<br/>
<br/>        “哗——”地一声，大雨倾盆而至。
<br/>
<br/>        童筱颖一直往前跑，不知道跑了多久，雨水从枝繁叶茂间落下，打湿了她的长发和身上的衣裙。
<br/>
<br/>        树林里的树排列整齐，像是人工种植，虽然抬头不见天空，但有一条笔直的石子路是通往前方，所以当她跑到一半时，前方便传来了一道亮光。
<br/>
<br/>        童筱颖缓了口气后，接着一股脑儿地跑出了树林。
<br/>
<br/>        眼前一片光明的时候，一幢像哥特式古堡的建筑赫然屹立在视野中。
<br/>
<br/>        石头堆积的麻灰色外墙，复古的玻璃窗，错落有致的放哨塔，前方还有一条护城河。
<br/>
<br/>        童筱颖仰起头来，骇然地望着眼前这幢突兀的古堡，在风雨雷电下，古堡更加诡谲。
<br/>
<br/>        突然“嘎吱”一声，伴随着铁链转动时所发出的金属碰撞声，古堡的大门徐徐放下。
<br/>
<br/>        童筱颖心生畏怯，始终裹足不前，雨水从她的头顶流下来，她浑身湿凉，背后寒意来袭，驱使她不得不继续往前走。
<br/>
<br/>        进入拱形大门后，周遭是城墙围起来的一块空旷的前坪，前方还有一张大门，门边蹲坐着两只石狮。
<br/>
<br/>        童筱颖走到这正大门前，当她举起手来欲敲门，身后又传来那“嘎吱”声和铁链转动的金属碰撞声。
<br/>
<br/>        身后的那张门已经关上，她别无选择地抛开所有顾虑，敲响了眼前的这张大铁门。
<br/>
<br/>        大门豁然打开，一个身着墨蓝色女仆装，戴着耳麦的年轻女人出现在了童筱颖的视野里。
<br/>
<br/>        年轻女人看到童筱颖的时候，惊怔地瞪大了眼睛。
<br/>
<br/>        童筱颖也感到难以置信地看着面前的这个年轻的女人。
<br/>
<br/>        “雅珊，先带她去沐浴。”
<br/>
<br/>        就在两人面面相觑的时候，倏忽间，一道凌厉又低沉的声音突兀地在季雅珊的耳麦里响起。
<br/>
<br/>        季雅珊接到命令后，收回了思绪，嘴边牵出一丝微笑，温和地说道：“请跟我来。”
<br/>
<br/>        童筱颖仍旧惊讶地看着季雅珊，刚张嘴想要询问她是什么情况的时候，只见季雅珊凝了凝眉头，眸光微沉，似乎是暗示她先不要乱说话。
<br/>
<br/>        季雅珊随之淡定从容地转过身去。
<br/>
<br/>        童筱颖不得不闭上嘴，默默地跟在了她的身后。
<br/>
<br/>        果然，她真的是季雅珊！
<br/>
<br/>        童筱颖完全没料到，三年前，那个跟她深交要成为彼此挚友，为了“夜惑”里的十爷，却突然间音讯全无的季雅珊，竟然会出现在这里。
<br/>
<br/>        季雅珊领着童筱颖去了二楼尽头，她推开面前的房门，向童筱颖迎了迎手，毕恭毕敬地说道：“这个房间里有浴室，你先去泡个热水澡，免得着凉。”
<br/>
<br/>        “谢谢。”童筱颖讷讷地向季雅珊点了下头。
<br/>
<br/>        季雅珊接着说道：“我去给你拿衣服。”然后，她转身离开了。
<br/>
<br/>        童筱颖进去后，才发现，这是个古典欧式皇室风格的房间，她正打量着这里，没过一会儿，又传来了一阵轻轻的敲门声。
<br/>
<br/>        “请进。”童筱颖应道。
<br/>
<br/>        季雅珊手捧着干毛巾和新衣服，再次出现在了童筱颖的面前。
<br/>
<br/>        她亲切中又透着一丝疏离，微笑着对童筱颖说道：“你好，我是雅珊，此刻起，是你的女侍，有什么需求，你可以随时跟我说。”
<br/>
<br/>        “我……”
<br/>
<br/>        “你心里的疑问，待会八爷自会告知，还请小姐，先沐浴更衣。”季雅珊直接打断了童筱颖的话。
