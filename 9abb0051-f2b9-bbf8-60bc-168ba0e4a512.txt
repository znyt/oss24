　　叶少阳点点头，问道：“毛师兄，刚那个僵尸，是什么东西，你对僵尸有研究，见过没有？”
<br/>　　毛小方仔细想了想，道：“僵尸有太多种，常规的有干尸血尸水尸飞僵尸等等，这个你也明白，但不同形态之下，会产生很多非常规的僵尸，没有轨迹可循的，刚那只身上有尸气，严格说肯定是僵尸，但是怎么形成的，我不清楚，不过……”
<br/>　　“不过什么？”叶少阳追问道。
<br/>　　毛小方想了一会，说道：“身上没有皮毛，一般都与血有关，而且它脸上爬进爬出的那是尸虫，尸虫在地面上是无法长期存活的，所以我推断，这僵尸一定是来自地下。”
<br/>　　来自地下……叶少阳想起那些鬼魂，也都是从地下冒出来的，显然，那僵尸虽然难对付，但跟那些鬼魂一样，也只是一个仆人的角色，真正推动这一切的，是另外有一股神秘的力量。种种迹象标明，这山谷的地下一定有什么东西。
<br/>　　叶少阳道：“别的一会再说，我们还是说刚才的话题，我觉得有两种可能，一种可能就是，这道山谷，是那个躲在背后的神秘邪物的封印之所，有什么力量把它封印在了山谷之内，不能逾越山谷一步。”
<br/>　　毛小方缓缓点头，道：“有这可能。”
<br/>　　在法术界之中，这种事是经常会有的，有的邪物，是因为本身位列仙班，或者有不死之身（如尸王后卿），法术界也是没有办法，只能强行封印起来，但是更多的情况是，有些邪物没有不死之身，但实力太强，没有办法强行灭杀，只能退求其次，将一小块地方都封印起来，先让它不能出去闹事，别的将来再说。虽然在很多机缘巧合之下，有一些邪物突破了封印，重现人间，但是当初封印它们，实在也是无奈之举。
<br/>　　“还有一种可能，当然可能性不大，”叶少阳朝身后不远处的几座山峰看了看，吐出四个字：“势力范围！”
<br/>　　毛小方一惊：“势力范围？”
<br/>　　势力范围，是法术界的一个专用名词，从古代就有了，这背后，是一个有关法术界——确切说是邪物之间的规则。
<br/>　　邪物之间的关系，就像自然界的动物一样。在华夏的山川河流之后，有很多天地灵气充盈的地方，或者是阴盛阳衰的地方，大到一座山，小到一间死过人的房子，这些地方，往往会被一些邪修的邪物占据，因为资源有限，当然不想跟别人分享，所以一旦有新的邪物来抢夺，那就要打一架，打赢的留下，打输的就跑路，跑不掉的就会被灭掉。
<br/>　　邪物之间，与人类最大的区别，就是不会结盟，最多就是自己收一帮小弟，或者投靠强者。但同等级之间的邪物，绝对不会联手，一般都是闷头各自修炼，但人间法术界，则是互相帮忙，逐个击破。否则如果全世界的邪物联合起来，组成一个类似法术界一样的联盟，在人间不知道会掀起什么样的惊涛骇浪。
<br/>　　至于鬼域，无极鬼王走的就是这条路，势力极大，不过仅限于鬼域，对人间没什么影响。
<br/>　　如果一块地方，被先后两个实力差不多的邪物看上，互相又没法灭杀对方，但又都出于一些原因没办法也不想去寻找新的巢穴，那就会达成一种共识：双方各自占领一块地方，划清界限，双方都不能逾越界限。这就像是自然界中凶兽的领地一样，一旦侵犯别人领地，那又是不死不休的缠斗，这种战斗，对双方来说都会影响修炼，还容易节外生枝，被法师盯上。
<br/>　　所以除非有什么意外，正常情况下，邪物一旦定下了势力范围的界限，就绝对不会逾越。
<br/>　　毛小方想到这里，问叶少阳：“你意思是说，这山谷的出口，是势力范围的边界？”
<br/>　　叶少阳道：“有这种可能，之前我们都检验了，山谷里有没有结界，不知道地下有没有。”
<br/>　　毛小方站起来，朝远处几座山望过去，说道：“难道这附近还有什么邪物不成？山谷里的邪物已经够强大了，这邪物能跟他划界而治，肯定也很了不得。”
<br/>　　叶少阳道：“不管了，我们先离开这，今晚上莽撞了，也是对方先没发现我们，不然我们可能真的出不了这山谷。”
<br/>　　七星龙泉剑的灵力，是吸收日精月华积累的，用了一次，已经耗费大半，在几个小时之内再次使用的话，效果会大打折扣。
<br/>　　毛小方知道这附近有个镇子，带着叶少阳赶过去，先找个地方住一晚再说。
<br/>　　因为之前山路不能骑马，两人都把马给放了，因此走了两个小时的山路才走到毛小方说的镇子。
<br/>　　跟一切山里的世界一样，这个小镇也是附近所有村子的核心，虽然实在小了点，而且太过偏僻，在叶少阳的眼中，看上去跟古代似的。
<br/>　　镇上有一家小客栈，毛小方上次在这住过一回，认识路，带叶少阳找过去，找了个房间住下，并且让老板弄了点饭吃。
<br/>　　在房间里，两人商量了一下，直接调查山谷是行不通了：虽然明知道那股神秘力量藏身在山谷下方，但是找不到具体的位置，也是没有办法。如果是一百年后还好说，叶少阳完全可以找个挖掘机之类的过去开挖，但是这时代完全行不通，又不能雇人去挖地，否则太危险了。
<br/>　　想来想去，还是从侧面去打听。这也是法术界的常规手段：既然那座山谷里有邪物，住在这附近的山民不可能毫不知情，至少也会知道一些传说，两人决定先打听看看再说。
<br/>　　回到自己房间之后，叶少阳先想了一下在山谷中的经历，确定这不是一般的灵异事件，别说自己现在只有真人的实力，就算是全盛时期，也不一定能应付得了，不过，既然摊上这件事，以自己的性格自然不可能不管。也只能一步步去调查了。
<br/>　　这也是自己来到这个世界之后，遇到的第一个大麻烦。叶少阳觉得一切都好，就是身边没什么帮手，毛小方算一个，不过还是太少了。
