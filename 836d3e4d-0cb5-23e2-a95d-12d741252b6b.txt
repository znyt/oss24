　　望着拍卖台上，散发出浓郁药香的七彩色果实，慕清霄眼中闪过一丝笑意，确实是六品灵果‘七彩韵灵果’，比许多七品丹药的价值都高。
<br/>　　一枚灵果，塑造一位天才。
<br/>　　七彩韵灵果效果虽好，但实力到达斗王后，根基都已经完全奠定，资质根本不是六品灵果能改变的。
<br/>　　如果硬要服用七彩韵灵果，并不会有效果，所以只对斗王以下的修炼者有效。
<br/>　　否则，七彩韵灵果可不仅仅六品这么简单了，拍卖行也不可能将如此逆天的灵药拿出来拍卖。
<br/>　　此时，拍卖场中已经恢复宁静，几乎九成的客户，双眼中都满含贪婪与激动，六品灵果，完完全全算得上是本次拍卖会的压轴物品。
<br/>　　至于更强大的势力，一枚七彩韵灵果，还不足以让他们动容，毕竟只有斗王以下的修炼者能用，这种势力，根本不缺所谓的天才。
<br/>　　望着拍卖台下的视线，老者取出一个水晶玉盒，快速将七彩韵灵果收了进去。
<br/>　　“此玉盒乃是由千年寒冰炼制而成，内部可完美保存灵果，阻止药效流逝，毕竟灵药离开媒介后，往往都会流逝一些药效...七彩韵灵果，竞拍价三百万，现在开拍。”
<br/>　　三百万金币，已经是天价，要知道这仅仅是起拍价，定音锤不落下，谁也不知道要花多少金币，垂涎七彩韵灵果的修炼者，瞬间就被刷下去八成。
<br/>　　其中有一部分，是想买却不敢买的，正所谓，有钱买也要有命用，命都没了，还要七彩韵灵果有何用，这只是指一部分有钱无权弱者。
<br/>　　唯一留下的，也只有黑角域中颇为强大的势力，也只有他们有这种资本，当然，也不乏有修炼者暗中觊觎，只不过成功率微乎其微。
<br/>　　随着七彩韵灵果的出场，拍卖会的气氛已经进入了高峰期，某些大势力也开始竞价，至于价格自然让大部分修炼者望而却步。
<br/>　　贵宾室中，慕清霄喊来站在一旁的待女，将一枚普通的空间戒指递了过去，道：“将戒指给你们的鉴定师，其中的东西给我一个合理的价格。”
<br/>　　接过纳戒，待女恭敬的行礼，随后快步的走了出去。
<br/>　　此时，拍卖已经进行到了白热化的程度，七彩韵灵果起拍价虽高，但老者手中拍卖锤落下的时候，价格就开始飙升。
<br/>　　仅仅几分钟的时间，竞拍价就飙上了四百万，某些大势力的强者买七彩韵灵果，为的恐怕就是给自己的后代服用，或者是给门下优秀天才，拉拢人心。
<br/>　　拍卖场中竞价丝毫没有停歇，七彩韵灵果的价格也在节节攀升。
<br/>　　恐怖的价格使得许多人望而却步，即便清楚他们没有得到的可能，但看到大势力挥金如土，羡慕的同时，也觉得不虚此行。
<br/>　　半刻钟过去，价格攀升的同时也逐渐接近尾声，而七彩韵灵果的价格也达到了令人膛目结舌的地步，七百八十万的天价！
<br/>　　七百八十万的巨款，可是相当于乌坦城萧家，近十年的收入，哪怕在黑角域，某些大势力，也未必能洒脱的拿出来。
<br/>　　甚至，绝大多数修炼者，终其一生都不可能赚到如此多的金币！
<br/>　　价格到了这一步，终于有一些势力开始放弃，而一级贵宾包厢中，待女已经去而复返，恭敬道：“先生，本行愿意以一千万金币收购您纳戒中的所有物品。”
<br/>　　“成交。”
<br/>　　慕清霄语气平淡，似乎一千万金币对他来说，不值一提，如此轻描淡写，以至于旁边的待女都愣了几秒。
<br/>　　回过神来，待女脸上露出一个艳丽的微笑，眸子中甚至有一丝挑逗的意味，随后递过去一张卡片。
<br/>　　卡片上的图案与先前的一样，只不过颜色却从绿色换成了黑色。
<br/>　　“先生，这是本行的顶级贵宾卡，在本行所拍物品，一律只抽取1%的税，能够享用顶级贵宾专属包厢，先前所拍物品的价格，也全都结入卡中，顶级贵宾卡在全大陆都通用。”
<br/>　　接过卡片，慕清霄微微点头，并没有回应待女的挑逗，而是瞥了一眼卡片。
<br/>　　说是顶级贵宾卡，倒不如说是金币储存卡，只不过换了个名头而已，此卡他曾经在萧家得到过一张，只不过数额比较小，只有五千。
<br/>　　几分钟后，七彩韵灵果的价格已经到了八百二十万，竞拍者是黑角域，黑骷墓势力的一名灰袍中年。
<br/>　　寂寞无声的拍卖场在持续几分钟后，一道道目光相互对视，至于其余势力也都没有在加价。
<br/>　　倒不是他们拍不起，而是八百万已经是七彩韵灵果价值的极限，若是在继续加价，绝不是明智的选择。
<br/>　　黑骷墓将价格提高至此，明显让黑角域另外几大势力感到意外。
<br/>　　而且，拍卖会并没有到最后，也就代表后面还有压轴物品，价值肯定比七彩韵灵果要高。
<br/>　　物品的信息，他们多少有些内幕，与其将重金花在七彩韵灵果上，倒不如省下来，竞争接下来要拍卖的物品。
<br/>　　血宗少宗主，范凌眼神阴冷的瞥了眼灰袍上绘有骷髅头的中年，手掌轻轻颤抖几下，微微垂头，眼中闪过一抹狰狞与杀意。
<br/>　　虽然他是血宗少宗主，但资质也就一般，几十年下来，利用他老爹的资源，修为也堆到了大斗师境界，但依旧改变不了，他资质一般的事实。
<br/>　　但有了七彩韵灵果，他就能改善资质，彻底成为天才，可七百五十万，已经是他的极限，却没想到灰袍中年如此不给面子。
<br/>　　就在老者准备敲下定音锤的时候，二楼一级贵宾包厢中，忽然响起一道颇为平淡的竞拍声。
<br/>　　“八百二十一万。”
<br/>　　霎时间，拍卖场陷入短暂的寂静，寂静过后，所有人的视线都汇聚在贵宾包厢中。
<br/>　　他们是想看看，除了各大势力的强者，这一级贵宾包厢中，到底谁不知死活，敢与黑骷墓硬钢！
<br/>　　只是下一秒，几乎所有人都愣住了，这特么不就是刚才，竞拍残缺地图的黑袍傻吊么？
<br/>　　坐在软椅上，慕清霄摸了摸小医仙细腻的柔荑，心中一片温柔，七彩韵灵果对这小妮子来说，是相当不错的灵药。
<br/>　　为博佳人一笑，一掷千金又何妨。
