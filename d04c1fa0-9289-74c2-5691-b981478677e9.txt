　　第600章
<br/>　　程大雷气得直想骂娘，自己没招谁没惹谁趴在屋顶上，正听燕不归师徒二人的八卦听得认真。冷不防燕不归突然向自己出手。
<br/>　　他确实小看了燕不归和白无双师徒，两位绝世高手，都有不错的耳力。双反在谈话过程中，已发觉了程大雷，虽然还无法判断出对方的身份，但敢孤身闯戎族王宫的，满天下也并没有几人。
<br/>　　哗啦啦声响，燕不归撞破屋顶，一眼就认出了程大雷。仇人相见，他的牙齿恨不得咬碎，口中怒喝：“程大雷。”
<br/>　　程大雷根本没搭理他，撒丫子就往外逃。他和燕不归交过手，知道这人了得。而他还有个没露面的师父，想也知道比燕不归只强不弱。两大高手合力对付自己，程大雷讨不到半点便宜。况且，这可是戎族王宫，最不缺的便是擅长箭法的戎族汉子。如果自己被逼到角落，到时候万箭齐发，白衣老尼犯过的错误，程大雷可不想再犯一次。
<br/>　　于是，程大雷的第一个念头就是逃，他的速度很快，轻身功夫施展到极致。眨眼间便要逃出王宫。
<br/>　　“想走，留下吧。”
<br/>　　凌空一掌拍了下来，那白无双不知怎地，竟出现在程大雷面前。很显然，比轻身功夫，他要比程大雷更快。
<br/>　　戎族也是藏龙卧虎，倒是不可小瞧了天下英雄。程大雷身子坠地，就势一滚，沿着墙角疾奔。此刻，他的匹夫剑已经出鞘，宛若黑夜中的鬼魅，一心向外逃去。
<br/>　　此刻，整个王宫已被惊动，幸运的是，王宫内并没有多少侍卫。他毕竟不像大武皇帝，身边有三千鱼龙卫。不幸的是，这些人中有一个走火入魔的燕不归。
<br/>　　燕不归一心一意就是要杀掉程大雷，所以他一直缀在程大雷身后。也就是仗着轻身功夫好，程大雷并没有被他赶上，可对于这座陌生的王宫，程大雷很多路径并不熟悉，于是也无法将燕不归甩脱。
<br/>　　整座王宫从睡梦中被惊醒，对程大雷展开围杀。火把点燃，将整座王宫照得亮如白昼。
<br/>　　无形间，这帮助了程大雷，他借着光线，已看到一条向外的通道。一柄羽箭向程大雷背心扎来，程大雷挥手接住，转身就是一招百步飞剑。
<br/>　　不说是必杀一剑，但威力绝对不容小觑。燕不归原地跌倒，才堪堪将这一箭避开。
<br/>　　也就是靠这一箭之力，程大雷才与燕不归拉开距离。窜到宫墙前，眨眼就要离开王宫。
<br/>　　那白无双又出现在程大雷头顶，一掌朝着他天灵盖拍下，出手稳准狠，就是要程大雷的性命。
<br/>　　当年，青叶佛孟知了也靠这一招拦住了程大雷，但今天的程大雷可不是当初，他一剑上撩，砍的是白无双的喉咙。对方不容情，程大雷更没有留情的意思。
<br/>　　“老不死的，给我滚。”
<br/>　　这一剑，实在是程大雷一生巅峰，着实是逼急了程大雷。今天要闯不过白无双这一关，程大雷的性命也得交待在此地。
<br/>　　便是白无双这种已修成老妖怪的人物，也不敢硬接这一剑，只能选择避让。
<br/>　　程大雷纵身跃下宫墙，再不做丝毫停留，逃入了夜幕当中。
<br/>　　出得王宫，其他人想要跟上程大雷的脚力已不容易，只有燕不归带着几人追了出去。
<br/>　　“发生什么事了，发生什么事了？”
<br/>　　衣衫不整的呼延力走到外面，他今天刚刚成亲，正在床上安慰着美娇娘，冷不防发生这种事。当然，这也不是呼延力第一次成亲了，戎族人都有个好身体，妻妾成群，子孙无数。
<br/>　　白无双从墙头飘飘然落下，周围人暗暗吃惊。平时大长老都是以智慧服人，虽然教导出燕不归这样的徒儿，但白无双亲自露功夫的机会并不多。大家还是第一次知道，大长老身上还有这样的绝技。
<br/>　　“王上放心，不过是个蟊贼，已被赶走了。”
<br/>　　“什么蟊贼，长老认得他是谁么？”呼延力问。
<br/>　　“似乎是帝国通缉的要犯程大雷。”白无双顿了顿道。
<br/>　　“是他。”呼延力一惊，白毛汗都要吓出来。当初就是这程大雷刺杀了草原王，自己才有机会成了部落的王。怎么，他又过来搞暗杀，这混蛋对刺杀有瘾么。
<br/>　　“长老，你可一定要保护我，别让我再被杀了。”
<br/>　　“王上放心，他这次被发现，想胆子再大也不敢来了。”
<br/>　　“也不好说呐。”呼延力长叹一声。
<br/>　　“王上，我正有一事要向您禀报。”
<br/>　　呼延力被这件事一搅，已没有半点睡意。他坐在大殿内打着哈欠，问道：“这里只有我们两个了，大长老有什么要紧事呢？”
<br/>　　“属下刚得到消息，北蛮部向我族求援，合力消灭突然冒出来的火之部落。”
<br/>　　“长老的意思，咱们帮不帮他们？”
<br/>　　“如今这件事，对我们是个机会。”白无双道：“我们可以趁着这个机会，吞灭北蛮部，一统草原。随后，率领数十万铁骑南下，便可一举攻占帝国，占了帝国花花江山……到时候称孤道寡……”
<br/>　　白无双说着，眼中已有兴奋的神色。却见呼延力坐在那里，不停打着哈欠。
<br/>　　“长老，咱们要攻打帝国？”
<br/>　　“那当然是最后一步，首先要做的，还是统一草原。”白无双道：“一步一步计划，如果顺利的话，十年之内当成全功。”
<br/>　　“咱们为何要攻打帝国？”
<br/>　　“这……”白无双愣在那里，一时竟不知该如何回答。
<br/>　　“长老，如果咱们缺衣缺粮，可以去帝国抢么。但说打下整个帝国……是不是太麻烦了些。”呼延力认真思考着，他当然也觊觎帝国的花花江山，美女如云。可用十年时间去做一件事，是不是值得。毕竟，戎族的美女也不少么。
<br/>　　“这件事再议，再议……”呼延力站起身：“长老也早些休息，一切事情你看着安排就行。”
<br/>　　呼延力走后，白无双呆呆立在那里，整个人像突然苍老了几岁。
<br/>　　这样一个人，还真无几分天下之君的气魄呐。
<br/>(本章完)
