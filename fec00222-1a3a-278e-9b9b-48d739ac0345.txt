　　在这虚空深处，隐蔽到极点的地方。
<br/>　　一团毒雾，突然出现，将楚风眠跟寒月漓，都给笼罩在了其中。
<br/>　　“谁？”
<br/>　　楚风眠冷冷的开口道。
<br/>　　毒手魔君的身形，就是在这毒雾之中，缓缓的凝聚出来。
<br/>　　只见毒手魔君的脸上，尽是阴狠，一双眼睛贪婪的扫视在楚风眠跟寒月漓的身上，好像正在盯着两个巨大无比的宝藏一样。
<br/>　　毒手魔君看着楚风眠大笑着开口道。
<br/>　　“受了伤，居然还敢离开武王城，来到这虚空深处，真是在自己找死！”
<br/>　　“是你？”
<br/>　　楚风眠看着毒手魔君，脸色冷静到极点。
<br/>　　不过在这冷静之下，楚风眠身上的灵力波动，却是极为混乱，虽然是在极力隐瞒，但是却根本瞒不过毒手魔君的双眼。
<br/>　　之前楚风眠可是硬抗了那夜牧老祖的轰击，夜牧老祖，可是长生之境的大人物，楚风眠就算是全身而退，也不可能这么快的速度回复过来。
<br/>　　现在楚风眠身上混乱的灵力，正是最好的证明。
<br/>　　如今楚风眠脸上的冷静，在毒手魔君看来，只是一种伪装，外强中干的证明。
<br/>　　“小子，得罪了我们地魔门，打伤了冷面魔君，还杀了葬天，你以为你真的可以从我的手中逃走吗？”
<br/>　　毒手魔君脸上尽是冷笑，开口道。
<br/>　　“本来你一直躲在武王城里，也许我还拿你没什么办法，但是你既然敢跑出来，今日我就将你，彻底的斩杀在这虚空深处。”
<br/>　　“你敢杀我？我的背后，可是北邙学院，以我现在的身份，你要是杀了我，北邙学院也不会善罢甘休的。”
<br/>　　楚风眠镇定自如的开口道，看起来极为冷静。
<br/>　　“北邙学院不会删吧干休？小子，看来你还不清楚吧，夜天君在北邙学院中的地位。”
<br/>　　毒手魔君突然大笑起来，他的嘴角露出了几分讥讽笑意道。
<br/>　　“看你现在就是一个死人了，我就让你当一个明白鬼吧。”
<br/>　　“夜天君背后的夜家，才是北邙学院中最为庞大的势力，夜天君更是夜家培养的绝世天才，你敢杀了他，回到北邙学院去，你也是死路一条，我就算是杀了你，你以为北邙学院会为了一个死人，去来找我的麻烦？”
<br/>　　毒手魔君的脸上露出几分狰狞，一双眼睛不断是从楚风眠跟寒月漓的身上扫视而过。
<br/>　　“更何况，在这虚空深处，我杀了你，没有任何人可以知道，没有人会知道是我杀了你。”
<br/>　　“今天你死了，要怪就怪你实在是太狂妄了，太过狂妄的人，终究只有死路一条。”
<br/>　　毒手魔君开口的一刻，他身上的灵力便是猛然凝聚出来，浩瀚瘴气席卷而来，在这瘴气之中，两道巨大无比的毒手，突然摄出，直接向着楚风眠抓了过来。
<br/>　　这每一道毒手，都是一种幽绿之色，上面蕴含了天下百种剧毒，稍微沾染上，就算是对超越生死的恐怖人物来说，都是要极为危险。
<br/>　　“没有人知道，那正好，那我杀了你，你们地魔门的人，也不会有着任何的麻烦！”
<br/>　　楚风眠突然嘴角露出几分冷笑。
<br/>　　就在这毒手魔君出手的一刻，楚风眠也是突然一步上前，祖龙至尊剑突然就出现在楚风眠的手心之中，剑锋闪耀。
<br/>　　一剑随着楚风眠挥动，直接斩向那两道毒手。
<br/>　　“毒手魔君，你真的以为你今日是吃定我了？觉得我已经重伤，没有办法反抗了？我不过就是在引蛇出洞罢了，虽然没有将冷面魔君引来，有些遗憾，但是既然你送上门来了，那我就先杀了你！”
<br/>　　楚风眠冷笑连连。
<br/>　　他身上的气势，节节攀升起来，本来在毒手魔君看来，楚风眠身上亏空的灵力，一瞬间便是彻底的补充完全。
<br/>　　楚风眠身上的伤口，都是在瞬息之间，几乎是一眨眼的功夫，全部恢复完成。
<br/>　　现在的楚风眠，看起来绝对是一种最为完美的状态。
<br/>　　他这次故意引诱，是想要去对付那冷面魔君的，这冷面魔君在擂台上，逃了出去，也是得到了前十的名额。
<br/>　　不杀他，他也可以进入到归墟天中去，楚风眠本来想要趁着这个机会先杀了他的，不过没想到地魔门一方，只有毒手魔君一个人来了。
<br/>　　不过也无所谓，来了一个毒手魔君，楚风眠一样要杀。
<br/>　　毒手魔君也是地魔门培养出来的绝世天才之一，他更是已经前去过归墟天，肯定是得到了无数好处，杀了他对楚风眠也有着巨大的好处。
<br/>　　“破！”
<br/>　　楚风眠剑锋出窍，剑光闪烁，这两道毒手，被直接斩断，就连那毒瘴，都被直接清除。
<br/>　　这祖龙至尊剑的锋芒，都足以克制无数毒物，足以驱散毒瘴。
<br/>　　“你你你！”
<br/>　　看着楚风眠突然出手，毒手魔君脸色顿时大吃一惊。
<br/>　　他本来追逐楚风眠一路，想要斩杀楚风眠，也正是想着现在楚风眠身受重伤，正好是趁着楚风眠最为弱势的时候，杀了楚风眠。
<br/>　　但是没想到，楚风眠的伤势，居然是装出来了，现在看起来，哪里受到了什么伤势。
<br/>　　“不可能，你的伤势怎么可能恢复，你以为这样，你就可以将我吓走？就算是你的力量，还在巅峰如何？就算是你还有全力，今日我也一样杀了你！”
<br/>　　毒手魔君冷哼一声，毫不退缩。
<br/>　　这毒手魔君也是上一届武斗大会上成名已久的天才，现在更是只差一步，都足以超越生死，踏入碎命之境的人物。
<br/>　　就算是现在楚风眠的实力，有些出乎了他的意料，但是他也丝毫不惧。
<br/>　　“万毒鼎！出！”
<br/>　　毒手魔君长啸一声，从他的手中，一尊幽绿色的鼎，突然出现。
<br/>　　这鼎上，散发出无数毒瘴气，不知道是淬炼过多少种毒物，才形成了如此地步。
<br/>　　这一尊鼎，虽然还未达到天级地步，却也是无限接近于天级灵器的地步，上面的力量，极为古老，甚至是一件数万年的古老灵器，已经是开始向着天级灵器进化的地步。
<br/>　　尤其是这万毒鼎上面的力量，更是最为适合毒手魔君。
<br/>　　在毒手魔君的手中，这一尊万毒鼎的威力，比起天级灵器，还要可怕。
