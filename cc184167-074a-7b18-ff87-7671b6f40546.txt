　　进入比赛室，叶帝等人便是迅速的将自己的键盘还有鼠标连接到了电脑上，接着打开英雄联盟进入游戏中调试了起来。
<br/>　　这个时候已经是下午4点钟，好在上午的时候已经打完一场bo5比赛了，要不然如果从下午开始的话，那还不知道最后一场比赛结束要到几点钟呢。
<br/>　　片刻之后，等双方都感觉差不多了之后，这才跟现场的裁判示意可以开始比赛了。
<br/>　　“好的，荣耀白队和eP战队的第一场比赛现在正式开始，这一把白队选择了蓝色方，eP战队则是在紫色方。”看着双方进入ban选画面，米粒连忙开口喊道。
<br/>　　“现在其实对于eP战队来说ban什么英雄挺难抉择的，毕竟白队这边需要针对的人太多了，而eP战队这边只需要针对好他们的下路就好了。”林雅楠跟着开口说道。
<br/>　　“是啊，而且大魔王这个点真的太难针对了，无论是卡萨丁还是沙皇，亦或者是妖姬这种中单，他哪个都玩的非常的出色，都是值得一ban的，但是现在就三个ban人位置，而且香蕉和夜雪的表现还这么亮眼，如果我是eP战队的教练，估计我现在也是懵。”米粒笑着说道。
<br/>　　双方飞快的ban人，叶帝他们这边ban掉了：女警、老鼠还有锤石这几个英雄，摆明了就是针对eP战队的下路，毕竟这个战队之所以这么出名，其主要原因就是因为他们的下路组合emp和aco两人。
<br/>　　emp是十王者选手这个就不用多说了，aco本人的实力也是非常强劲的，当初大师段位只有100人的时候，他可是位列89名的，那个时候能够挤入到大师段位的选手，其实就已经是非常的厉害的了。
<br/>　　而eP战队这边则是选择ban掉了：鳄鱼、盲僧还有妖姬这三个英雄，显然在他们的眼里这三个英雄是威胁最大的三个了。
<br/>　　“eP战队这边将鳄鱼、盲僧还有妖姬这三个英雄ban掉了，感觉这可能是最正确的选择了，毕竟这几个英雄在白队的手里可是都有着世界第一的称号呢。”看到eP战队的ban人，林雅楠开口说道。
<br/>　　“没错，而且有一点非常重要，那就是这三个英雄都是前期非常强势而且还能够带节奏的英雄，我相信就算是ST在面对白队的时候也不敢将这三个英雄全部给放出来，如果真的放出来的话，那么比赛恐怕坚持不到30分钟就要结束了。”米粒同样一脸自信的说着。
<br/>　　“大魔王他们三人已经用这几个英雄无数次证明自己了，只不过虽然eP战队ban掉了这三个英雄，不过场上可是还有着很多其他强势的英雄没有ban掉呢，就比如之前大魔王表现的非常好的龙王，我到现在还忘不了那最后一幕，实在是太震撼了。”林雅楠继续说道。
<br/>　　在他们说话的时候，场上双方已经开始了选人，这一边白队上来第一选直接抢下了蜘蛛这个如今版本依旧强势的打野英雄。
<br/>　　不出意外的，轮到eP战队这边选人的时候，eP战队直接抢下了龙王这个英雄，在这个英雄经过叶帝开发出来之后，便是已经成为了赛场上最为热门的中单英雄之一。
<br/>　　龙王清兵快，而且游走能力还强，虽然线上能力一般，但是在前期只要将兵线压制住，对手是完全不敢和他对拼的，当然了最重要的还是这个英雄能够带节奏，团战时的作用也够大。
<br/>　　无论是q技能的眩晕，还是大招的击退，在团战中都是非常有用的技能。
<br/>　　双方飞快的选人，很快各自的阵容便是确定了下来，
<br/>　　叶帝他们这边选择了：上单皇子、打野蜘蛛、中路蛇女、下路则是小炮和布隆这两个英雄。
<br/>　　而eP战队则是选择了：上单慎、打野挖掘机、中路龙王、下路则是大嘴和风女这两个英雄。
<br/>　　“eP战队不愧是以ad为核心的队伍啊，典型的四保一阵容，这一局就看emp如何发挥了。”看着对面的阵容，米粒开口说道。
<br/>　　“没错，这一局就看白队能不能够在前期就压制住大嘴这个点了，只要将大嘴给压住，那么eP战队这边基本上就已经输一半了。”林雅楠跟着开口说道。
<br/>　　“两边的阵容后期能力都很强，不过在ad这个位置上，到了后期的输出是没有人能够和大嘴比的，大嘴开启w技能的那几秒钟，完全可以将对面的前排当脆皮来打的。”米粒继续说着。
<br/>　　在两人说话的时候，场上双方已经是调换好了英雄，天赋和符文都已经准备完毕，正在那里等待着比赛开始。
<br/>　　因为蛇女无法购买鞋子，而且这个英雄还是持续型输出英雄，所以这一局叶帝这边选择带的是闪现和疾步这两个技能，两个都是保命技能，而且疾步在团战的时候还可以帮助自己追击对手，持续输出。
<br/>　　很快双方便是进入到了游戏之中，开局双方都是五星站位，都没有想要入侵对方的想法。
<br/>　　1分40秒双方正常对线，叶帝这边一级学的是自己的q技能，龙王学的自然是w技能，所以在一级清兵上面叶帝这边是比不过龙王的。
<br/>　　不过龙王这边在将第一波兵线推掉的时候，同样的双方的第二波兵线也是从中路的防御塔走了出来，加上蛇女的q技能是aoe伤害，冷却时间还短，他本身的清兵速度也不慢，所以等到对方的兵线推到自家塔前的时候，叶帝同样也是将对方的兵线给清掉了。
<br/>　　只不过这个时候他比对方的速度稍微慢了一些而已，倒是对方的兵线稍稍向着自家这边的防御塔这边压进着，不过这些并没有太大的关系，如今叶帝需要做的就是牵制住对面的龙王就好了。
<br/>　　蛇女这个英雄到达二级的时候清兵速度就要快上很多了，毕竟他的e技能的冷却时间只有0.75秒，当然了如果持续放技能的话，是非常耗费蓝量的，所以怎么个清理兵线还是要看叶帝自己斟酌。
<br/>　　总之在中路的对线上，虽然龙王的确实发挥了自己清兵上的优势，不过却是并没有太多的游走机会，每次对方游走的时候，叶帝这边总是会迅速的清理掉兵线跟上来，导致eP这边的中单总是半途而返。
