　　两人于是一起下山。路上，叶少阳一直注意观察周围的环境，一起看上去都十分真实，叶少阳也渐渐放下了戒心，觉得自己可能是多想了，也许真正的危险就在之前的幻境中，被自己识破了，对方那些没使出来的手段也就没用了。
<br/>　　山路的边上，就是一条溪流，是从黄果树瀑布流下来的水，十分湍急，这两天大概是因为下雨，水位升高，从山上奔涌而过，气势磅礴。
<br/>　　一座木头搭建的浮桥，从溪流上横过去，从上面走的时候，谢雨晴把手机拿出来，对着溪流拍照，结果因为浮桥晃动，她没站稳，人抓住栏杆晃了两下，手机竟然掉在了水里。
<br/>　　谢雨晴惊叫一声，本能地伸手去捞，没想到那郎安上的铁链没固定好，她俯身一抓，人没站稳，竟然一头扎进了水里
<br/>　　叶少阳飞身冲过去，但谢雨晴已经被水冲走了，脑袋在水面浮浮沉沉的，一边对叶少阳伸手叫救命。
<br/>　　叶少阳立刻就要冲进去救人，刚爬过栏杆，猛然想到，怎么就这么巧合？手机掉到水里，人也跟着掉到水里，而且谢雨晴伸手很不错，不会就这么容易掉到水里吧？
<br/>　　会不会是幻觉？想要通过谢雨晴来骗自己下水？
<br/>　　叶少阳犹豫起来。
<br/>　　“少阳……”谢雨晴在水中拼命游着，一边冲他呼救。
<br/>　　万一……这是真的呢？
<br/>　　看着谢雨晴在水中挣扎的样子，纵然他怀疑这是假象，还是无法坐视不管。
<br/>　　纵然只有百分之一的可能，他也不敢赌。
<br/>　　不管了！
<br/>　　叶少阳顺着溪水畔飞奔起来，追上谢雨晴，然后一个猛子扎进了水里，朝谢雨晴快速游过去，将她从水里抱起来，让她抓着自己的肩膀，然后朝岸边游。
<br/>　　然而……水流太急了。纵然他水性十分不错，一时半会还是游不过去。
<br/>　　他越来越感到身上谢雨晴的重量，压的他不断沉入水中。叶少阳拼命冲刺，还是支撑不住了，喝了好几口水。
<br/>　　这时候，他们离岸边也越来越近了。
<br/>　　叶少阳干脆吸了一口气，沉入水下，双手托着谢雨晴的腰，让她的脑袋能够伸到水面上去呼吸。
<br/>　　一口气很快就用尽了，叶少阳倒是可以探出去吸气，但是那样就会乱了节奏，在如此湍急的水流中，真是不进则退，他吸一口气不要紧，这些努力也就白费了。
<br/>　　叶少阳只有死撑着。
<br/>　　因为大脑缺氧，意识很快就模糊了，只剩下一个模糊的念头，就是至少要把谢雨晴送上岸。
<br/>　　谢雨晴也拼命地游着，双脚在水里乱踢，两只手也用力按着他的脑袋，借助反作用力往前游。
<br/>　　叶少阳心中咯噔了一下，似乎明白了什么，但是……他还是不敢赌，只好什么也不管了，苦苦支撑着谢雨晴上岸。
<br/>　　总算到了岸边，叶少阳也精疲力尽了，用尽最后一丝力气将谢雨晴往前推了出去，谢雨晴上岸了，他自己还没等站稳，就被狂乱的水流冲走了……
<br/>　　冰凉的溪水，将他裹紧了漩涡之中，随波逐流，而他的意识，也一点点模糊起来。
<br/>　　要死了吗？那就这样吧。
<br/>　　至少，谢雨晴得救了。
<br/>　　就算自己赌错了，至少也是问心无愧。
<br/>　　这是叶少阳最后的念头。
<br/>　　之后，他陷入了半昏迷的状态，下意识觉得自己好像穿梭在无尽虚空之中，不知道走了多远，突然一个激灵，叶少阳睁开了眼睛。
<br/>　　一片黑暗。
<br/>　　叶少阳坐起来，深吸了一口气，朝四周看去，什么都看不到，他摸了摸身边，是湿润的岩石，这么说，这里还是山洞？
<br/>　　又摸了摸身上，是干的，没错，那就说明自己之前猜的没错，那是幻境……一切都是假的，谢雨晴也是假的。
<br/>　　但是……如果操控幻境的人就是为了让自己跌入水中，元神消磨，那为什么自己现在没事。
<br/>　　叶少阳站起来，大声呼唤谢雨晴的名字，但没人理会。
<br/>　　这时候，他看到了两点金光，从黑暗中亮起来，情不自禁地走了过去。
<br/>　　是那尊地藏菩萨的雕像，跟之前自己看到的一模一样，只是睁开了眼睛。
<br/>　　金光，就是从他眼珠子里射出来的，应该就是佛光了。
<br/>　　叶少阳被这佛光吸引了，呆呆地仰头望着神像的脸，突然，佛像的脑袋中间出现了一道裂缝，裂缝里也透着佛光。
<br/>　　裂缝越来越大，也越来越多，很快就蔓延到了全身，几秒钟后，只听得一声清脆的巨响，佛像崩塌了。
<br/>　　一阵极为耀眼的佛光，照得叶少阳睁不开眼，本能地将眼睛捂住了。
<br/>　　过了一会，佛光散尽，叶少阳睁眼看去，雕像已经不见了，岩壁上出现了一方状如佛像的裂缝，外面居然是蔚蓝的夜空，星光斑斓之下，能看到远处有一座山的暗影，在自己与这座山的暗影之间，是一望无尽的草原，间或长着一些十分高大的树木。
<br/>　　这是什么地方？
<br/>　　叶少阳一脸懵逼，回身叫了几声谢雨晴，没人理会，叶少阳想了下，干脆就到前面去看看，于是走上了草地。
<br/>　　脚下软软的感觉，引起叶少阳低头看了一眼，自己居然是光着脚的，心想还真是奇怪，不管它了，先走再说。
<br/>　　走出山洞，顿时有一股温暖的风吹来，浑身都舒服，叶少阳忍不住回头看了一眼，吃惊地发现，山洞没有了，自己身后是一望无尽的草原，然后有个下坡，不远处是一望无尽的水面，水也是蓝色的，像镜子一样平静。
<br/>　　这是什么仙境宝地？
<br/>　　叶少阳正不知所措，对面突然有人叫了自己一声：“小天师……”
<br/>　　叶少阳猛然转头，对面不知什么时候多了一个人，是一个头戴僧帽的老和尚，身上袈裟破破烂烂，手里提着个葫芦，敞胸露怀，看上去微胖，一只手拍着肚子，冲自己笑着，好像自己有多么可笑似的。
<br/>　　“你……”
<br/>　　“得罪得罪，小天师，你不认识我，我可认识你呢！哈哈哈……”和尚冲他招手，一边打开了葫芦，灌了一口，从他的表情看，似乎是在喝酒。
