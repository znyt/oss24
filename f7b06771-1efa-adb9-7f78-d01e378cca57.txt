　　现在楚风眠全身灵力消耗殆尽，才勉勉强强可以催动这始祖天龙之爪打出一击。
<br/>　　而且这还是只能够发挥出始祖天龙之爪十分之一的力量。
<br/>　　这始祖天龙之爪的灵力雄厚，足以是楚风眠的数十倍。
<br/>　　现在楚风眠却是起了要吞噬了这始祖天龙之爪的心思。
<br/>　　这种想法简单是胆大包天，蛇吞象一般的想法。
<br/>　　不，简直足以称得上是蛇吞鲸，要吞噬比起楚风眠强横数十倍力量的始祖天龙之爪。
<br/>　　这种想法只有疯狂才能想到。
<br/>　　不过楚风眠能够成就如今的实力，也是经历了一次次的疯狂，做过无数人认为无法做到的事。
<br/>　　楚风眠曾经借助神树的力量，曾经炼化过青铜天魔骸骨，今日也未必不能炼化了这始祖天龙之爪。
<br/>　　“吞天秘术！给我吞噬！”
<br/>　　想到这里，楚风眠便是毫不犹豫，将吞天秘术的力量全然催动起来，将这始祖天龙之爪吞噬到了楚风眠的身躯之中去。

<br/>　　这种行为，几乎相当于一个人吃下自己身体数十倍的食物，肯定是会被撑死。
<br/>　　可楚风眠却是硬生生的承受了下来，也幸好是楚风眠凝炼出了不朽真身，不然他也不敢如此疯狂。
<br/>　　不过楚风眠感觉到，他想要将这始祖天龙之爪吞噬到身躯之中去，已经是消耗了大量的力量。
<br/>　　想要将这始祖天龙之爪炼化，比起单单吞噬了这始祖天龙之爪，还要困难无数倍。
<br/>　　不过与此同时，楚风眠也感觉到，他的血脉之力，对着始祖天龙之爪却是并不抗拒，甚至也是无比欣喜这，想要让楚风眠吞噬了这始祖天龙之爪。
<br/>　　楚风眠的龙皇血脉，达到极致，便是曾经始祖天龙的血脉。
<br/>　　本属同源。
<br/>　　所以要将这始祖天龙之爪融入身躯之中，楚风眠身上的龙皇血脉却并不抗拒，甚至是帮助楚风眠，在炼化这始祖天龙之爪。
<br/>　　不过这始祖天龙之爪之中，却是传递出了反抗的力量。
<br/>　　这始祖天龙之爪可是曾经始祖天龙身躯的一部分，楚风眠虽然是拥有龙皇血脉，可是在这始祖天龙之爪看来，却只是血脉稀薄的小家伙罢了。
<br/>　　根本不配炼化他，自然是要无比抗拒。
<br/>　　可惜始祖天龙以死，这始祖天龙之爪也已经是经历了数千万年，甚至是更加古老的时间，早已经是没有了多少可以反抗的力量。
<br/>　　如果有外力相助，还可以发挥出无比可怕的力量，可如果失去了外力，这始祖天龙之爪却只能够是勉勉强强守护住自己，抵挡这楚风眠的炼化。
<br/>　　“神树现身，给我吞噬！”
<br/>　　楚风眠咆哮一声，在他身躯之中的神树，也加入了进来，一齐来帮助楚风眠来吞噬这始祖天龙之爪。
<br/>　　这神树，可是就连青铜天魔骸骨都曾经吞噬过。
<br/>　　始祖天龙，虽然是凌驾于十方天魔之上，可现在只是这一道龙爪，未必是无法炼化。
<br/>　　神树的力量也在疯狂的吞噬这始祖天龙之爪，但是这始祖天龙之爪，却是俨然不动。
<br/>　　就在楚风眠的身躯之中，无论楚风眠是如何炼化，都无法从其中剥夺出一点点的灵力来。
<br/>　　整个始祖天龙之爪，就这样在楚风眠的身躯之中，俨然不动。
<br/>　　“无法炼化？”
<br/>　　楚风眠眉头一皱，他可以吞噬了这始祖天龙之爪，可却是无法炼化？
<br/>　　楚风眠曾经炼化过青铜天魔骸骨，更是借此凝聚出了半步不朽真身。
<br/>　　可没有想到，这始祖天龙之爪，仅仅只是一道龙爪，却是这样的难缠。
<br/>　　没有办法炼化，楚风眠将这始祖天龙之爪吞噬到身躯之中去，也没有任何的意义。
<br/>　　这就相当于，楚风眠面前有着一座宝箱，可楚风眠却是根本打不开一样，只能够是看着，哪怕是在手里，对楚风眠来说，也没有任何的好处。
<br/>　　可现在放弃？
<br/>　　楚风眠摇了摇头，这始祖天龙之爪他无论如何也不想放弃。
<br/>　　“我就不信，这始祖天龙已经是死去了千万年，这一个龙爪我都无法炼化！”
<br/>　　楚风眠仰天咆哮，肆无忌惮，不在保留，将身上所有的力量，都给爆发了出来。
<br/>　　大量的灵力，在疯狂的消耗着，哪怕是楚风眠拥有神脉，也无法抵得上楚风眠这样的消耗。
<br/>　　刹那之间楚风眠身躯之中的灵力便是都被消耗一空。
<br/>　　楚风眠大手一挥，将他空戒之中的两座灵泉，都给拿了出来。

<br/>　　可现在楚风眠需要大量的灵力，正好是足以从这两道灵泉之中来补充。
<br/>　　两座灵泉被楚风眠拿了出来，楚风眠催动起吞天秘术来，先是将那六绝宗的灵泉，一把吞噬。
<br/>　　楚风眠身躯之中消耗的灵力，全部是从这六绝宗之中吞噬来补充，以可以保持楚风眠长时间的爆发出最强的力量。

<br/>　　始祖天龙之爪跟楚风眠左手凝聚在了一起，其中的力量开始结合在了一起。

<br/>　　就像是荒鎏金，便是将剑鼎融入到了他的身躯之中，来提升力量。

<br/>　　可楚风眠却全然不顾了，他二世为人，本就是逆天行为，又有何惧？
