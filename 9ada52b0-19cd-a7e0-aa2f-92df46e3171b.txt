    
<br/>　　<br/>　　长安东郊，有一片荒芜的大野地。<br/>　　<br/>　　此地属于长安城郊区，是一片待开发的地段，这几年国家大力开发房地产，采取了从城市包围农村的政策，拆拆建建的从市区延伸到了郊区。<br/>　　<br/>　　这片地方属于长安东郊，以前一直都是片荒无人烟的大野地，按理来说此处离长安也不算太远，就算盖不了厂房，工厂什么的至少也能种地吧，无故荒废了也挺可惜的。<br/>　　<br/>　　不过，长安城里的老一辈人都知道，这个地方不太平，不干净，老人要是这么说那就只有一个可能了，这里可能会有脏东西。<br/>　　<br/>　　据说，九五年左右，当时有个台商来长安投资，看上了东郊大野地这块地方，跟官方谈完了以后很快就把这块地皮给拿下来了，于是开春的时候就破土动工准备建个厂子，没过多久材料都运了过来，施工工人也开始进驻，但在动工的第二个星期却出了一件怪事。<br/>　　<br/>　　这时候本来是要挖地基的，当时地基挖了能有三四米深时，没想到却挖了一堆散乱的白骨出来，当时就给工人吓了一跳施工方赶紧报警，警方来了之后封锁现场把那堆白骨全都给启了出来，大概零零散散地能有五六具左右，甚至还能明显的从这些白骨上看出有刀伤和枪伤来，后来警方经过排查和法医的检验，才知道这堆骨头可能都有几十年的历史了，建国之前就已经被埋在地下了，初步判断觉得有可能是当年r国人造的孽，在那个兵荒马乱的年代，死人什么的基本上都随便一扔就完事了。<br/>　　<br/>　　既然是历史问题也不是什么刑事案件，这个插曲很快就被掀过去了，施工方继续开始开工干活，但没想到只隔了一天，再次打地基的时候又发生了个意外，有工人从地下挖出个破烂的草席，草席里面居然裹着一具尸体，更让人意外的是这草席里的尸体都没有腐烂，还穿着身很破旧和很有年代感的衣服。<br/>　　<br/>　　这具尸体一挖出来工人们都炸了，等警察来了之后也懵了，挖出白骨还可以解释，但挖出一具尸体来就不是小事了，并且这尸体保存的非常完整，甚至面容都能看的清清楚楚的。<br/>　　<br/>　　有住在东郊附近的老人过来看热闹，就说这里以前是一片乱坟岗，那时候还能零[海棠书屋 www.htsw.info]散的看见十几个坟头呢，后来不知道什么原因坟头都没了，但附近的人一到晚上谁也不敢经过这一片。<br/>　　<br/>　　再往后，就没有然后了。<br/>　　<br/>　　台商撤资了，悄无声息的就走了，工地彻底停工了，这里又变成了一片荒芜的大野地。<br/>　　<br/>　　不过，有附近的人曾经看见，挖出尸体的第三天晚上，忽然来了一帮人封锁了这片地方，然后深夜的时候就冒出了火光，那火烧了很久，期间还伴随着焦糊的味道，等隔天天亮了后，有胆大的人过去发现，有一块地方的土都被烧的焦黑了，周边的野草也被烧的干干净净的。<br/>　　<br/>　　那里，就是挖出那具被草席包裹着尸体的地方。<br/>　　<br/>　　一过二十年，此处一直都荒废着，哪怕是长安附近都被开发了，这里也不知道是什么原因，一直无人问津。<br/>　　<br/>　　这天晚上，东郊外的野地里，九点钟左右的时候开过来一辆车，车门推开下来三个人，脑袋上都包着头帕穿着对襟雄衣，这三人除了一个看起来年纪颇大外剩下那两个大概都四十岁左右上下的年纪，皮肤黝黑身材不算高大。<br/>　　<br/>　　从车上下来后，有两个打开了后备箱从里面开始往外拿出工具，年级颇大的那个老人则是径直朝着野地中间走了过去，然后蹲下身子用手随意的扒拉着地上的尘土，片刻后露出一片焦黑的泥土来，就好像是曾经被火给烧过一样。<br/>　　<br/>　　这人叹了口气，头也没抬的说道：“可惜没了一具，当初被挖出来之后就给烧了，我们都来不及给带走，白白的糟蹋了”<br/>　　<br/>　　“东干叔，其他的没事吧？”那两人拎着铁锹走过来问道。<br/>　　<br/>　　东干叔摇头说道：“没事，从那以后这片地方就没人来了，埋的又比较深，轻易没人发现·······我指地方，你们下手，快点给挖出来我们今天晚上连夜就走”<br/>　　<br/>　　“那行，东干叔，我们请尸吧！”<br/>　　<br/>　　请尸，一个专业而又生僻的名词，据传古老的湘西巫族人会将尸体养于一处阴气旺盛之地，待到孕养成之后就会将尸体挖出来然后带走。<br/>　　<br/>　　这种养尸和湘西赶尸却又有着很大的不同，湘西的赶尸只是将客死异乡的人尸体送回家乡安葬，和蛊毒、落花洞女一起，并称为湘西三邪。<br/>　　<br/>　　养尸就不一样了，这是从湘西古老苗族中传下来的一门巫术，养尸人会特意去寻找一些尸体然后孕养在地下，等到尸体成型之际就会再给重新挖出来，当尸体被挖出来的时候它们几乎还保持着原貌没变，肉身一点不腐，甚至毛发，牙齿和身上的毛孔都能清晰可见，皮肤还松弛有度就跟刚死不久一样。<br/>　　<br/>　　国内曾经有两处著名的养尸地，一是闽南的永安养尸地，还有个就是玉溪的抚仙湖，水下据说站立着不少的尸体，每当水流流动的时候这些水下的尸体还会随着水流摆动，就像人还没死潜在水下行走一样。<br/>　　<br/>　　只不过，后来随着这两处地方被大众所熟知了，两个养尸地才逐渐的荒废掉，到现在基本已经都不存在了。<br/>　　<br/>　　至于为什么要养尸，外人就不知晓了，只有湘西古老苗族部落里那些大巫才清楚，这是他们代代相传的秘辛，外人从不得知。<br/>　　<br/>　　漆黑的夜空下，铁锹翻动着地上的土石，没过多久两个人就将脚下的这处地方挖出了个能有四五米深的土坑，随后一个已经腐烂破旧的棺材给挖了出来。
