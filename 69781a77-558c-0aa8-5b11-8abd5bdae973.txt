<br/>    “这里又出现了亡灵魔法的气息。”三团黑影出现在了李振邦吓跑金广城那些人的位置。<br/>　　<br/>　　    “没错，气息很浓郁，也很纯正，看样子那个人对亡灵魔法的了解还是很深刻的，就是等级应该不高。”看着面前坑里的一堆低阶的骷髅战士和骷髅魔兽的骨头，其中一人笑着说道。<br/>　　<br/>　　    “家里面传来消息，那个人不是咱们的人，估计是一位野修，我们要尽快找到他。我们亡灵魔法师本来就少，这么有天赋的人，带回去就是大功一件啊！”<br/>　　<br/>　　    提到功劳，三人不由得心中都是一喜。亡灵魔法师家族可是赏罚分明的，做的好，赏的也多，做的差，罚的也狠。<br/>　　<br/>　　    亡灵魔法师家族并不是传统意义上的血缘家族，其实亡灵魔法师家族和魔法师工会很像，不过他们的首领叫做族长而不叫会长。<br/>　　<br/>　　    亡灵魔法师本身就是一个很稀少的存在，自从那次大清洗以后，亡灵魔法师彻底的衰落了下来。多数强大的亡灵魔法师都被处决了，少数人躲藏了起来。<br/>　　<br/>　　    随着时间的推移，存活下来的亡灵魔法师们又渐渐聚集在了一起，形成了一个族系，一直生活在黑暗之中。亡灵魔法师虽然稀少，但是亡灵魔法师的生命要远远比正常人要长，在一些特定的情况下，说他们是永生也不为过。<br/>　　<br/>　　    每年亡灵魔法师家族都会派出一部分人在整个大陆上悄悄的寻找新诞生的亡灵魔法师，也会招揽一些为了活的更久而甘愿为他们服务的强者。<br/>　　<br/>　　    为首的一名亡灵魔法师突然皱起了眉头，其他两名亡灵魔法师也发现了异常。他们三人似乎被包围了，不过他们并不在乎。因为亡灵魔法师最擅长的并不是单挑，而是混战，人越多他们造成的伤害就越大，对他们就越有利，除非对方的等级远远高于他们。<br/>　　<br/>　　    “就是他们三个，他们都是亡灵魔法师！”一名白银战士指着包围圈里的三个人高声喝到。如果李振邦还在这里，就会看到这人正是之前金广城的那名领头的白银战士。<br/>　　<br/>　　    三名亡灵魔法师有些诧异，他们不过是刚来到这里一会儿，怎么会被那个白银战士认出来的呢？而且从现在的情形来看，他们显然是有备而来，并不是偶然相遇的。<br/>　　<br/>　　    白银战士也很纳闷，怎么自己离开这一会儿，对面的三个人竟然把黑披风黑斗笠换成黑斗篷了，不过这些都不重要，他们抢走的那个小兔崽子也不见了。他拉了这么多人来，一个是要找回刚才的场子，更主要的是要抢回小兔崽子身上的那两块儿石头。<br/>　　<br/>　　    白银战士心中猜测，那个小兔崽子肯定被这三个亡灵魔法师给宰了，他可不相信亡灵魔法师会有什么好心，恐怕那两块儿石头已经在亡灵魔法师的身上了。<br/>　　<br/>　　    本来这群人已经逃走了，不过幸运的是路上他们又遇到了一些自己人，其中还有一名黄金战士。而且还碰到了一些其他队伍的人，他们打着金广城的名义，把大家组织了起来，组成了一个三四十人的大团队，回来消灭亡灵魔法师来了。<br/>　　<br/>　　    一个是利益的诱惑，遗迹的线索可是可遇不可求的。再一个，有亡灵魔法师存在，谁都没有安全感，人多了大家胆气也就壮了。<br/>　　<br/>　　    这三四十人的队伍战斗力还是很惊人的，黄金战士三名，五级魔法师两名，四级魔法师四名，白银战士十几名，其余都是黑铁战士。<br/>　　<br/>　　    三名亡灵魔法师看到对方这些人的实力以后，也不由得一阵咂舌。如果仅仅是三名黄金战士他们倒真不会放在眼里，可是还有还几名魔法师在，那就不一样了。如果魔法师和黄金战士配合好，外加白银战士敲敲边鼓，他们就会很被动。<br/>　　<br/>　　    不过三人心中还是比较镇定的，对方这些人明显不是一起的，只不过是临时搭伙而已。不逼到一定程度，谁也不会真的玩命，彼此间的配合肯定不会十分默契。<br/>　　<br/>　　    众人虽然将三名亡灵魔法师包围起来，但是谁也没有主动出击。能来到这里的人没有人是傻子，谁都知道亡灵魔法师的厉害，也没有人想要成为亡灵魔法师的傀儡，所以双方都还是比较克制的。<br/>　　<br/>　　    “把那个孩子给我交出来，否则就别怪我们不客气了。”之前的那名白银战士首先开口叫道。不过他可没有傻到跑到亡灵魔法师面前叫嚣，只是站在人群人叫喊着。<br/>　　<br/>　　    三名亡灵魔法师对视一眼，他们自然不知道什么孩子，不过他们现在也猜测到了，他们三个估计是替他们要找的那个亡灵魔法师背锅了。不过身为亡灵魔法师，他们也有亡灵魔法师的傲气，根本不屑于去解释什么。而且他们已经认出自己三人的身份了，这件事情自然也不会简单的解决了。<br/>　　<br/>　　    白银战士很疑惑，因为他没有感觉到对方那名黄金战士，不过疑惑归疑惑，对面三名都是亡灵魔法师这一点却是不容置疑的。<br/>　　<br/>　　    “孩子！哼！你过来拿啊！”一名亡灵魔法师阴恻恻的说道。<br/>　　<br/>　　    白银战士自然是不敢过去拿的，甚至不自觉的往后缩了一缩。不过他很快就反应过来了，又往前走了一步，刚才后缩的动作让他有些恼羞成怒。<br/>　　<br/>　　    “哼！孩子我们可以不要，但是你要把那两块石头给我们，否则我们一拥而上，将你们剁成一滩烂泥。”<br/>　　<br/>　　    “嘿嘿，你既然想要石头，那我就给你好了！”另一名亡灵魔法师声音沙哑的说道。<br/>　　<br/>　　    双臂一挥，两道黑影闪电般的奔着白银战士飞了过去。白银战士哪里敢接，身形一闪避开了两道黑影。不过他身后的那名黑铁战士就没有这么好的运气了，本来站在白银战士身后就是一个盲区，白银战士突然闪开，根本没有给他反应的时间。<br/>　　<br/>　　    只听噗噗两声，那名黑铁战士惨叫一声，整个人就飞了出去，然后重重的落在了地上。<br/><br/><br/>
