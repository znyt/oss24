　　刹那之间。
<br/>　　这一刀的刀锋，已经逼近楚风眠。
<br/>　　荣北的刀，便是将快字一决，领悟到极致的刀。
<br/>　　刀锋一出，全场心惊，如此快的刀锋，又有谁可以抵挡。
<br/>　　“瞬刀？”
<br/>　　楚风眠脸色平静道极致，眼角微微一抬，冷静到极点。
<br/>　　面对着恐怖至极的一刀袭来，楚风眠的眼神中，甚至都是满不在乎。
<br/>　　“你的瞬刀，还差了一点，能够看清刀锋的瞬间，根本没有任何的意义！”
<br/>　　只见楚风眠缓缓抬起手，凌空一抓，那近乎无法看清的刀影，突然就在他的面前，凝固住了。
<br/>　　“挡住了？”
<br/>　　无数众人视线，纷纷看了过来，而接下来，等那刀影停滞下来的一刻，他们每个人，都近乎无法相信自己的眼睛。
<br/>　　这荣北，恐怖至极的刀锋，居然是被楚风眠，直接抓在了手中。
<br/>　　那一把刀，被楚风眠握在手中，无法在任何移动。
<br/>　　“怎么可能！”
<br/>　　荣北刚刚那一刀的刀锋，在场的任何人都明白其中的可怕，就连刀锋都无法看清，想要闪躲，都是极为困难的一件事。
<br/>　　可是现在，居然是被人挡住了，而且是徒手挡下，甚至是就连刀锋都被人抓在了手中。
<br/>　　这一幕，让所有人的脸色大变，任由是谁都想不到，居然有人可以抓住如此可怕的刀锋。
<br/>　　就算是荣北本人的脸色，都是震惊到了极致，尤其是在楚风眠的口中，更是说出的“瞬刀”两个字。
<br/>　　荣北的刀法，也并非是他自己领悟出来的，而是他曾经偶然得到奇遇，在一处远古洞府之中，得到的刀法。
<br/>　　这刀法的名字，便是名为瞬刀。
<br/>　　一直以来，利用这瞬刀，他不知道是失败过多少武者，他自己做梦都想不到，居然被被人徒手挡下。
<br/>　　甚至是就连刀锋，都被楚风眠抓住了。
<br/>　　任由他如何用力。
<br/>　　这刀也一直被楚风眠死死的抓在了手中，根本就不是他可以撼动的。
<br/>　　楚风眠看向荣北，冷冷的开口道。
<br/>　　“瞬刀之道，其实不过是剑走偏锋罢了，仅有速度，若是被人看穿，则是一文不值，你没有达到无刀客的境界，你的瞬刀，也终究只能对付一些废物罢了。”
<br/>　　“你到底是什么人？”
<br/>　　听到无刀客三个字，荣北的脸色完全变了。
<br/>　　他的刀法，便是自称自无刀客，这无刀客是远古时代的一尊强者，号称无刀客，便是从未有人，见过他的刀。
<br/>　　但是所有人，却也都是死于在他的刀锋之下，他便是将刀法的快字，修炼到大成之人。
<br/>　　荣北的刀法，除了强大，鬼神莫测的速度，也是他最为强大的手段。
<br/>　　但是这一切，也都是被楚风眠所看穿。
<br/>　　“你的太多了，我接了你一刀了，你也该接我一剑了！”
<br/>　　楚风眠摇了摇头道。
<br/>　　他可懒得跟这些荒古门弟子说太多。
<br/>　　七大宗门的弟子，在楚风眠的眼中，都是死人。
<br/>　　“剑！”
<br/>　　楚风眠一声轻喝，赤炎剑便是出现在了他的手中，楚风眠握剑，他身上的气势，便是就跟曾经，完全不同。
<br/>　　“一剑万重！万剑合一！”
<br/>　　“轰！”
<br/>　　楚风眠手中的剑，猛然落下，在这一剑落下的一刻，也近乎是强大到极点的剑意，一同落下。
<br/>　　一剑落，天地灭
<br/>　　这一剑落下，其中的剑锋，疯狂的节节攀升起来，变的庞大无比，化为了足足有着千丈之长的恐怖剑锋。
<br/>　　“战！”
<br/>　　同时楚风眠身上战龙决的力量，也是被他直接催动起来，在这剑锋之上，凝聚出了无数龙力。
<br/>　　这一剑落下，不止有着剑的锋利，更是有着龙的庞大力量。
<br/>　　庞大剑势落下，简直压得人，透不过气来。
<br/>　　“这是什么剑术？”
<br/>　　“此人的实力，到底是达到了哪一步？”
<br/>　　“这真的是神力境四重的武者可以做到的吗？”
<br/>　　一张张震惊的脸庞。
<br/>　　这一剑，简直是比起荣北的那一刀，还要恐怖无数倍。
<br/>　　荣北的那一刀，只是快，快到了极致，但是力量却并没有太过强大，让人感觉到，也有着一线生机，可以抵挡。
<br/>　　但是如今楚风眠的这一剑，却是绝对恐怖的力量，锋利的剑锋，雄厚到极点的力量，合二为一的一剑。
<br/>　　人们感受到这剑锋的力量一刻，一个个都是已经心生绝望，丝毫没有任何一点敢于抗争，甚至是觉得可能对抗的勇气。

<br/>　　荣北在这力量之下，都是节节败退，喃喃的说道。
<br/>　　他的双眼中，都满是惧意，在这恐怖一剑的面前，他的心中都没有任何还敢于反抗的能力。
<br/>　　“北域蝼蚁？七大宗门的人又如何？敢来对付我，来一个，我就杀一个，七大宗门的人也一样！”
<br/>　　楚风眠看着荣北，眼神中满是杀意。
<br/>　　“就先杀了你这个出头鸟再说！”
<br/>　　“咔嚓！”
<br/>　　荣北面前的刀，瞬间破碎开来，他的刀，也是一把地级灵器。
<br/>　　但是就算是地级灵器，在如今楚风眠这一剑的面前，也没有任何的意义，直接毁灭。
<br/>　　就连荣北的身躯，都被在这剑锋下，开始被吞噬。

<br/>　　“生死境四重的武者，都无法凝聚出如此可怕的刀锋吧！”
<br/>　　“荣北也要败在他的手中了！”
<br/>　　剩下的三尊荒古门弟子，看着这一幕，都是大喝一声道。
<br/>　　“该死，速速出手相救！”
<br/>　　“住手！”
<br/>　　三道厉喝，同时响起。
<br/>　　那剩下的三尊荒古门弟子，也是忍不住了，同时出手。
<br/>　　无数武技，向着楚风眠轰击过来。
<br/>　　“住手？“
<br/>　　楚风眠看了一眼那三尊荒古门弟子，脸色不屑的说道。
<br/>　　“你们以为你们是谁，还敢让我住手！”
<br/>　　“战龙之爪，给我破！”
<br/>　　楚风眠顿时凝聚血脉中的力量，足足是六千多滴远古战龙精血，被他一下子凝聚。
<br/>　　这一次这战龙之爪凝聚出来的样子，更是栩栩如生许多，一枚枚龙鳞上，都刻画有着古老的纹路，比起曾经凝聚出的，都要细致许多。
