　　第571章
<br/>　　楚云生回头望了一眼，山头的火还在烧。他麻溜快的向前走，一刻也不敢留。
<br/>　　这件事如果传出去，让天下人知道是自己放火烧楼。那就将全天下的读书人都得罪了，然后就不是谁要弄死自己的问题，而是自己就算死了，尸体也得从坟墓中挖出来大卸八块。
<br/>　　自己将会成为千古罪人，自己后代也将盯着骂名抬不起头来。当然，自己目前没有后代。
<br/>　　走了好，走了好，走了一了百了。
<br/>　　关键是，自己下一步该往哪里走。这把火烧毁了万卷藏书，也同样点燃了楚云生体内的野心。
<br/>　　天下大乱，诸侯林立，自己未必不能插一脚乱世，留下自己的名字。
<br/>　　十八路诸侯，两家天子，自己该投奔何处？
<br/>　　不若去凉州投奔程大雷，虽然程大雷不在二帝十八王之列，但天下谁人都明白，他是一股不容忽视的势力。或许，最终还将影响天下的归属。
<br/>　　可他终是个山贼，已经是人人喊打，人人欲杀的程度，况且凉州与东海相距太远。自己一瘸一拐的拐到凉州，还不知要等到什么时候。
<br/>　　不过，如今想这些也是多余，现在琅琊城还封着城门呢，谁也出不去。自己要做的，是找个地方忍一忍，等度过这场风波，自己再好好想一想，让天下那一路诸侯当自己的主公。
<br/>　　琅琊城他还是很熟的，毕竟在这里住了太久，犄角旮旯可以存身的地方太多了。
<br/>　　当他一瘸一拐走过街道，寻找藏身之地时。并没有注意到，路旁一间药铺中，传来淡淡的血腥味。
<br/>　　今夜的事情令他很激动，但老实说，确实也将他吓坏了。
<br/>　　这是一家叫做杏林堂的药铺，外表看上去平平无奇，杏林堂也不算太特殊的名字。几乎每座城池，都能找到一间名叫杏林堂的药铺。
<br/>　　但今夜，药铺里面的情形却有些特殊。
<br/>　　药铺上下十三口人，包括家里养着的一只守门犬，此时都倒在血泊中。
<br/>　　燕不归视若无睹，举着蜡烛翻检着药柜中的草药。他走过的地，淅沥沥流下血来。他身上受的伤的确很重，身上大大小小不知有多少伤口，能拖着伤躯来到这里已极其不容易。
<br/>　　不过草原生活的人，与天斗，与人斗，与恶劣的气候与野兽斗。比这更严重的伤，他也并非没有受过。多多少少，他还是有一些草药方面的知识。
<br/>　　外敷内用，将身上的伤口包扎，首先要做的事情，便是止血。之后，他拖着伤躯，来到院子中央。
<br/>　　七八具倒在地上，血流了一片，不过，燕不归还是收拾出来一块干净地方，那里摆着黑贞的尸体。
<br/>　　他咕咚咚灌了一口酒，喉结鼓动，最后目光落在黑贞死去的面庞上。
<br/>　　总有许多话适合搁在心底，他从来就是个沉默寡言的人，不太会说话。今日即便面对一个死人，其实也说不出什么话。只是沉默的喝下一坛坛酒，等身上的伤口愈合结痂，然后，才好去杀更多人。
<br/>　　三日不过一弹指，这一日下午，程大雷出现在一处码头上。
<br/>　　程大雷已买下一座小舟，现在正是海水退潮的时候，乘舟下海，用不了太多时间，海水就可以将程大雷送到目的地。
<br/>　　“大当家，我与你一起去吧？”刘发财持刀立在程大雷身后。
<br/>　　三日时间，他的伤势并未完全痊愈，但已恢复行动能力。
<br/>　　程大雷冲他摇摇头：“你们等我到明天，明天太阳出来前我还没有回来，你们就想办法去凉州。”
<br/>　　程大雷目光落在二人身上，华正则他倒是信得过，这是个有原则的人，坚守言出必行。而对于刘发财……老实说，他不是很信得过。
<br/>　　“大当家放心就好。”刘发财揽住华正则的肩膀：“他不去，我拿刀押着他去。”
<br/>　　程大雷撇了撇嘴。
<br/>　　“大当家可一定要平安回来。”刘发财关切道。
<br/>　　与夫子交过手，程大雷也会过书院的剑阵。老实说，都很棘手，那一夜，如果不是一场突如其至的大火，程大雷还说不好能不能活着离开书院。
<br/>　　而白沙岛上，定然是千军万马，天罗地网，比那一夜的情形更加残酷。
<br/>　　可程大雷找不到不赴约的理由，江湖事，祸不及家人，书院竟然对两个女人动手。每想到这件事，程大雷就咬牙切齿想骂娘。
<br/>　　总也没有太多时间，见潮水已开始退，程大雷牵牛登舟，立在甲板上，顺着潮水向茫茫大海而去。
<br/>　　刘发财看着他远去的背影，忽然想起什么，扯着嗓子大叫：“大当家，大当家……”
<br/>　　“你干嘛，这么远，海风又大，他已经听不见了？”华正则纳闷的问：“你想说什么？”
<br/>　　“唉……”刘发财叹了口气：“我该问问大当家有没有什么遗言留下，这一去高手如云，那里是容易回来的。”
<br/>　　“……”华正则。
<br/>　　差不多同样时间，在琅琊城另外一处靠岸的地方，一艘小船也被冲入大海。
<br/>　　燕不归撑着重剑立在船头，海风将他的黑袍撩起，他无动于衷，目光看着前方，眼神无比坚定。
<br/>　　看起来，他要去的是和程大雷同一目的地。
<br/>　　……
<br/>　　从高处看，白沙岛宛若一片落叶孤单单漂浮在汪洋大海中。这里与其说是岛，倒不如说是海屿更合适。有时候浪太大了，整座岛都会被淹没。海上苦修的书院弟子，就得颠颠坐船离岸。
<br/>　　书院总有不怕死的读书人，宣称要与天斗，与风浪斗，与波涛大海斗，展现读书人的浩然正气。
<br/>　　结果……寻找他们的尸体，总是一件很麻烦的事情。
<br/>　　今日，白沙岛上的人有些多。东海一地的高手，书院培养出来的精英弟子，以及夫子。
<br/>　　“对付姓程的一个人，摆这么大阵仗，还把一个女人绑了。这件事，有些不太地道吧。”
<br/>　　“同程大雷这样的恶贼斗，讲什么江湖规矩，就是要不择手段。”
<br/>　　孟玄青立在白沙岛的最高处，岛上的一切情形尽收眼底，御风而立，他忽地有种世外高手的感觉。
<br/>(本章完)
