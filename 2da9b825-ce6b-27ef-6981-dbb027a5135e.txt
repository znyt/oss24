　　萧逸离去后，进入了野外某处森林，随便找了座大山，轰出个山洞，在洞口做好掩饰后，走了进去。
<br/>　　他要炼制五行丹。
<br/>　　从乾坤袋中拿出九株八宝玲珑参，他心中尽是喜悦。
<br/>　　这九株天材地宝，珍贵无比，耗费了他全部身家才买得。
<br/>　　随便一株，都能让洞玄境武者凭空突破两重修为。
<br/>　　但他不是用来吞服，而是炼丹。
<br/>　　冰鸾剑赋予的丹方，每一个都厉害无比。
<br/>　　上次的金脉丹，效果已经够恐怖的了；这次的五行丹，乃是阶品更高的丹药，其效果有多强，不言而喻。
<br/>　　炼制五行丹的其它材料，他早就备齐了。
<br/>　　如今加上八宝玲珑参，他可以开炉炼丹了。
<br/>　　四品炼药炉，离火金炉；紫炎凝聚而出，材料有条不紊地加入。
<br/>　　完美的炼丹手法，在他手中展现得淋漓尽致。
<br/>　　紫炎，将整个山洞照耀得火光一片；少年，在火炉前双手变幻，潇洒自如。
<br/>　　数个时辰后，萧逸深呼吸一口气，收回紫炎。
<br/>　　而炼药炉中，已经出现了九粒金光熠熠的丹药，美丽至极。
<br/>　　“五行丹，成了。”萧逸自信一笑。
<br/>　　“耗费如此多珍贵天材地宝，才炼制而成，效果必是不凡。”萧逸充满了期待，从炉中拿起一粒，抛入口中。
<br/>　　金色丹药在半空中抛了个漂亮的弧线，最终落入口中。
<br/>　　丹药入腹，化作精纯药力，通过第一条真脉分散，而后游走于全身四肢百骸。
<br/>　　几乎是肉眼可见的速度，萧逸全身都在起着变化。
<br/>　　全身经脉、五脏六腑、气丹、骨骼，都以一种极快的速度增强着。
<br/>　　半晌，当药力全部被吸收，萧逸顿觉舒畅无比。
<br/>　　“原来如此。”萧逸自语道，“五行丹的功效，不提升修为，也不提升实力，而是提升身体的整体基础素质。”
<br/>　　同一时间，他的皮肤上，溢出了层层黑色粘稠之物。
<br/>　　那是他身体内的杂质。
<br/>　　五行丹，显然有着洗经伐髓的功效。
<br/>　　“我的肉体，修炼了修罗战体后，已经无比强大完美。可是五行丹，却还是能淬炼出杂质，效果果然强大。”萧逸惊讶地说着。
<br/>　　五行丹，给他的身体，带来了一次升华。
<br/>　　或者从更简单的角度来说，他的身体变得更强壮和健康。
<br/>　　若他不是武者，只是没有修炼过的普通人。那么，他的身体素质，也可算得上是百病不侵。
<br/>　　接着，他将第二粒五行丹吞入腹中。
<br/>　　药力，通过第二条真脉散发，他的肉体，再一次得到升华。
<br/>　　待得九粒五行丹全部服用，他的身上，已经布满了一层浓浓的黑色粘稠之物。
<br/>　　“体内的杂质，几乎完全排出，我的身体，已然完美。真气在体内，可随意通畅地游走。”
<br/>　　“或许，这已经可以媲美道体了吧。”萧逸满意地笑笑。
<br/>　　所谓道体，这是易老指导他修炼时，他知晓的东西。
<br/>　　武者，修的是武道；但，真正达到洞玄境后，才算真正接触武道为何物；也才真正算踏上武道之路。
<br/>　　而道体，便是天地而生，最契合武道的一种体质。
<br/>　　故，拥有道体的人，乃是天地间的宠儿，最是适合修炼，修炼速度也极快，几乎可以百分百确定必会成为一代强者。
<br/>　　道体，又分许多种。
<br/>　　其中，最出名的便是先天道体。
<br/>　　一般的武者，从凡境修为，到后天，先天，身体在不断强化；凡境炼体，后天炼脉，先天真脉成。
<br/>　　这是一个打好根基的过程，以便日后真正踏入武道之路。
<br/>　　而拥有先天道体的人，一出生，婴儿时，身体已经如先天武者一样，不单止主脉没有堵塞，甚至已经拥有九条真脉。
<br/>　　虽说不是拥有先天境实力，但他们身体通畅，没有杂质，修炼起来极快。
<br/>　　别的武者，从小修炼，十数年才能成为先天境武者，打好根基；这还是天才，否则还不一定能十数年就突破到先天境。
<br/>　　而先天道体，一出生就打好了根基。
<br/>　　两者的差距，可想而知。
<br/>　　哪怕是洞玄境以后的修炼，他们也比普通人更容易感悟武道，突破得更快。
<br/>　　这是天生的差距。
<br/>　　所以，每一个拥有先天道体的人，都必然是天才中的天才。
<br/>　　当然，这样的天才极少，萧逸甚至没有听说过。要不是易老说出来，他根本不知道有道体的存在。
<br/>　　萧逸内视一番，体内，九条真脉，熠熠生辉。
<br/>　　原本一红一篮的冰火气丹，此刻也变得巨大、凝实了许多；两道光芒，极为耀眼。
<br/>　　道体，除了某些幸运之人天生便拥有外。
<br/>　　还能后天通过修炼某些特殊功法而成。
<br/>　　例如白家的白冰雪，她之前就是要炼就雪魄之体。
<br/>　　这种后天而成的道体，显然是比不上先天而成的道体，但也比普通人好得多。
<br/>　　而萧逸如今的身体，显然也可以成为道体的一种。
<br/>　　这由九粒五行丹而成的道体，也不知算什么道体，萧逸姑且命名为冰火道体。
<br/>　　“也不知我这冰火道体，与先天道体，或者雪魄之体，孰强孰弱。”萧逸笑笑自语着。
<br/>　　他不曾见过拥有先天道体的人，也不曾和白冰雪打过交道，自然不知彼此差距。
<br/>　　但他确信，自己的冰火道体绝不弱。
<br/>　　现在的他，见识尚浅，眼光尚不足。
<br/>　　否则，他一定会知道，先天道体，不可能九条真脉熠熠生辉，不可能气丹拥有耀眼光芒。
<br/>　　拥有这种异象的道体，必是完美到极致的道体。
<br/>　　这些他都不知道。
<br/>　　他现在，已经够满足的了。
<br/>　　因为，这将代表着，他的修炼速度会更快，实力提升得更快。
<br/>　　这样，距离他回归萧家的日子，也将更短。
<br/>　　萧家对于北山慕容家而言，什么都不是，甚至不屑被对付。
<br/>　　但，他有一种直觉，某些意外，或者说某些危险，正越来越靠近萧家。
<br/>　　他说不准这些危险来源于哪里。
<br/>　　但他相信自己的直觉。
<br/>　　他的直觉，从未错过。
<br/>　　只有拥有足够的实力，他才能将这些危险和意外泯灭掉。
<br/>　　......
<br/>　　在山洞中，他休息了一晚。
<br/>　　第二日天蒙蒙亮，他便马不停蹄地赶去猎杀妖兽。
<br/>　　半月后，二十个任务上的妖兽，尽皆被他斩杀，他也赶回了裂天剑派。
<br/>　　对于普通剑派弟子来说，到野外或者森林中击杀妖兽，是比较危险的行为。
<br/>　　有妖兽的地方，必然危机四伏。
<br/>　　很可能，任务指定的妖兽还未找到，率先就遇上其他妖兽的袭击了。
<br/>　　又或者，误入其他更强妖兽的领地，深陷危机。
<br/>　　总之，在外历练，危险多多。
<br/>　　所以，普通剑派弟子完成这些任务，一般比较难，也耗费时间。
<br/>　　但，对于一个经验丰富的猎妖师来说，只要掌握了这些妖兽的信息和行踪，将它们一一斩杀，并不难。
<br/>　　而对于萧逸这个天才猎妖师来说，更是轻而易举。
<br/>　　半月时间，他已然完成任务，并回到剑派。
