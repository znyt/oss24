　　第117章
<br/>　　蛇矛已经生锈，看不出本来模样，但入手极沉重。
<br/>　　“这兵器你想卖多少钱？”
<br/>　　“一百两。”
<br/>　　“嗯？”程大雷心道竟然这么便宜。
<br/>　　年轻人目光闪烁：“给十两银子就行。”
<br/>　　“咦？”
<br/>　　“您多少给点，别让我白跑一趟就行。”
<br/>　　看来这年轻人也未必知道这兵器的价值，喊出一百两只是想撞撞运气，在见到程大雷眼神后立刻就怯了。
<br/>　　本来一百两程大雷还有些犹豫，谁知道这兵器是不是传说中那柄。但价钱如此便宜，对程大雷来说，也只是洒洒水啦。
<br/>　　程大雷掏出十两碎银，塞到年轻人手中，然后抬手抄起地上的蛇矛。
<br/>　　蝴蝶扇一扇翅，多少事情被改变，英雄埋没荒野，本该吟血的神兵也蒙尘，程大雷将兵器抛给张肥。
<br/>　　“喏，这是你的了。”
<br/>　　不知是不是错觉，在张肥接到兵器时，程大雷隐隐觉得那蛇矛发出一声低吟，宛若龙鸣。
<br/>　　是因为物归原主么。
<br/>　　耿星河拍手笑：“恭喜欧公子得到宝物，不知这兵器有什么奇特之处，公子说来，也让我们开开眼界。”
<br/>　　其他客人脸上也投来笑意，至于这笑容是表达善意，还是有嘲笑程大雷被人骗了的心思，那就说不好了。
<br/>　　“一件小玩意儿，不足挂齿。”程大雷坐回酒桌。
<br/>　　耿星河报以干笑，又重新提起刚才的话题：“欧公子怎么一直不吃东西，是饭菜不合口么？”
<br/>　　“哈……蛮合口的。”程大雷象征性的夹了一筷子，可脸上的表情却如同嚼蜡。
<br/>　　“……”耿星河。
<br/>　　大家都在大块朵颐，虽然不是为吃饭来的，但再如何说，渔阳楼的手艺还是不错的。可程大雷这种表情是什么意思，大家吃得都是猪食么。
<br/>　　“如果不是很可口，欧公子也不要勉强，让客栈再做就是。”耿星河道。
<br/>　　“怕他们也做不出来我想吃的菜。”程大雷摇摇头，冲身边的柳芷道：“把我们带的辣椒交给厨房，让他们做几道菜送来。”
<br/>　　“是。”柳芷答应一声去了。
<br/>　　耿星河道：“欧公子，你刚才提到的辣椒是什么东西？”
<br/>　　“是我家乡的一种土特产，不算什么。”
<br/>　　“不知欧公子这次来扬州，带了什么货物？”
<br/>　　“今天主要是喝酒，不谈生意的事情。”
<br/>　　这时耿星河有些摸不着头脑，今夜大家谁不是为做生意来的。难道眼前这位是某个大世家的子弟，狂到不把银子当银子的地步。
<br/>　　二人继续在一起闲侃。过不多时，有岳阳楼的伙计端着食盒进来，将里面的食物搁在程大雷面前。
<br/>　　“爷，您往锅里放的是什么东西，一放进锅里，整个厨房都冒香气，馋得我们厨子口水都流出来吧。”
<br/>　　“你没有偷吃吧？”
<br/>　　“这个不敢，不敢，您的人一直在旁边看着。”
<br/>　　就连耿星河都向盘子里投来目光，见程大雷立刻开始大块朵颐，看他的吃相，像是吃得天上龙肉一般。
<br/>　　“欧公子吃得就是那辣椒么？”
<br/>　　“耿兄，可以一起尝尝？”
<br/>　　“那就却之不恭了。”
<br/>　　耿星河夹了一筷子，立时感觉辣味在口腔里炸开，眼泪差点落下来。
<br/>　　程大雷哈哈笑道：“第一次尝到都是这样，来，把菜分给大家，让大家都尝尝。”
<br/>　　辣椒在桌子上传开，诸人还是第一次尝到这种食物，随之便造成不错的反响。
<br/>　　耿星河放下筷子，拍拍手道：“太辣了，我是吃不惯的，不过这种食物若是拿出去，当是能换些钱的。”
<br/>　　耿星河这种人，凭借商人的直觉，他就能觉察到其中的商业价值。
<br/>　　“欧公子，这辣椒你还有多少？”
<br/>　　“倒是有一些的。”
<br/>　　“卖嘛？”
<br/>　　程大雷注视着耿星河的眼睛，轻轻摇摇头：“来，喝酒，今天咱们不谈生意。”
<br/>　　“……”耿星河。
<br/>　　……
<br/>　　次日天明，程大雷就迎来了登门拜访的人。都是昨夜酒宴上见过的客商，对方开口便要二三百斤辣椒，价钱也出得足够高。
<br/>　　程大雷直接把人哄出去了，他们根本不是程大雷要等的人。
<br/>　　在中午时分，耿星河亲自登门，程大雷把对方请到屋里，嘱咐柳芷沏上一壶茶。
<br/>　　这才是程大雷要等的人。
<br/>　　“欧老弟，我已经让人去码头问过，听说你带来一船的货，我冒昧猜一下，船上都是辣椒？”
<br/>　　这才是聪明人呐。程大雷看了耿星河一眼，直接问道：“什么价？”
<br/>　　“痛快，老弟你要什么价？”
<br/>　　“我不要钱，我要粮食。”
<br/>　　“多少？”
<br/>　　“三十万斤。”
<br/>　　耿星河沉默片刻，道：“这辣椒只有你一个人有么？”
<br/>　　程大雷点点头。
<br/>　　“好，这船货我要了，三十万斤上好稻米，十天内凑齐。”
<br/>　　程大雷摇摇头：“我等不了十天。”
<br/>　　“那七天。”
<br/>　　“抱歉，我也没有七天时间。”
<br/>　　“连七天都等不了么？”耿星河咬咬牙：“五天，最少五天！除了三十万斤稻米，我再把胡姬送给老弟。让她陪老弟快活几天，等五天后，我把三十万斤稻谷装上船。”
<br/>　　程大雷当真有几分动心，却又无可奈何的摇摇头。
<br/>　　耿星河一怔：“五天都不行。”
<br/>　　“我很想答应耿兄，但奈何我真的没有这么长时间，我最多再等到明天。”
<br/>　　耿星河也没有纠缠：“那既然这样，我就不叨扰老弟了，但老弟，在一天之内能拿出三十万斤稻米的，怕是没有几个。”
<br/>　　“那也没有办法，我只能再等等。”
<br/>　　看着耿星河离去的背影，程大雷轻叹口气，事情就真的做不到么。
<br/>　　在下午时分，程大雷等到一个他绝对想不到的人。是漕帮的孙德龙，他穿着一身黑衣，脸上堆满谄媚的笑。
<br/>　　程大雷摸不准对方的笑意，摆摆手道：“坐。”
<br/>　　“公子在这儿，哪有我坐的份儿，我站着伺候就行。”
<br/>　　程大雷更加糊涂：“你来找我有什么事？”
<br/>　　“我斗胆问一句，阁下可是……”孙德龙比划出一个六的手势。
<br/>　　这是什么意思？程大雷皱起眉头，也同样比划出一个手势：“六六六？”
<br/>　　
<br/>(本章完)
