　　第270章
<br/>　　林掌柜立在门口，目送着四人远去。他这才掏出那枚玉佩，对着太阳照了照，通体翠绿，握在掌心像握着一滴水。西北本就产玉，可这样的宝玉也十分罕见，恐怕也就琴川关那样的地方才能拿得出来。
<br/>　　正在此时，有一玉带锦冠的男人从远处过来，走进门口道：“林四，干嘛呢，在门口杵着？”
<br/>　　“大官人……”林四点头哈腰：“我刚接了一单生意，要一船雄黄，这是他们付的订金。”
<br/>　　“要这么多雄黄？”西门大官人接到玉佩时眼睛一亮：“是块好玉呐，出手挺大方的，正好让我拿去送人。”
<br/>　　“是西市的潘娘子么，潘娘子肯定会喜欢的。”林掌柜陪着笑。
<br/>　　“不是她，换人啦。”西门官人打量着手中的玉佩：“是隔壁花家的瓶姐儿……”
<br/>　　“……”林四：您老这手可是够快的。
<br/>　　不过他也知道自家东家的风格，从祖辈继承来产业，东街逛，西街混，偌大家业混得越来越少，身边女人却是越来越多，浩甲城的大姑娘小媳妇，没少被东家得手的。关键是，被他得手后的女人还没一人说他不好……啧啧，想起来东家的为人也值得称赞。
<br/>　　“他们要那么多雄黄干嘛？”西门春长。
<br/>　　“应该是驱湿毒吧，反正琴川关那边，用这些比较多。”林掌柜。
<br/>　　“言之有理，赶快准备吧，琴川关那里可不能得罪，交货时，便是银子上吃点亏，也莫要计较。”
<br/>　　“是是，我已经让人去准备了，咱们可能还一时拿不出这么多火，还要去别家药铺借一些。”
<br/>　　……
<br/>　　三天后，一艘货船在码头上装货，程大雷四人皆在，他们这三天内就住在浩甲城。
<br/>　　“李公子在哪里，李公子在哪里？”
<br/>　　正这个时候，一道呼声响起，大家看到一个短脖大头的男人撩着袍子跑过来。
<br/>　　“李公子，这位就是我家西门大官人。”
<br/>　　程大雷愣住了，传说中潘驴邓小闲的西门大官人竟然就是这份尊荣，脑袋大，脖子粗，不是大款就是伙夫啊。
<br/>　　“一早听到李公子要来，就想来看李公子，可总找不到时间，现在总算赶上了。”
<br/>　　这西门春长身后还跟着一个女人，柳条腰一吹就倒，绣帕遮面，盈盈一双桃花眼偷眼打量着几人。
<br/>　　程大雷春心荡漾，心道这女人是不是对我有意思呐。再看李行哉却是神态坦然，似乎没有瞧见女人抛过来的媚眼。
<br/>　　程大雷想想也是，人家是王子，想要什么样的女人没有，怎么看的上一个妇人。相比之下，程大雷在女人的见识上就太少了。
<br/>　　“哈，这位是在下的女伴，这次陪在下一起押船。”
<br/>　　“大官人，你亲自押船？”林掌柜吃惊道。
<br/>　　“是啊，有什么不可以么，这次李公子的货我亲自送。”
<br/>　　程大雷撇撇嘴，心道是不是普天下姓西门的都对女人这么上心，怎么看这西门春长也不是工作狂，大概是这女人想坐船了，所以他就特意过来。
<br/>　　啧啧，还真是好男人的标榜啊。
<br/>　　船离开港口后，沿着河道向西，一路风平浪静，站在船头望去，水泊荡漾，芦苇连绵。
<br/>　　西门大官人和他那叫瓶儿的女伴，立在船头有说有笑，倒也没有刻意去和李行哉套近乎。
<br/>　　程大雷和徐神机窝在船舱里，偷眼看着这一幕，心中啧啧称奇。
<br/>　　“你说这男人有什么好的，又丑又矮，是怎么糊弄住这么漂亮的女人。”
<br/>　　“大概是有钱吧？”徐神机道：“有钱什么女人找不到。”
<br/>　　“有钱容易，有钱还能把女人哄得这么开心就不容易了。”程大雷一边偷看一边道：“所以可能不仅仅是钱的事儿。”
<br/>　　“欧兄弟，你们在这儿干嘛呢？”
<br/>　　林掌柜突然到程大雷身边，倒是将程大雷和徐神机吓了一跳。
<br/>　　“没什么，没什么，聊一些技术细节。”程大雷说着直起身来。
<br/>　　“喔，咱们再有两个时辰就能到琴川，我想问一下，派来接货的人准备好了吧？”
<br/>　　“早就准备好了。”徐神机拍着胸脯，忽然想起什么，道：“林掌柜，你们跑这条水路，就不怕遇到强盗么？”
<br/>　　“嘿，哪里有强盗，自从莫将军到琴川后，附近的山贼水匪都没踪影了。这里方圆百里，一个强盗都没有。”
<br/>　　黑吃黑呀，还吃得这么干净。
<br/>　　“假如遇到呢？”程大雷问。
<br/>　　“没有假如！”林掌柜很坚定。
<br/>　　“万一遇到呢？”
<br/>　　“没有万一！”林掌柜斩钉截铁。
<br/>　　“万一，我就是呢？”程大雷眯眼笑道。
<br/>　　“欧兄弟，你可真是会……”林掌柜抬头触及到程大雷的眼神时，猛然一愣，他忽然醒悟道：这个眼神，不对！
<br/>　　只听程大雷口中呼啸一声，一脚将林掌柜踹倒在地上，口中大叫。
<br/>　　“打劫啦，打劫啦，蛤蟆大王开始打劫啦，男人站左边，女人站右边，不男不女站中间……”
<br/>　　货船上一片大乱，李行哉握着根木棍躲在船舱内，口中道：“开始了么，开始了么，我这是第一次，有点紧张？”
<br/>　　“你上次不才劫过么？”福德勒。
<br/>　　“那也才第二次，我小小紧张一下不可以么？”
<br/>　　“也不是第二次，你在青州偷过鸡，在草原骗过人羊羔，上次还抢过馒头。”
<br/>　　“好啦！”李行哉忽然喝了一声，举着木棍冲了出去：“打劫啦……”
<br/>　　船上约有十几个伙计，此时都已被程大雷放倒，降下白帆，船就停在河边。
<br/>　　此时船上的人都抱着脑袋蹲在一角，蛤蟆寨的弟兄已经从水里上了船。
<br/>　　西门大官人此刻才醒悟，自己是中了圈套，他抱着脑袋问：“你们是什么人？”
<br/>　　“说起我的名字，可是大大的有名……算了，说出口你们也不知道。”程大雷举着剑道：“说说吧，你们是想吃馄炖，还是想吃板刀面？”
<br/>　　“什么是馄炖，什么是板刀面？”
<br/>　　“没文化了不是。”程大雷道：“混沌就是你们跳下去淹死，板刀面就是我把你们宰了，再把你们丢下去。”
<br/>(本章完)
