　　混沌之中，一道道凶猛的混沌之气如同旋涡一样狂涌而来，仿佛要将一切摧毁，而旋涡的中心，却是一个大蛋。不管有多少混沌之气，全部都被巨蛋吸收，这些狂暴的混沌之气就好像是没是威力一样。实际上并不是这样的，魔神生于混沌，与混沌的关系，就好像鱼鱼海水一样。而这些混沌之气，却被魔神吸收，日复一日，魔神越来越强大。
<br/>　　“大道三千，条条可证道。大道之下，衍生法则，因果循环，大道无边……
<br/>　　执此法则，掌因果，为诸果之因，是为因果之主……魔神传承，因果相传，无处不在。”一道玄之又玄的声音，在元释的元神之中响起，缓缓流淌，这声音，包含道韵，是一种道的体现，让元释沉迷。
<br/>　　从这传承之中，他明白了许多，又好似出现了更多不明白的东西，这样的矛盾，使得元释对着神魔的传承，有了更深层次的明悟。
<br/>　　混沌不记年，就这样过了不知多久，某一瞬间，轰的一声，巨蛋炸开，魔神庞大的身躯一瞬间涨到万丈至高，而且还在继续增长，直至百万丈，好似到了极限，这才不再增长。紧接着庞大的身躯猛地一震，混沌之中瞬间掀起了风暴，向着四面八方扩散出去，让混沌震动。
<br/>　　那人首蛇身的庞大魔神，猛然间睁开双眼，放出两道摄人的黑白光芒，全身散发着黑白两色光芒，昂首一吼，他的气势轰然崛起，像是要与天争高，与混沌比广。顿时间，风云变色，混沌倒卷。
<br/>　　“吾为因果大道，混沌魔神元释，今日出世，当为因果之主，大道鉴之！”
<br/>　　浑厚的声音响彻混沌，那道身影屹立苍穹！
<br/>　　与此同时，混沌虚空之上，九条法则链条相互交织，若隐若现，尤其是第九道法则，黑白两色，若隐若现。混沌之中，九条法则的道韵也越来越浓。如此一来，对于大道的领悟，也就更加容易了。
<br/>　　同时，混沌之中，另外八个不同的方向，也同时释放出八股不同的威压，那威压瞬间就弥漫整个混沌，顿时引起了混沌风暴，久久不能停歇。不久，那八股威压也熄灭了。
<br/>　　同时，元释也知道了如今已有九个混沌魔神孕育而出，而其他八个一点也不比自己弱，甚至更强。
<br/>　　其实，那股威压是元释故意释放的，他只是想试试这混沌如今水有多深，到底有多少人已经孕育而出。如今，得到了结果，元释也不敢耽搁，他抓紧时间，闭目参悟起法则来。
<br/>　　他知道，以因果法则的威力来说，怎么也可能是排名前五苏醒的。但，现实情况确是他排名第九，他知道，这是因为他的意外到来而导致的，现在知道混沌中还有八个比他先苏醒，他如今哪敢怠慢，连忙闭目参悟法则。
<br/>　　……
<br/>　　这一刻，在苍茫混沌之中的某一处，有着一个混沌旋涡，旋涡中心，一尊庞大无比的古兽睁开双眼，在这风暴内，古老的声音传出：“第九个苏醒的魔神，是因果魔神？”
<br/>　　另一处混沌之中，一个坐落在巨石之上的巨人，他生有三颗头颅，六只臂膀，此刻，他蓦然睁眼，仰望混沌，感受到虚空那新出现的法则之力，目光深邃，露出凝重之色，幽幽道：“因果之道？越先苏醒的魔神就越强大，不知他有如何？不知那最后的混沌劫，又有谁能度过？哎！”
<br/>　　另一处混沌深渊内，一个生有一千只手臂，每一个手臂的手掌之中，都有一只眼睛，每只眼睛之中，都放射出不同的光芒，有贪婪，有淡定，有凶狠，有慈祥，有悲怨，有欢喜，有毁灭，有怜生……千中情绪，集于一身！同时，一道古老目光投向混沌：“第九个，嘿嘿！是因果道？呵呵，那又如何？”
<br/>　　混沌一处风暴之中，一个手提巨大斧头的巨人，看上去是那么威武霸气，他前进的脚步突然一停，道了一句：“因果之道，小道耳！”继续前进。
<br/>　　另一处混沌隐秘之处，一个头顶散发无量造化之光的玉牒的神秘人，目光没有丝毫波动，好像新出现的魔神在他庙前不值一提一般，继续参悟他的大道。
<br/>　　……
<br/>　　不同的混沌魔神对于这位新出现的因果魔神有不同的反应，有轻视的，有不屑一顾的，当然，也有重视的。仅仅如此罢了，并没有害怕的。
<br/>　　他们都是混沌魔神，无上强大的存在，他们都是大道之下去，掌握前十法则之修，对于新出现的魔神，彼此都有比较之心，也都有自己的傲气，又怎会让他们服气别人？他们并不会认为自己就比别人弱。
<br/>　　不管别人怎么想，元释自己却在抓紧时间领悟法则，他现在刚刚孕化而出，虽然排在前十，但他不会自大到自己就一定比后来者厉害。他比任何人都清楚，他们的敌人是能够开天辟地，开出大千世界的大神盘古。现在再不努力，以后就没时间努力了。
<br/>　　在这个世界，证道分四步走：
<br/>　　第一步：蜕凡证仙。这一步追求褪去凡身，证得仙身，当然，这个“仙”并不是真正意义上的仙，仅仅只是一个境界。真正的仙是走仙道极致，永恒不朽的存在。这一步又分为炼精化气、练气化神、炼神返虚、炼虚合道四个境界。
<br/>　　第二步：寻仙证圣。这一步已经开始接触法则了，通过参悟法则一步步达到“圣境”。这个“圣境”是指混元大罗金仙这个境界，同时也指是“圣人”。但混元大罗金仙不为圣人，而圣人却为混元大罗金仙。这一步有分为人仙，地仙，天仙，玄仙，金仙，太乙金仙，大罗金仙几个境界。
<br/>　　第三步：混元证天。这个“天”指的是“天道境”，而不是“天道”或“天道圣人”。这一步为混元大罗金仙十二重天，或者是圣人十二重境。
<br/>　　第四步，也是最后一步，合道证不朽。这里证得是大道圣人，也为大道境，或者不朽境。
<br/>　　而第二步与第三步之间，还有一个准圣境，严格来算，要算在第二步。当然，这个准圣境只是后来的鸿钧道祖自己开创的。而，混沌魔神并没有这一境界。多出了这么一个境界，以后第二步强者通过积累，成就准圣之后，在冲击第三步，就会容易许多。
<br/>　　因为是前十个孕化而出的混沌魔神，而且还掌握排名前十的大道法则，所以元释现如今的境界还只是刚过第二步，踏上第三步的开始，也就是混元大罗金仙第一重天境界。以后的魔神，就算出现了，苏醒时最多也只是大罗金仙之极。
<br/>　　别看这个境界对比后世已经很高了，那是因为混沌魔神追求的是混元之极直接证大道圣人，而不是天道境，他们哪一个不是大道孕化而成的？哪一个不是直接修炼大道而不是天道？哪一个不是心高气傲之辈？又怎会甘心只证一个小小的天道圣人或者天道境？要是后者还好说，与天道地位相同，也不受天道控制，要是前者就惨了，随时会受到天道的控制，自由不受自己控制，虽然圣人是代天牧守苍生，是天道的代言人，能够近距离的参悟天道，但天道不是大道，虽然天道也很厉害。那时参悟大道法则就会变得极慢，进步也很慢。
<br/>　　所以，现在能拼，每一个都会去拼，在他们眼中，只有证道。谁挡在他们之前，谁就死。如同盘古，他修炼的是力之道，力之极致，开天辟地。盘古想要证道，唯有开天辟地。若是任由盘古开天，到时候混沌就会残缺，道基就会不全，因为大家同为混沌魔神，修炼的都是混沌大道，没有了混沌，以后还怎样证道。
<br/>　　所以，盘古要开天证道，就是等同于断别人道基。断人大道，乃生死大仇。其他混沌魔神到时都会前去阻止，就是拼了命也要阻止。
<br/>　　不成道，毋宁死！
<br/>　　时间就在众多魔神陆续苏醒中流逝，随着混沌魔神苏醒的越多，混沌也变得越来越乱，一场风暴，就要席卷整个混沌。
