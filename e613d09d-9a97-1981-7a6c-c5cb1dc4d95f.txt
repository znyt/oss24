        &lt;h3&gt;第三卷第二十九章  好丈夫张函&lt;/h3&gt;
<br/>
<br/>        陆玉回来了，张函就打消了出去的念头。
<br/>
<br/>        他一如既往地是个好丈夫，去陆玉单位接陆玉下班。
<br/>
<br/>        他换下了工作服，身穿一件干净的白色衬衫，扣子只松松系了下面几个，而上面却敞开着，露出健壮的小麦色肌肤，下身是一条深色的西服裤，毫无褶皱。
<br/>
<br/>        好看而清澈的丹凤眼，像是藏有琥珀一样地明亮，高挺的鼻梁，嘴唇微厚，紧紧的抿着，手中拿着一束玫瑰花，站在陆玉的单位门前。
<br/>
<br/>        “陆总，你家张总又来接你了。”陆玉办公室的同事笑着对陆玉说。
<br/>
<br/>        陆玉从窗户看下去，张函站在哪里，帅气得让人眼花，三十来岁的年纪，看起来也就二十多岁的样子。
<br/>
<br/>        自从张函说他忘记了之前的事情之后，他对家里的人都很好，而且也上进，尤其是对陆玉，但是陆玉却知道，他们只是表面看着好，实质性的事情他们却从来都没有接触过。
<br/>
<br/>        张函开始健身，开始保养，很是注重自己的生活，原本看着病歪歪的身子，经过他自己的锻炼和保养，已经有了一身的肌肉，再加上这几年工作上的历练，他身上自然而然散发出来一股摄人的气势。
<br/>
<br/>        陆玉叹了口气，年少那么紧紧的跟着张函，谁知道嫁过来之后，他和她都不住在一起，俩个人真正的是相敬如冰，在外人看来他们就是恩恩爱爱的夫妻。
<br/>
<br/>        婆婆和妈妈常常催着他们要一个孩子，可是她到哪里去生孩子？期间和一个男朋友在一起，是怀过一次孕，但是她哪里敢要啊。
<br/>
<br/>        “你们夫妻真是恩爱呢，这多少年了，张总还一如既往地的对你这么好。”同事感叹的说。
<br/>
<br/>        “呵呵呵。”陆玉
<br/>
<br/>        “你这一次调走，张总同意吗？”
<br/>
<br/>        “他不同意，但是是上头决定的，我和他都没有办法啊。”
<br/>
<br/>        “是我，我也不愿意啊，你们年龄那么小，就两地分居。”
<br/>
<br/>        “现在交通不是方便吗，我也就是过去过度，过两年就回来了。”陆玉边说便拿起包，和同事道了再见。
<br/>
<br/>        “你来了。”
<br/>
<br/>        “嗯，我们一起出去吃饭？”张函替陆玉拉开车门。
<br/>
<br/>        “算了，我要回我妈家里去，你要不要去？”
<br/>
<br/>        “那我们就一起去吧。”
<br/>
<br/>        陆家在胡同巷里，这里原来是一些达官贵人住的地方，陆家算是几大家族的人，是后起之秀，他们原本不是京城人，只是有些贡献，所以就住在了这里。
<br/>
<br/>        今天陆家人都在家里，原本也就是探讨陆玉调离京城的事情。
<br/>
<br/>        张函送陆玉回去，陆宇霆正在逗他四岁的儿子玩，他看见张函来了，对儿子说：“去，找妈妈去，我陪张叔叔说说话。”
<br/>
<br/>        陆玉牵着陆宇霆的儿子说：“走了，找妈妈去了。”
<br/>
<br/>        “你今天咋有空过来了？”俩个人坐在客厅里。
<br/>
<br/>        “我去接陆玉，她说要回来我就一起来了。你现在是悠哉悠哉啊，下班回来陪着娇儿美妻，日子过得是幸福滋润哦。”张函打趣。
<br/>
<br/>        “你也可以啊，你和陆玉不是也很好吗，就是没孩子，你们俩个是谁的问题？”
<br/>
<br/>        “我的问题，我不育。”
<br/>
<br/>        “你不育？”陆宇霆吃惊的问，当然，陆宇霆一直认为张函没有正面见过陈子昂，也是不知道自己有孩子了。
<br/>
<br/>        而且他也从来都没有给张函说过有关陈子昂的事情，俩个人就那一次在饭店的时候碰上，心照不宣的再也不提陈子昂的事情，似乎他们都不愿意触碰自己所谓的心里的那道光。
<br/>
<br/>        张函不育，那陈子昂的那两个孩子是谁的？莫非陈子昂还会有别的男人？想到这里，陆宇霆觉得心里一阵荒凉，他当初可是看上了陈子昂的清纯和坚强。
<br/>
<br/>        “是啊，我的问题。不过你得为我保密，不能让别人知道了。”张函坦荡荡的承认。
<br/>
<br/>        “陆玉要去甸城，有和你商量吗？”陆宇霆问。
<br/>
<br/>        “我才知道时间不久。”
<br/>
<br/>        “父亲说甸城有一部分产业，让她过去也好，能照看着点。怕是你要辛苦一阵子了。”
<br/>
<br/>        “甸城还有产业啊？”
<br/>
<br/>        “我也是才听说，并不知情。”
<br/>
<br/>        俩个人正聊着，陆战君和蔡少芬回来了。
<br/>
<br/>        “张函过来了啊？”
<br/>
<br/>        “嗯，爸妈，你们回来了？”张函站起来
<br/>
<br/>        “坐吧，坐吧，又不是外人，那么客气干嘛。”
<br/>
<br/>        蔡少芬是知道张函的情况的，看着张函也一表人才，可是女儿跟着他.......
<br/>
<br/>        作为一个女人，哪里还有幸福可言，不过张函这几年确实表现的不错，他们谁也挑不出来张函的毛病，再说两家关系如此，况且当初也是陆玉自己选择的，也只能如此了。
<br/>
<br/>        而且，张家的势力也在慢慢的恢复当中。
<br/>
<br/>        陆玉从甸城回来就一直闹着要去甸城上班，把他们夫妇都闹得没有了脾气，只好找陆老爷子，陆老爷子才动用了一些关系，为陆玉调动了工作。
<br/>
<br/>        陆玉去了外地，过上三年五载的，回来也就能升一级，这对她也是有好处的。
<br/>
<br/>        他们家和秦家关系不错，两家相交也有很长时间，有一些生意上的往来，陆玉过去也是有很大的好处的。
<br/>
<br/>        目前各大家族手中都有自己的经济支撑，他们陆家就有点力不从心了。
<br/>
<br/>        让陆玉过去再开拓开拓关系，也盯着点秦家，自是好的。
<br/>
<br/>        “张函，玉儿要去甸城，没有给你提前说，我们也考虑了玉儿过去对她的前程有帮助的。”陆站君对张函说。
<br/>
<br/>        “玉儿工作我不托后腿，只是她走这么远，我不放心。”
<br/>
<br/>        “没什么不放心的，你知道，谷家和秦家本身就是我们的朋友，这些年一直都和他们有往来，我们在那边也投资了一些项目，因为也没有人打理，每年的收入也不多，玉儿过去盯着他们。”
<br/>
<br/>        “原来投资的有项目啊，我还以为仅仅就是关系好呢。”
<br/>
<br/>        “张函，你如今和玉儿是夫妻，我们两家是一家人，陆家和张家同气连枝，陆家好了张家自然也就好了，玉儿过去也就是一段时间，你如果想过去了我也想想办法，你也过去。”
<br/>
<br/>        “爸，我过去也就是过度，难道你还让我们夫妇一直呆在那里给你看家？”陆玉不高兴了。
<br/>
<br/>        “我不是怕你过去没有人照应吗？”
<br/>
<br/>        “我和谷正娟还有少卿他们关系都好，我怕什么啊？”陆玉怎么可能让张函过去呢？
<br/>
<br/>        “爸，你们决定了就不用改了，我会经常去看玉儿的。”
<br/>
<br/>        “是啊，爸，反正交通方便，说去就去，说回来就回来。”
<br/>
<br/>        陆家的事情张函是没有发言权的，但是张家的事情陆家可是有发言权的。
<br/>
<br/>        陆玉去甸城已经成了定局。
