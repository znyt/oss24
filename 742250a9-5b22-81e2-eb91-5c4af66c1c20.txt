　　“这……怎么可能！”
<br/>　　不但百里长松满脸震惊，周围的围观众人，也全部看傻了眼。
<br/>　　“这家伙，竟然如此轻松，就破解了百里前辈的招式？”
<br/>　　“而且还能将百里前辈击退，这是何等的实力？”
<br/>　　“他年纪轻轻，怎么会有如此恐怖的战斗力？”
<br/>　　……
<br/>　　“你……你到底是什么人？”
<br/>　　百里长松两眼死死盯着叶尘，再次大吼了一声。
<br/>　　能从他的控制中挣脱开来，对方至少也是和他一个级别的强者。
<br/>　　可是，如此年轻的强者，他还闻所未闻，便是他们魔界那位第一天才，也和此人相差甚远。
<br/>　　因而，叶尘的身份，让他有些惊疑不定。
<br/>　　叶尘冷冷一笑，
<br/>　　“凭你，还不配知道我的名字！”
<br/>　　说完这话，叶尘也来的再和他继续纠缠，直接双指作剑，冲百里长松猛然一指。
<br/>　　轰！
<br/>　　一道璀璨的紫色剑芒，犹如流星般，直奔百里长松爆射而去！
<br/>　　“什么！”
<br/>　　百里长松再次脸色狂变，
<br/>　　因为他从这道剑芒之上，无比清晰的感觉到，一股强大的威胁感！
<br/>　　‘冲霄境！此人绝对是冲霄境！’
<br/>　　百里长松在心头狂吼，已经来不及去思考，魔界何时有了如此年轻的冲霄境强者，当即脚下暴退，迅速退到了酒馆门口。
<br/>　　“嗯？”
<br/>　　就在这时，那道令人心悸的剑芒，却忽然消失不见了。
<br/>　　‘难道刚刚是我的错觉？其实对方并没有那么厉害？’
<br/>　　百里长松见状，不由得松了一口长气，可是心里总感觉哪里不对头。
<br/>　　就在这时，
<br/>　　轰！
<br/>　　那道突然消失的剑芒，竟陡然出现在了百里长松的后方！
<br/>　　“什么！穿梭空间？这是什么武技？”
<br/>　　等百里长松反应过来，已经为时已晚，那道紫色剑芒，瞬间便穿透了百里长松的脑袋。
<br/>　　彭！
<br/>　　百里长松的脑袋登时爆碎，然后尸体缓缓倒了下去。
<br/>　　轰！
<br/>　　整个酒馆里的众人，看到这一幕，顿时个个目瞪口呆。
<br/>　　成名数十年的传奇人物百里长松，竟然被一个不知名的年轻人，一招秒杀了？
<br/>　　若非亲眼所见，任谁也无法相信，这竟然是真的！
<br/>　　“冲霄老祖！这定是一位冲霄老祖！”
<br/>　　“传说上古的萨满一族，有返老还童之术，数百岁还年轻如少年！”
<br/>　　“定是如此！不会错了！”
<br/>　　……
<br/>　　哗啦啦！
<br/>　　众人反应过来之后，顿时纷纷跪倒一片，冲着叶尘顶礼膜拜起来。
<br/>　　在魔界之中，冲霄境强者被称为冲霄老祖，是魔界至尊级的人物，受万人敬仰，普通人见到，都必须要跪拜行礼才行。
<br/>　　“噗通！”
<br/>　　赫连鹏飞愣了好一会，回过神来之后，也当即跪倒在了地上，然后磕头如捣蒜，拼命的哀求起来，
<br/>　　“前辈！我错了！是我错了！我愿意自断双臂，求您饶我一命！”
<br/>　　如果赫连鹏飞事先知道叶尘的真实实力，他说什么也不敢，去找一位冲霄老祖的麻烦。
<br/>　　然而，面对赫连鹏飞的哀求，叶尘一脸漠然，
<br/>　　“我给过你活命的机会，可惜你没有珍惜！”
<br/>　　说完这话，叶尘缓缓抬起了手臂。
<br/>　　赫连鹏飞见状，顿时差点都吓尿了，眼看哀求不成，当即又厉声道：
<br/>　　“就算你是冲霄老祖又如何？我赫连将军府一共有四位冲霄老祖，你若是杀了我，你也休想活着离开魔皇城！”
<br/>　　“哦？是吗？那我倒是很想试一试！”
<br/>　　说完这话的同时，叶尘的手掌已经缓缓落下。
<br/>　　轰！
<br/>　　一道巨大的金色巨掌虚影，重重朝赫连鹏飞砸落下来。
<br/>　　“不！！”
<br/>　　赫连鹏飞顿时大吼了一声，然后转身拔腿便要逃跑，可惜在那股强大的气势压迫之下，他根本无法挪动身体，只能眼睁睁的看着那道巨掌落下。
<br/>　　“你会后悔的！你一定会后悔……”
<br/>　　赫连飞鹏不甘的大吼一声，不过没等他的话说完，便已经彻底淹没在那巨掌之下，瞬间变成了一堆肉渣。
<br/>　　“嘶~~~”
<br/>　　整个酒馆里的众人，看到这一幕，顿时纷纷倒吸了一口凉气。
<br/>　　“他竟然……真的把赫连公子给杀了！”
<br/>　　“纵然他真的是冲霄老祖，那赫连将军府，可是足足有四位之多！难道他不要命了吗？”
<br/>　　“嘘！小点声，小心别被他听到，否则会要了你的小命！”
<br/>　　……
<br/>　　将众人的议论声听在耳中，叶尘也不由得暗暗感叹，
<br/>　　他无意招惹麻烦，可是麻烦却总是自己找上门来。
<br/>　　不过，身为修真之人，当直抒胸臆！
<br/>　　管他是谁，招惹到自己头上，杀了便是，却也没有什么好后悔的。
<br/>　　“掌柜的，现在这个人死了，天字号房间可以给我们了吧？”
<br/>　　叶尘轻描淡写的轰杀了赫连飞鹏，转而朝酒馆掌柜淡淡道。

<br/>　　“是是是！我这就带前辈去包厢！”
<br/>　　说着，酒馆掌柜颤巍巍的从地上爬起，领着叶尘一行人，朝天字号包厢走了过去。
<br/>　　众人看到这一幕，顿时再次愕然，
<br/>　　“这……杀了赫连将军府的嫡系公子，竟然像没事人一样！”
<br/>　　“难道他就一点也不怕赫连家族的报复吗？”
<br/>　　“这也太牛掰了吧！”
<br/>　　……
<br/>　　却说叶尘一行人，进入天字号包厢之后。
<br/>　　那酒馆掌柜，似乎生怕叶尘记起他之前的怠慢之举，好酒好菜当即纷纷端了上来，并且表示所有一切免费。

<br/>　　纳兰柔然和一众侍卫，却一点也高兴不起来，个个满脸愁容，坐立不安。
<br/>　　“叶前辈，要不我们还是……赶紧走吧！”
<br/>　　最终，纳兰柔然忍不住小声劝道。
<br/>　　叶尘却依旧一脸平淡，
<br/>　　“不急！等戴安娜回来再说！”
<br/>　　纳兰柔然顿时急了，
<br/>　　“可是，赫连将军府在魔皇城势力极大，一旦让他们得到消息，我们恐怕……”
<br/>　　不料，叶尘索性直接闭上了双眼，淡淡道：
<br/>　　“他们若敢来，一并杀了就是！此事你无需多言！”
<br/>　　“是！”
<br/>　　纳兰柔然听出叶尘话中的不悦之意，顿时不敢再多劝，只好乖乖退到一旁，不过俏脸之上，愁色却越发浓郁……
