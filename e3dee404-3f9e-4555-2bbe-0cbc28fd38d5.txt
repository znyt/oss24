        “小景，爸爸陪着你坐旋转木马，你开不开心呀？”
<br/>
<br/>        “小景，你简直就是你爸爸的缩小版！”
<br/>
<br/>        一个个视频里，全是傅昼景和季小景，旁白自然全是季思妍的声音。
<br/>
<br/>        所以，那天的白天，傅昼景陪了季思妍和季小景一整天，而晚上才回来陪她童筱颖这个“正宫娘娘”。
<br/>
<br/>        童筱颖苦笑，果然如她所料。
<br/>
<br/>        真的是季思妍把傅昼景给叫回来的。
<br/>
<br/>        这些视频里，傅昼景帅气的脸上，露出了一个父亲对一个孩子的那种关爱之情。
<br/>
<br/>        就算季小景是私生子，但终归是傅昼景的亲生儿子，傅炎廷的亲曾孙。
<br/>
<br/>        所以，他们不会对这个孩子视而不见。
<br/>
<br/>        她给季思妍自己丈夫的联系方式，可不是为了让他们继续背着自己“叙旧”。
<br/>
<br/>        童筱颖以为，傅昼景和季思妍会私下处理好这件事情，但是没想到，他们会继续背着她“再续前缘”。
<br/>
<br/>        季思妍是什么心思，已经显而易见了。
<br/>
<br/>        她表面上说，不是来破坏他们的婚姻的，但暗地里，却又在做着一件又一件破坏他们婚姻的事情。
<br/>
<br/>        这个表里不一的女人，跟她玩起了“宫心计”。
<br/>
<br/>        童筱颖不想再被季思妍玩弄于鼓掌之中，于是放下了数码相机，抱起季小景，然后掏出手机，给自己和季小景录了一段小视频。
<br/>
<br/>        “小卫，你说，我要不要带着这孩子，去大海里寻找，这世上到底有没有美人鱼？”
<br/>
<br/>        小视频录好后，童筱颖想都没想，便直接发送给了傅昼景。
<br/>
<br/>        长时间的压抑，会逼疯一个人。
<br/>
<br/>        而她童筱颖，就是那个被逼疯的女人。
<br/>
<br/>        得知她的世界里还有季小景这个孩子的存在后的每个夜晚，她童筱颖都在独自以泪洗面。
<br/>
<br/>        床上的枕头湿了又干，干了又湿，她不想洗，索性把枕头扔了换了个新的，但是那个新的又被她给哭花了。
<br/>
<br/>        最近，傅昼景回来得很频繁，但每次都是晚上才会陪她，第二天早上就离开了。
<br/>
<br/>        她一直以为，傅昼景只是知道那孩子的存在，也知道要救那孩子的性命，但是不会去陪伴那孩子。
<br/>
<br/>        可结果，她想错了。
<br/>
<br/>        傅昼景每次回来，都是白天一整天陪孩子，晚上才会回她这儿过夜，第二天继续若无其事地回去出差工作。
<br/>
<br/>        一个男人，把日子过得这么累，何必呢？
<br/>
<br/>        童筱颖随后拿了季小景的衣服，给季小景洗完澡后，便按季思妍留给她的笔记本上的法子，哄季小景睡觉。
<br/>
<br/>        她为什么要照顾一个小三的儿子呢？
<br/>
<br/>        把这个小三的儿子掐死，她和小卫之间，就不存在任何问题了。
<br/>
<br/>        童筱颖的脑海里，甚至闪过一丝这样邪恶的念头。
<br/>
<br/>        但终究，这只是一个一闪即过的念头，她没胆子去杀人，更没勇气去结束一个孩子的生命。
<br/>
<br/>        童筱颖咬住下唇，在季小景睡着后，终于按捺不住，匍匐在床边，失声痛哭了起来。
<br/>
<br/>        如果不是爷爷傅炎廷跨国快递过来的那些东西，她或许不会看清自己现在所要面对的事实。
<br/>
<br/>        这孩子，不管是自己丈夫的爷爷还是自己的丈夫，他们都认了他。
<br/>
<br/>        那为什么就她童筱颖还不愿意去承认？！
<br/>
<br/>        两个小时后，傅昼景气喘吁吁地跑了回来。
<br/>
<br/>        这是童筱颖第一次见到他如此狼狈的样子。
<br/>
<br/>        只见傅昼景连鞋都顾不上脱，他双眼猩红，蓬头垢面，疾步来到她的面前。
<br/>
<br/>        “小景呢？”
<br/>
<br/>        事情彻底揭开后，他回来的第一句话不是跟她解释，而是将她从沙发上拽了起来，捉住她的手腕，冷冽地质问她，他儿子的下落。

<br/>        傅昼景仅仅只说了这三个字，就把童筱颖对他仅存的所有幻想，全部给一一撕碎了。
<br/>
<br/>        童筱颖忽然发现，她真的是错得太愚蠢了。
<br/>
<br/>        她自以为是，告诉季思妍自己丈夫的联系方式，自认为自己的丈夫不会去理会季思妍，更不会去在意那个孩子。
<br/>
<br/>        可笑，她居然拿这荒唐的自信去赌自己丈夫对她的爱。
<br/>
<br/>        童筱颖以为，傅昼景会为了爱她而选择回归家庭，漠视季思妍和季小景。却没想到，他傅昼景却背着她在跟私生子培养父子之情。

<br/>        对，父子之情，血浓于水。

<br/>
<br/>        “童筱颖，你疯了！”傅昼景红着眼睛怒吼，二话不说，拿起茶几上置物篮里的车钥匙，转身出了门。
