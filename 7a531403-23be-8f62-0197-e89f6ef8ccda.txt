　　第4章
<br/>　　蛤蟆岭下，黄三元面沉似水，清点着人员伤亡情况。
<br/>　　其实并没有什么伤亡，除了林少羽不知所踪之外，其他人全部安然无恙。可是，小城主的未婚妻却被山贼劫走，以城主的残暴，黄三元根本无法想象回到黑石城会遭遇什么。
<br/>　　“我问你们，是想死还是想活！”黄三元突然开口。
<br/>　　有兵丁大着胆子道：“黄管家，我们自然想活……”
<br/>　　“既然想活，我来问你，今天劫我们的山贼一共有多少人？”
<br/>　　“一……”这兵丁忽然福至心灵：“足足一百人！”
<br/>　　“一百人！”黄三元冷哼一声：“你眼睛瞎了嘛，山贼明明有三百余人，我们拼命搏杀，留下山贼十几具尸体，可惜依旧没护住少奶奶，林少侠也下落不明。”
<br/>　　黄三元突然抽出短刀，在胳膊上划出伤口，不顾流血，大声道：“随我回城，请兵！”
<br/>　　众兵丁有样学样，纷纷在自己身上做伤，完好的皮甲扯破，朝身上肉多的地方下刀。纵然疼地龇牙咧嘴，可相比黑石城主的残暴，这，也算不得什么。
<br/>　　人群渐渐走光，从草丛中走出一人，正是林少羽。望着天空繁星，林少羽孤零零站在那里，口中发出一声长叹。
<br/>　　“枉我林少羽以侠客自居，黄管家又如此信任我，最后却害的苏家姑娘被山贼劫走。”林少羽攥紧手中铁枪：“黄管家放心，林少羽一定替你将苏家姑娘救出，方不愧这七尺男儿之身！”
<br/>　　……
<br/>　　姓名：徐神机（臭名昭著的糟糕军师）
<br/>　　年龄：49岁
<br/>　　技能：无
<br/>　　隐藏属性：算无一策的狗头军师。
<br/>　　『愚者千虑必有一得，这是比愚者更愚的存在，在这个乱世肯收留他的人，大概只能用善良解释』
<br/>　　程大雷看着徐神机，一副生无可恋的表情。他本以为『普通』已经是最低级，看到徐神机才知道，原来普通之下，还有『糟糕』。尤其是这个隐藏属性……
<br/>　　别人是算无遗策，你是算无一策，这样的人留着做什么，当吉祥物么。
<br/>　　徐神机道：“大当家，大当家，你怎么了？”
<br/>　　“没什么，大概是我脸比较黑吧。”程大雷揉揉脸：“说吧，什么大事？”
<br/>　　“大当家，那女人我已经问过，名叫苏樱，是黑石城小城主的未婚妻，咱们把她劫来，可是闯下大祸。你可不知道黑石城的厉害，老寨住就是在于黑石城交战时中箭，不治而亡。这次咱们又招惹了他，如果大兵过来，咱们都得死。”
<br/>　　秦蛮和徐灵儿也在此时凑过来，程大雷问：“有谁知道黑石城情况？”
<br/>　　“我知道。”秦蛮道：“黑石城距离蛤蟆岭一百五十里，城主名叫韩虎踞，此人好色如命，残忍暴虐，城中有五千精锐守军。”
<br/>　　“五千对两个，大当家你觉得有把握吗？”徐神机主动把他和他女儿隔离出去。
<br/>　　“那你说怎么办，人已经劫了，要不你把人送回去，说我们错了，给人家跪下磕头，看他会不会饶过你。”程大雷。
<br/>　　“那……算我没说。”
<br/>　　程大雷已经对这军师不报任何希望，他将目光望向秦蛮，或许这壮汉有谋算之才。
<br/>　　“秦大哥，依你看，我们该如何办？”
<br/>　　“我觉得与其被动防守，不如主动出击，我们一路杀到黑石城，杀了韩虎踞，以有心算无心，这件事至少有七成把握。”
<br/>　　程大雷和徐神机都目瞪口呆的看着程大雷。四个人去主动攻击五千人，还七成把握……
<br/>　　得，又一个智力值为零的莽夫，看来还得自己担任起出谋划策的重任。
<br/>　　“依我看，咱们还是收拾东西，赶紧跑路，再晚就来不及了。”徐神机。
<br/>　　徐灵儿和秦蛮点头：“如今也只能这么办……”
<br/>　　“逃，绝对不行。”程大雷斩钉截铁。
<br/>　　“为何？”徐神机不明白程大雷怎么突然如此坚定。
<br/>　　程大雷本来也想逃跑，可徐生机的隐藏属性太吓人，这厮的隐藏属性是算无一策，也就是说，每出一个主意都是错误选项，要听他的逃跑，肯定遭殃。
<br/>　　“原因有三，其一，我们绑架的是少城主的未婚妻，这个人可以很重要，也可以很不重要，黑石城的少城主还缺老婆么？所以，五千人来攻根本不可能，至多一二百人。其二，咱们距离黑石城有一百五十里，一来一去，至少要五天时间，这五天时间就是我们的机会，占据地利之便，来一场伏击，未必守不住。”
<br/>　　嘟，接到任务『第一次山寨防守』，本任务难度过大，超过山寨目前实力。
<br/>　　连系统都给出提示，可见情况严峻。
<br/>　　得，既然任务都接了，就舍下命拼一场。
<br/>　　“那第三呢？”徐神机呆呆问，他没想到程大雷竟诌出这么多道理。
<br/>　　“第三么……”程大雷环视一圈：“我的父亲，蛤蟆寨上一任大当家就是在黑石城的围攻中，中了箭伤，最后伤口感染而死。秦大哥与韩虎踞更是有血海深仇。此仇不报，何以七尺立在天地间，所以这次，守得住要守，守不住也得守。”
<br/>　　“着！”秦蛮重重一捶桌子：“大当家，秦某这条命以后就是你的！”
<br/>　　徐神机没有秦蛮这么激动，他更是不明白什么叫『守不住也得守』。想了想，徐神机问：“由黑石城到这里，一共有两条路，一条是官道，能容两辆马车并行，一条是羊肠小道，中间还要经过一段一线天的峡谷，咱们要伏击，大当家怎么知道他们走那条路？”
<br/>　　“这就要靠军师你了？”程大雷意味深长的笑道：“军师仔细想，好好想，他们会走那条路？”
<br/>　　徐神机紧皱眉头，调动他为数不多，甚至从未动用过的脑细胞，最后斩钉截铁道：“必然是从官道来！”
<br/>　　程大雷哈哈一笑：“我已经知道他们走那条路了。”
<br/>　　徐神机更加困惑，自己都没把握，程大雷为何这么信任自己。
<br/>　　“依徐军师看，这次危机，咱们能度过吗？”
<br/>　　徐神机长叹一声：“不瞒大当家，我不报任何希望。”
<br/>　　“本来我也没有把握。”程大雷拍拍徐神机的肩膀：“但现在，有五分数了。”
<br/>　　就是一张卫生纸，一条底裤也有他的用处，程大雷深刻体味到这句话的道理。谁说徐神机没有用，他就是一个排除错误选项的超级武器啊。以后只要自己和谁有仇，就把徐神机派过去，有他出谋划策，肯定没过多久，对方就家破人亡，妻离子散。
<br/>　　想到这里，程大雷忽觉神清气爽，他吹了一声口哨，向自己的房间走去。
<br/>　　该是见见自己未来的压寨夫人了。
