<br/>    午时未到，陈明已经是在魔宗之外落下。
<br/>    
<br/>        陈明看向周围，历来防御森严的魔宗，此刻竟然没有多少人驻守，陈明笑了笑，“看来无疆等我已经等的有些急了，走吧。”
<br/>    
<br/>        众人跟随着陈明腾空，朝着魔宗深处飞去，周围所过，根本没有任何人阻挡他们。
<br/>    
<br/>        无疆抬头看了看天色，午时已经快到了，就在这时，无疆忽然看到陈明带着九位正道君主，已经是腾云在刑场前落下。
<br/>    
<br/>        李素衣能够感受到周围的魔宗修行者的神色变化，他回头，陈明正在缓缓朝着他走来，李素衣不知道对陈明说什么，千言万语化作一句，“师父！”
<br/>    
<br/>        陈明看向李素衣，给他一个放心的眼神，而后陈明看向无疆，无疆此刻也在看他。
<br/>    
<br/>        无疆狠戾笑道：“我倒是没有想到，你居然真的敢来！”
<br/>    
<br/>        陈明道：“魔宗而已，都来过几次了，有什么不敢来的，哦，对了，我来的时候你没有发现我！”
<br/>    
<br/>        无疆怒视向陈明，“本来我以为这个天下没有任何人是我的对手，但是你，陈明，你一步步将魔宗逼到绝境，我不得不说，你是一个人物，但是.......”
<br/>    
<br/>        “但是你做的这些又有什么用，我只要活一日，我便是这天下第一，这天下，没有任何人能够违抗我的意志！”
<br/>    
<br/>        无疆起身，从背后拔出一柄淡蓝色的长刀，而后朝着李素衣掷出，“知道什么叫做仙之下吗？那就是无可阻挡，无可违逆，我要杀的人，没有任何人能够保得住，你来了，我就让你亲眼见证你的三弟子死在你的面前！”
<br/>    
<br/>        魔宗修行者齐齐大喝道：“大宗主独步天下！”
<br/>    
<br/>        “大宗主的意志，没有任何人能够违抗！”
<br/>    
<br/>        陈明飞身而上，站在李素衣的身前，他朝着无疆掷出的长刀伸出左手，张开五指，滚滚地法力涌出，将无疆掷出的长刀定在空中。
<br/>    
<br/>        陈明抬头看向无疆，嘴角露出一抹笑容，“听说你是仙之下，巧了，我也是！”
<br/>    
<br/>        无疆有些惊诧地看向陈明，“这不可能，之前，你都只是君主级而已，怎么可能这么快晋升到仙之下！？”
<br/>    
<br/>        正道九位君主，也是在此刻真正相信了陈明的实力，陈明，真是是仙之下！
<br/>    
<br/>        而且是一位刚刚突破，正在壮年的仙之下！
<br/>    
<br/>        而魔宗剩下的五位宗主，也是惊诧不已，陈明，是仙之下了！
<br/>    
<br/>        仙之下，是整个大陆之上最强大的战力，这是不是意味着，无疆，将不再是唯一的一位仙之下。
<br/>    
<br/>        陈明没有回头，直接开口道：“带他们走！”
<br/>    
<br/>        龙君心领神会，连忙朝着李素衣飞去，直接化身为一条金色的长龙，伸出爪子抓住李素衣，而后在人群之中一把抓住了小猫儿。
<br/>    
<br/>        无疆哪里会让龙君这么轻易带走李素衣和小猫儿，“杀了他们！”
<br/>    
<br/>        埋伏已久的魔宗五位君主也在在此刻冲出，朝着龙君飞去，正道其他八位君主自然不会在此时看戏，瞬间就已经和魔宗五位君主战到了一处，正道八位君主，战魔宗五位君主，人数之上已经占了上风，龙君自然是轻易便已经飞走了。
<br/>    
<br/>        不过还有着一些大君或是魔王去阻挡龙君，不过这些兵力都无法阻拦龙君，龙君迟早能杀出去。
<br/>    
<br/>        君主们和魔宗的大军瞬间战作一团，陈明自连天的烽火之中看向站在魔宗之巅的无疆，两人对视，都看到了对方眼中浓烈的杀意。
<br/>    
<br/>        无疆伸手，淡蓝色的长刀回到他的手中，瞬间雷光万丈，“这是一场只能用生死分胜负的战斗，我将全力以赴，因为那是我对你最大的敬意。”
<br/>    
<br/>        陈明抱着手，挥动万道天竹，万道天竹之中散出一阵清风，自清风之中，一柄柄灵剑出鞘。
<br/>    
<br/>        无疆看向陈明，然后道：“知道我这把刀的名字吗？它叫做冬雷。”
<br/>    
<br/>        无疆的左手张开，出现了一柄黑色木剑，在木剑的剑柄上，有着一颗嫩芽，“千年之前，御剑宗是整个大陆之上最为强大的，大陆之上流传着一句话，刀有冬夏，剑分春秋，你应该有一把秋霜，与
<br/>    
<br/>        我手中的这一把春泥乃是一对，两柄合一便是仙器春秋，春泥秋霜，剑顺天命，修剑的，讲究一个顺心意，也就是顺天命，而我们修刀的，则是喜欢逆命，劈断那些我不认的命，故而我的两把刀叫做冬雷和夏雪。”
<br/>    
<br/>        “而我手中这一柄，便是春泥，作为这个大陆之上最强大地两人之间的一战，我想，应该不输千年之前我师父和御剑宗宗主的那一战。”
<br/>    
<br/>        无疆道完，将春泥丢向陈明，落在陈明的面前，陈明伸手，春泥如同是回到了久别重逢的主人手中，不过瞬间便已经被陈明炼化，陈明拿出秋霜，将两柄剑贴合，瞬间合成了一柄新的剑，而新的剑的剑身上，也没了泥字和霜字，只剩下春秋。
<br/>    
<br/>        陈明看向无疆，无疆要动真格了，他之所以将春泥给自己，并不是因为蠢。
<br/>    
<br/>        无疆并不蠢，不然他也做不到魔宗大宗主。
<br/>    
<br/>        他是数百年的天下第一，他冠绝一世。
<br/>    
<br/>        这是他的骄傲，他现在做的，不过是自己更加骄傲，更加狂妄，唯有最为狂傲的刀修，才能劈出最完美的刀，让自己胸中刀意更加强烈三分。
<br/>    
<br/>        同样的，他还想要打击陈明的道心，陈明毕竟只是挑战者，天下第一的名头，还在无疆的身上。
<br/>    
<br/>        无疆这么做，无非是在告诉陈明，我根本看不上你，你根本不是我的对手。
<br/>    
<br/>        他看见陈明握剑，就知道他成了，他的刀意更加浑厚，而陈明的剑意，变得低落。
<br/>    
<br/>        只是他没有想到的是，他遇到的，是陈明，陈明看向无疆，笑了笑，道：“你这种手段对我没用的，我此行而来，只有一个目的，为了达到这个目的，我不会在意其他的一切。”
<br/>    
<br/>        “你要知道，你的强大，只有死去才能让我安心啊！”
<br/>    
<br/>        “我没有那么多的弯弯绕绕，我现在心中只想着一件事，那就是杀你。”
<br/>    
<br/>        “只要杀意够浓烈，只要杀意够纯粹，其他的一切，都是假的，唯有活着，才能看到这天下谁主沉浮，这日月星辰，到底在谁股掌之间!”
