　　“大哥哥。”
<br/>　　丁秋月看着霎时醒来的萧逸，脸色一喜，亦松了口气。
<br/>　　“怎么，怕我醒不来？”萧逸笑笑。
<br/>　　“嗯。”丁秋月点点头，“不过我一直相信大哥哥一定能醒来。”
<br/>　　萧逸笑笑，摸了摸丁秋月的小脑袋。
<br/>　　下一秒，冰冷的目光，扫视着周遭一个个剑修。
<br/>　　最后，目光定格在冯济身上。
<br/>　　“你还未告诉我，你要如何不客气？”
<br/>　　冯济陡然脸色一黑，却只冷哼一声，不语。
<br/>　　全场剑修，面面相觑，亦无人回答。
<br/>　　萧逸耸耸肩，“呵，刚才我好像听到有人说，物以类聚，人以群分。”
<br/>　　“此话，倒是没错。”萧逸冷笑一声。
<br/>　　“废物，也只配与废物在一起。”
<br/>　　萧逸边说着话，亦边看向冯济。
<br/>　　显然，此话，他是在说冯济和那一个个学宫弟子。
<br/>　　“你说什么？”冯济与一众面容倨傲的学宫弟子脸色一冷。
<br/>　　“你们很清楚。”萧逸耸耸肩。
<br/>　　“怎么，很愤怒，要出手？”
<br/>　　“尽管出手便是。”
<br/>　　萧逸冷笑一声，“我也不介意半月之后，轰出剑帝碑空间的，是一具具尸体。”
<br/>　　“你…”一众学宫弟子勃然大怒。
<br/>　　冯济脸色愤怒，却终究不敢再多言，也未敢出手。
<br/>　　萧逸二字，在中域的名声，更多是臭名昭著，杀怒无常。
<br/>　　“罢了。”冯济负手而立，“剑帝碑空间内，参悟为主。”
<br/>　　“其他不必要的纠纷，我等懒得计较。”
<br/>　　“哼。”萧逸冷哼一声，亦懒得理会这群小人。
<br/>　　嘭…
<br/>　　恰在此时，远方，一道澎湃力量激射而来。
<br/>　　力量，瞬间打入萧逸体内。
<br/>　　萧逸不躲不闪，也未在意。
<br/>　　他知晓，这是剑气之路的开端，那块剑碑上所蕴含的力量。
<br/>　　但凡通过剑气之路的参悟者，都会得到剑碑上的力量。
<br/>　　当然，这份力量并不算太多。
<br/>　　那是属于剑碑数十年里吸收天地灵气所蕴含的力量。
<br/>　　万剑碑林，每数十年开启一次，剑帝碑空间亦是如此。
<br/>　　这些力量，是给予在其中参悟的剑修的一份馈赠。
<br/>　　萧逸感知了一下体内。
<br/>　　体内元力，增加了一成左右。
<br/>　　这也代表着，他的修为，瞬间从圣王境九重，踏入了圣王境巅峰，体内气泉已满。
<br/>　　以他那庞大的气泉，增加一成的元力高度，这股力量倒也算可观了。
<br/>　　当然，这算不得什么。
<br/>　　能收到剑帖，并来这里参加盛事的剑修，哪个不是一方大势力的绝世天骄，又哪个不是成名多年的老牌剑修。
<br/>　　这些剑修，压根不在乎这些许力量。
<br/>　　对于所有剑修来说，那条参悟之路，剑道方面的提升，才是最重要的东西。
<br/>　　毫无意义，这些通过剑气之路的过百剑修，在剑道方面都大有裨益。
<br/>　　日后回自家势力后，立刻就可以通过大量修炼之物快速飙升修为与实力。
<br/>　　至于萧逸。
<br/>　　之所以第一个来到第一层尽头，却又是足足两天才醒来，原因有些特殊。
<br/>　　两天前。
<br/>　　他来到尽头之外，距离走完剑气之路，也仅剩数米，不过就是几步的事。
<br/>　　可便是这几步距离，却生生困了他两天。
<br/>　　“一悟三千年。”萧逸不禁暗暗摇头。
<br/>　　不错，那最后一个画面，是斩星前辈的参悟画面。
<br/>　　斩星前辈，竟是闭关三千年，苦苦悟道。
<br/>　　萧逸亦置身其中，苦苦悟道。
<br/>　　说起来，这短短八天的参悟；对于在场任何一个通过了参悟的剑修来说，都是一次极其难得的经历。
<br/>　　短短八天，他们在参悟状态中度过的岁月，何止万年。
<br/>　　当然了，那些画面，终究不是真的连贯的一生，故时间上倒也缩水大半。
<br/>　　而另外一点。
<br/>　　事实上，自己亲身经历如此长的岁月画面，本该会有极大影响。
<br/>　　但这些画面，乃是断断续续，参悟者本就可以清晰知道自己是自己。
<br/>　　故其实真要对自己的思想记忆造成太大影响，倒也不会。
<br/>　　“可惜，我并不走斩星剑道。”萧逸暗暗遗憾。
<br/>　　虽说他代入了斩星前辈的身份，经历过了斩星前辈的所有战斗与参悟和修炼。
<br/>　　但说到底，这是斩星前辈自己的剑道之路。
<br/>　　而不是他萧逸自己的剑道之路。
<br/>　　若是萧逸愿意放弃他自己的一身剑道，从头修炼的话。
<br/>　　只要他拥有足够的修炼资源，他可以在顶多数十年的短暂时间内，就让自己成为斩星前辈那个层次的强者。
<br/>　　只是，这般‘复制’一个斩星前辈，又何意义？
<br/>　　他萧逸便是萧逸，他走的，只会是自己的剑道之路。
<br/>　　包括在场这些通过参悟之路的剑修，他们也经历过了他们所参悟的剑碑主人的一生战斗、感悟和修炼。
<br/>　　他们若愿意放弃自身剑道，在拥有足够的修炼资源下，短短数十年，成就到剑碑主人的那个高度。
<br/>　　只不过，他们既然能通过参悟之路；就代表着，他们比剑碑主人更为优秀。
<br/>　　假以时日，或百年后，他们自己也能让自己的剑道之路走到那个层次，甚至更强。
<br/>　　又何必急于一时，逐末舍本。
<br/>　　而走过参悟之路后，除却剑道层次的提升外，最重要的一点，便是…
<br/>　　“能否有真正更大的收获，便看这最后一剑了。”萧逸双眼一眯，暗暗想着。
<br/>　　“有把握进入第二层吗？”萧逸看向丁秋月，轻笑问道。
<br/>　　丁秋月摇了摇头。
<br/>　　“我感觉走到第一层尽头，已是极限了，最后一步，踏不出去。”
<br/>　　丁秋月尽是苦恼之色。
<br/>　　事实上，在场的这过百剑修，同样是这般境地。
<br/>　　他们早早便过了剑气之路，来到了尽头。
<br/>　　但最后一步，却踏不出去，也无法进入第二层。
<br/>　　“岳天，帮我照顾一番秋月。”萧逸看向岳天。
<br/>　　“是。”岳天点了点头。
<br/>　　但很快，岳天又反应了过来，“域主，你呢？”
<br/>　　“我？自然是进入第二层。”萧逸笑笑。
<br/>　　“剑帝碑之外见。”萧逸摸了摸丁秋月的小脑袋，自顾踏前一步。
<br/>　　第一层尽头处，有一巨大屏障。
<br/>　　那便是阻碍所有剑修进入第二层的存在。
<br/>　　锵…
<br/>　　冷焱剑凭空而现，一剑劈出。
<br/>　　萧逸蓦地闭上了眼睛，再度进入了参悟状态。
<br/>　　脑海中，再次出现了画面。
<br/>　　这，便是最后一步的画面。
<br/>　　画面中，斩星前辈，仿佛处于虚空之中。
<br/>　　斩星前辈，眼睛紧闭。
<br/>　　半晌，斩星前辈眼眸一睁，手中斩星，重重劈下。
<br/>　　这一剑，宛若斩破虚空。
<br/>　　这一剑，乃是他感悟三千年，一身剑道悉数融合的最强一剑。
<br/>　　斩星，重重劈下。
<br/>　　面前，是一巨大星辰。
<br/>　　剑落，巨大星辰轰然破碎，湮灭在剑气之中。
<br/>　　这，才是真正的斩星之剑，最强一剑。
<br/>　　现实中，萧逸猛地睁开了双眸，冷焱剑一劈而过。
<br/>　　屏障，轰然而开。
