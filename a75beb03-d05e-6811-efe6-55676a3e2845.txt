　　慕清霄当然知道，帝释天虽册封他为天门使者，却也不可能信任他，任谁也不会信任，一名突然出现的强者，特别是在这个时代。
<br/>　　所以，帝释天的目的，自然就是让神母来监视自己。
<br/>　　不过，慕清霄也不需要帝释天的信任，他只需要一同前往神龙岛，夺取龙源才是他的目的，而刚才的力量和剑意，就是表现给后者看的。
<br/>　　屠龙在即，帝释天绝不会舍近求远。
<br/>　　神母穿着一身青衣，长相貌美，慕清霄只是随意瞥一眼，就看出她金丹三层的实力。
<br/>　　神母其名骆仙，是现任洛阳城城主。
<br/>　　骆仙小的时候就失去了父母，沦落到街头成了一名孤儿，机缘巧合下，帝释天在一个寒冷的夜晚，将她带回天门，多年来抚养成人，纳为入室弟子，并授以武艺。
<br/>　　天门以帝释天为尊，但其中也是派系林立，天门共分为三系，人界，地界，天界！
<br/>　　因为骆仙的修炼资质尚高，所以帝释天传授她《圣心诀》，将其命名为天门使者【鬼子神母】，并主管着天门【人界】的一切大小事务。
<br/>　　跟在骆仙身后，慕清霄优哉游哉的走在庄园中，庄园中戒备森严，高手无数。
<br/>　　两个时辰后，骆仙耐心的给慕清霄介绍天门的情况，介绍完毕，给他安排好房间后，便匆匆离去。
<br/>　　这座庄园很大，各处都遍布着天门的高手，宗师境界不知凡几，而骆仙却将他的卧室，安排在[人界]最中心区域，可见对他的防备之心。
<br/>　　......
<br/>　　......
<br/>　　时光荏苒，岁月如梭，转眼又过去半年的时间。
<br/>　　天门内部，人界中心区域的宫殿中，慕清霄盘坐在床榻上，意守丹田，掌心向天，意境与周围融合，天地中的浓郁灵气，不断灌入他体内。
<br/>　　在这半年的时间里，慕清霄一直待在天门中，帮忙处理事物，许多重要的事物，都是由骆仙处理，帝释天对他的怀疑，倒是减轻许多。
<br/>　　恐怕在这半年内，帝释天没少调查自己的身份和来历。
<br/>　　当然，慕清霄根本不需有半点担忧，自己又不是这里的原居民，在江湖上也没有名声，完全是突然间冒出来的，帝释天能调查到才怪。
<br/>　　处理天门事务的同时，慕清霄在修炼方面也没有落下。
<br/>　　他的境界依旧停留在金丹八层巅峰，但体内的阴阳灵气，就宛如磅礴浩荡的黄河，用之不尽，抽之不完。
<br/>　　比起同境界的金丹强者，慕清霄体内的阴阳灵气，绝对要浓郁数十倍，区别也完全不同。
<br/>　　而且，随着他的修炼，差距还在不断拉开。
<br/>　　当修士抵达这种境界后，自身实力想要提升，需要花费太多的时间修炼，当然也有修炼的捷径，那就是机缘，但机缘可不是说来就来的。
<br/>　　半年的时间过去，六百年一次的惊瑞之日中将到来，屠龙在即，帝释天绝对会有所行动。
<br/>　　“咚咚咚！”
<br/>　　就在这时，慕清霄耳中传来一阵敲门声：“何人?”
<br/>　　“慕使者，门主召见。”
<br/>　　“好，我马上就过去。”
<br/>　　听到帝释天召见自己，慕清霄眼中闪过一丝精茫。
<br/>　　在半年的时间里，他的举动都很正常，除了平时装作狂妄，对权势极为痴迷外，天门中的事务，都尽心尽力去处理。
<br/>　　帝释天派出手下，在世界各地调查慕清霄的身份，最终一无所获，所以对他也信任起来。
<br/>　　屠龙在即，帝释天召集自己，慕清霄就知道，自己的等待没有白费，这一天终于来临了。
<br/>　　当慕清霄来到后山冰窖中，也就是所谓的天界时，神母，冰皇，神将等人已经全部到齐。
<br/>　　其中居然还有一位熟悉，一位陌生的身影。
<br/>　　熟悉的身影自然就是步惊云，而那位陌生的身影，恐怕就是皇影了。
<br/>　　慕清霄只是淡淡的瞥了眼步惊云，仿佛完全是在看不相干的人，缓缓走上前来，抱拳道：“慕清霄，拜见门主。”
<br/>　　“嗯，既然人已经到齐，我有一件事情需要宣布，惊瑞之日即将到来，你们立功的时刻也来临了。”
<br/>　　帝释天端坐在千年寒冰椅上，目光中满含期待。
<br/>　　“本座所创的圣心诀能让人不死，却不能防止衰老，可想要不死不老的话，唯有龙元，所以本座需要你们出手，帮本座屠龙。”
<br/>　　听到帝释天的话语，慕清霄心中不屑到了极点。
<br/>　　这老不死的玩意，真是爱给自己脸上贴金，明明是凤血让你不死不灭，怎么就变成你所创的《圣心诀》了。
<br/>　　要是你创出的《圣心诀》能够让人永生，你岂不是要上天！
<br/>　　紧接着，帝释天就开始宣布，一系列屠龙计划，甚至对在座的各位，都有所承诺。
<br/>　　怀空的要求，就是让帝释天放过余楚楚和第二梦。
<br/>　　步惊云并没有提任何要求，但心中却策划着，如何获得龙元，等实力暴增后，宰了帝释天这个老王八。
<br/>　　在场所有人，各有各的心思。
<br/>　　听完帝释天的屠龙计划后，慕清霄心中极为平静，脸上却表现的极为震惊，装作一副无知的模样。
<br/>　　“怀灭，出海的事情，其余神兵可否都已经备妥？”
<br/>　　“启禀主人，都已经准备好了。”
<br/>　　“非常的好，事情就是这样，你们都下去吧，半个月后启程，怀灭留下，我还有事情要吩咐你。”
<br/>　　慕清霄对接下来的事情，并不感兴趣，隐晦的瞥了眼步惊云，径直离去。
<br/>　　天门内各处都有暗哨，两人也不方便接触，所以他也没有打招呼，免得让帝释天心生猜忌。
<br/>　　等慕清霄等人远去后，帝释天目光落在神将身上。。
<br/>　　“屠龙人选，就按照之前所安排的，步惊云，你们兄弟二人，破军。”
<br/>　　“断浪诡计多端，不是没有用的人，原本我打算让无名也参与其中，可是他冥顽不灵，本领却是比你们高不少，实在是很难操控。”
<br/>　　说着说着，帝释天眼中精茫大盛，大声笑道：“还好天助我也，半年前居然来了个慕清霄，本领丝毫不比无名差，由他来代替无名的位置，最适合不过。”
