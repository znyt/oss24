　　苏荷。
<br/>　　姓苏。
<br/>　　这个名字，在加上这三棵金竹，突然让楚风眠想到了曾经的一位旧人。
<br/>　　“回禀前辈，这三棵金竹，都是小女家中祖上之物，从家中祖屋中生长的，后来家道中落，祖宅被他人占据，家父临走时将这金竹取下的。”
<br/>　　苏荷轻声开口道。
<br/>　　“家中祖屋……你家祖宅边，是不是有着一尊江河，名为器江？”
<br/>　　楚风眠听到苏荷的话，突然开口询问道。
<br/>　　“不错，正是。”
<br/>　　苏荷抬起头，回答道。
<br/>　　“天兵苏家，没想到居然万年之后，便是沦为了这般田地。”
<br/>　　楚风眠看向这苏荷，不由的感叹了一声。
<br/>　　在万年之前，天兵苏家，是多么的强盛，就算是当时七大宗门，都不敢小看这天兵苏家，将其奉为上宾。
<br/>　　因为天兵苏家中，掌控有着天下第一的锻造之术，当初的无数天级灵器，都是天兵苏家锻造而出的。
<br/>　　因此，才被成为天兵之名，锻造天兵。
<br/>　　当时的苏家家主，跟楚风眠倒是还有着几分交情，那时的楚风眠，身为剑修，却没有灵力。
<br/>　　寻常天级灵器，那一件想要动用，都至少需要一定的力量，当时的楚风眠手无缚鸡之力，甚至都无法动用一件最为寻常的灵器。
<br/>　　所以他才拜访当时的天兵苏家，为的就是锻造出一把不需要灵力就足以动用的灵器。
<br/>　　后来历经种种，楚风眠成功了，得到了一把不需要任何灵力就可以动用的灵器，也因为跟苏家结缘。
<br/>　　没想到当初强盛的天兵苏家，居然是沦落到了这样一番天地，也是足以让人唏嘘的。
<br/>　　“这三棵金竹的价值，远超百枚极品灵石，我可以给你一次机会，你再报出一个价格来，我也不会沾了你的便宜。”
<br/>　　楚风眠看着苏荷，缓缓的开口道。
<br/>　　毕竟是旧人之后，这三棵粹金竹的价值，的确要远超百枚极品灵石。
<br/>　　苏家已经没落，楚风眠也不会去选择扶持她，不过给她一个公平的交易，楚风眠还是愿意的。
<br/>　　“小女，愿意将这三棵金竹，送个前辈。”
<br/>　　苏荷看着楚风眠，沉寂了一下，才再度开口道。
<br/>　　“我只需要前辈帮一个忙。”
<br/>　　“说。”
<br/>　　楚风眠直接开口道。
<br/>　　“我想让前辈，为小女报仇，我苏家虽然衰败，却本也是一方乡绅，可是却被王家所灭，只要前辈愿意帮小女报仇，灭了杨家，不止这三棵金竹，少女愿意为奴，侍奉前辈。”
<br/>　　苏荷跪在楚风眠面前，低声开口道，她的声音虽然柔弱，却是蕴含有着极强的杀意。
<br/>　　灭门之仇，不共戴天。
<br/>　　整个苏家，除了她一人逃出外，便是所有人，都是死在了王家手中。
<br/>　　“王家？那个王家？”
<br/>　　楚风眠思考了一下，也不记得有过一个什么王家。
<br/>　　楚风眠万年重生，在万年中崛起的家族，他也并不熟悉，更何况楚风眠又不是西域之人。
<br/>　　对着西域家族，更是知之甚少。
<br/>　　“青河边城的王家，王家家主，乃是终焉王朝的一尊公爵，道元公。”
<br/>　　苏荷出声解释道。
<br/>　　“道元公。”
<br/>　　楚风眠在脑海中搜索了一下这个名字，便是想了起来，这道元公，算是终焉王朝的一尊公爵，生死境五重左右的境界。
<br/>　　在终焉王朝的公爵中，都算不上是什么人物，青河边城，也只是一个小地方罢了。
<br/>　　“苏家，当年我似乎还欠他一份人情，既然如此，今日我就算是还了他当年的一份情了吧。”
<br/>　　楚风眠感叹了一声，对着苏荷出声道。
<br/>　　“好，一个小小王家，我帮你灭了他们，为你苏家报仇。”
<br/>　　“多谢前辈。”
<br/>　　苏荷看到楚风眠答应下来，脸色露出了几分喜色，急忙的将金竹递了过来，开口道。
<br/>　　“这是三棵金竹，还请前辈收下。”
<br/>　　“你就不怕，我直接离开？”
<br/>　　楚风眠收下这三棵金竹，笑着开口道。
<br/>　　“前辈说笑了，区区三棵金竹罢了，还入前辈的法眼就不错了。”
<br/>　　苏荷笑着开口道。
<br/>　　“是吗？天兵苏家的人，果然都是聪明人。”
<br/>　　楚风眠随意开口了一句，便是没有在说些什么。
<br/>　　楚风眠已经是说明了这三棵金竹的价值了，苏荷心中自然也清楚，不然的话她也不会趁机，想要让楚风眠帮她报仇。
<br/>　　现在急忙将三棵金竹送给楚风眠，也只是让博得楚风眠的一些好感罢了。
<br/>　　回想万年前，当初楚风眠想要从天兵苏家哪里，打造灵器的时候，也是被坑的很惨的。
<br/>　　不过现在想想，都已经成为过去了，楚剑白已经陨落了。
<br/>　　而天兵苏家，也已经衰落至此。
<br/>　　“既然如此，走吧，带我去青河边城。”
<br/>　　楚风眠挥了挥手开口道。
<br/>　　“是。”
<br/>　　苏荷急忙收拾好东西，便是带着楚风眠，直接一路离开，进入到了传送法阵之中去。
<br/>　　青河边城所在的地方，距离这东升城并不算近，半日时间，天色快要落幕的一刻，才走了大约一半的路程。
<br/>　　这其中大部分的路程，还都是通过传送法阵前往的。
<br/>　　九域之大，无边无际，单单是其中之一的西域，以楚风眠的实力，都无法走遍任何一个角落。
<br/>　　一路上，楚风眠无聊，也跟着苏荷交谈了不少。
<br/>　　“你身为天兵苏家的人？怎么没有学习锻造之术？”
<br/>　　楚风眠观察了一眼这苏荷，也有些奇怪的想到。
<br/>　　天兵苏家当初的强大，也并非是因为他们的实力，而是锻造之术。
<br/>　　现在这苏荷，虽然修的是武道，却并没有什么资质，资质平平罢了，穷其一生达到神海境都不易。
<br/>　　可是若是苏荷修炼的是锻造之术的话，那以她天兵苏家的血脉，只怕是足以在九域中，很快崛起。
<br/>　　“这，我苏家的锻造之术，在数千年前，便是已经失传。”
<br/>　　苏荷沉吟了一会，开口道。
<br/>　　“按照古籍记载，似乎是被他人夺去了，如此一来，我们苏家才逐渐摔落的。”
<br/>　　苏家曾经的繁华强盛，苏荷也曾经知晓。
<br/>　　只是现在的苏家，却是不如当年的万分之一了。
<br/>　　“被他人夺了去？看来七大宗门的人，还是不死心啊，夺了剑道，又要夺了苏家的锻造之术了。”
<br/>　　楚风眠心中不由的感叹了一声。
<br/>　　七大宗门如今的强大，不知道是当初夺取了多少家族，宗门的底蕴。
<br/>　　看来苏家，也没有逃过七大宗门的掠夺。
