死火凰也不敢小觑，可惜并不在这里。<br/>    宁涛的步速很快，三个仙后，两个干女儿，还有叶家兄妹俩紧随他的步伐往城主府走去。<br/>    “小叶老师，我要问你们一个问题。”宁涛说道。<br/>    叶凡仙慌忙说道：“仙王陛下，你、你叫我小叶就好了，老师……小民愧不敢当呀。”<br/>    宁涛笑了笑：“好吧，那我就叫你小叶，以后你也别自称小民了。”<br/>    叶凡仙点了一下头。<br/>    宁涛说道：“我要问的问题就是，待会儿见到你们的外公雷斯华德的时候，我是杀他，还是不杀他？”<br/>    如果雷斯华德不是叶归根和叶凡仙的外公，他都懒得问这个问题，肯定会杀掉。可是那人毕竟是叶归根和叶凡仙的外公，他要是不分青红皂白杀了，叶归根和叶凡仙要是理解还好，但要是不理解，兄妹俩的心里肯定会留下疙瘩，甚至是恨他，所以在见到那个人之前他要把这个问题问清楚。<br/>    叶凡仙和叶凡仙对视了一眼，却没人回答他这个问题。<br/>    宁涛也不给给兄妹俩压力，只是说了一句：“不用现在给我答案，等到了城主府，见到了雷斯华德再告诉我答案吧，那个人的命运我交给你们来做决定。”<br/>    兄妹俩同时点了点头，心情沉重的样子。<br/>    对于宁涛来说，这只是一个很简单的问题，可对于兄妹俩来说这却是血亲的生死问题。在来寒星城之前，兄妹俩不知道幻想过多少次杀了雷斯华德替父亲报仇，替母亲找回公道，可是马上就要实现愿望，直面这个问题的时候，他们却犹豫了。<br/>    兄妹俩毕竟是与世隔绝，内心良善，忠厚老实之人，杀外公这种事情怎么可能一下子就做出决定？<br/>    砰砰砰！<br/>    一串枪声突然响起，几颗枪械法器的子弹撕开虚空往正在街上行走的宁涛一大家子人飞来。<br/>    宁涛腰间的大日葫芦忽然一颤，一股奇大的吸扯力出现。那几颗子弹突然偏离了弹道，径直飞进了大日葫芦之中。<br/>    那几颗子弹没有爆炸，什么动静都没有。<br/>    大日葫芦连妖都可以收，区区几颗偷袭的子弹根本就不在话下。不过，如果是弹雨的话它就没法了。但宁涛还是有手段对付，那就是放出天赐天生床，让那床墙一样的挡在身前，还有什么子弹能穿透过来？<br/>    开枪的人躲在街边的一幢船形的房子里，那房子颇为大气，大概是一个贵族的房子。<br/>    不等宁涛招呼，一道火影便离开了队伍，一头撞进来射出子弹的二楼窗户之中。<br/>    轰！<br/>    烈焰冲开几乎所有的房门和窗户，紧接着就连屋顶都被掀了起来。<br/>    只是一转眼，整座船形的房子便化成了一团巨大的火焰，岩石都会被烧成岩浆，更何况是人。<br/>    偷袭的人长什么样，又是什么心态，这些都不重要了，他和那座房子里的所有人必然已经化成了灰。<br/>    不死火凰从火海之中走了出来，一身七彩的羽裳，身姿卓越，脸上没有任何表情，高冷无比。给人的感觉，她刚才只是划燃了一根火柴，而不是烧了一座房子，杀了多少人。<br/>    忽然，不死火凰张嘴，她的声音向四面八方碾压过去：“谁还想死，向我开枪！”<br/>    霸气！<br/>    这街上，想躲在房子里开暗枪干掉宁涛的人不知道有多少，也不知道有多少人觉得可以发动巷战消灭来犯之敌。可是不死火凰一出手，谁还敢开枪？<br/>    别说是向不死火凰和宁涛开枪，就算是随地吐口痰恐怕也要想一下会不会招来灭顶之灾！<br/>    街道彻底清净了。<br/>    宁涛说道：“我们走吧。”<br/>    他又带路往前走。<br/>    队伍最后，狐姬凑到了狐媚的耳边，声音轻若蚊呓：“看见了吗？你还着急不着急？”<br/>    狐媚吐了一下舌头。<br/>    一路上再无人开冷枪，倒是军营方向法符炸开的声音响个不停。在那个方向，火光冲天，空气中还隐约传来天人的惨叫声。<br/>    今夜，寒星城在哭泣，在流血。<br/>    最后一段路被甩在了身后，迎面而来的是一个宽阔的广场。<br/>    城主府就在广场的尽头，一座雄伟的船形建筑。<br/>    寒星城肯定有神庙，但并不在这里，宁涛也没有兴趣再进无的神庙看看。天池城的城主彭斯的记忆已经让他搞清楚了突然的信仰问题，也就没有必要再进什么神庙了。<br/>    广场上空荡荡的，一个人都没有。<br/>    城主府也是门窗紧闭，门缝和窗缝里就连一丝灯光都没有。<br/>    轰咔！轰咔！轰咔……<br/>    震耳的脚步声忽然从广场的一侧传来。<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>
